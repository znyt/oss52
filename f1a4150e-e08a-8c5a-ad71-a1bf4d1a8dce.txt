　　接下来的日子里，叶尘重新过上了平静的校园生活。
<br/>　　李虎三人虽然都对叶尘的身份十分好奇，可是叶尘不说，他们也都十分识趣的没有多问，一切都像之前一样。
<br/>　　而随着叶尘如今的修为，踏入金丹之境，修炼所需要的灵气再次暴涨，以地球如今的灵气浓度，远远不足以支撑叶尘的修炼，他索性也懒得再浪费时间修炼。
<br/>　　偶尔听听课，没事的时候就带着曦月四处游玩，日子倒也过的逍遥快活。
<br/>　　但，值得一提的是，
<br/>　　自从叶尘一人压倒米国之后，修仙武道之风，开始风靡全球，在华夏更是如火如荼，风气风貌和之前已经大为不同。
<br/>　　就连各大学之中，也纷纷开办了各种武道社，甚至有些大学更是直接纳入专业课程之中，专门培养武道人才。
<br/>　　以天海大学为例，大大小小的武道社，足足一下出现十几个，而那些最顶级的武道社，甚至专门请来了化劲宗师级别的武道高手坐镇！
<br/>　　如此一来，在众人见识到真正的武道高手之后，顿时越发激起了普通人的习武之念。
<br/>　　短短几个月之后，武道之风已经彻底风靡了华夏各大高校，甚至在政府的默许甚至鼓励之下，各大高校之间，已经开始举办一些颇具规模的武道交流会，类似于之前那些电子竞技类的大型赛事。
<br/>　　当然，相比于虚拟世界的电子竞技，这种拳拳到肉的比赛方式，显然要更加的刺激！
<br/>　　这也直接导致全球的游戏业，进入了一个前所未有的衰退期，电子竞技开始逐渐被各种武道比赛所取代。
<br/>　　对于这个现象，叶尘虽然有些始料未及，不过他却也乐见其成。
<br/>　　毕竟，当初四界之所以隐遁起来，将修真道法断绝于世，本就是为了一己私利，防止世俗界的人，和他们抢夺天地灵气。
<br/>　　如今全球武道复苏，必然会和四界争夺天地灵气，而对这于叶尘来说，未尝不是一件好事。
<br/>　　……
<br/>　　这一日，
<br/>　　叶尘在校外的别墅中，正打算为曦月炼制一些神魂丹药，李虎和陈东忽然急匆匆赶了过来。
<br/>　　“尘哥，大事不好了！唐威被人打成重伤，快要死了！”
<br/>　　“你说什么！”
<br/>　　叶尘听到这个消息，顿时霍然站了起来，眼眸中泛起一抹寒意，瞬间连周围的空气，仿佛都要凝固了一般。
<br/>　　陈东和李虎两人全都浑身一颤，险些没能忍住一下子跪倒在地上，好在那道恐怖的气息，只是一闪即逝，就如同没有发生过一般。
<br/>　　以叶尘如今的实力，哪怕只是流露出一丝真实的怒意，也远远不是普通人可以承受的，好在他反应够快，及时收起了气息，深吸了一口气，强行将心中的怒火压了下去，
<br/>　　“虎子，到底出了什么事？你慢慢说！”
<br/>　　经过李虎一番解释，外加陈东的补充，叶尘很快便了解了事情的原委。
<br/>　　原来，
<br/>　　前些日子，唐威在天海大学成立了一个叫做‘唐门’的武道社。
<br/>　　唐威本就是武道世家出身，而且最近又得叶尘暗中帮助，修为突飞猛进，已经达到半步化劲宗师的境界，在整个天海大学之中，绝对是凤毛麟角的存在！
<br/>　　因而这‘唐门’才刚刚成立不久，就有很多人踊跃加入，没用多久的时间，就已经招收了上千人，一跃跻身成为天海三大武道社之一，人气极高。
<br/>　　然而，
<br/>　　木秀于林，风必摧之。
<br/>　　唐门的崛起，引起了其他两大武道社的忌惮，于是便处处排挤，隔三差五便会有人挑衅。以唐威如今的修为，寻常的学生之中，自然没人是他的对手，可是另外两大武道社，请来了两位武道宗师坐镇！
<br/>　　今天，‘唐门’和同为三大武道社的‘龙盟’发生了冲突，唐威带人和龙盟的人理论，没谈拢，双方最终决定以擂台战的形式来解决。
<br/>　　然而，谁也没有想到，龙盟竟然请出一位化劲宗师，一掌便将唐威打成了重伤……
<br/>　　听完事情的原委之后，叶尘眉头微微一皱，脸色越发阴沉，
<br/>　　“唐威人现在在哪？立刻带我过去！”
<br/>　　“哦哦，好！”
<br/>　　通过这段时间的接触，李虎三人对叶尘，已经是发自内心的崇敬，对他的话自然是言听计从，当即带着叶尘便朝外面走去，一边走一边解释道：
<br/>　　“我们第一时间就把威哥送到了医院，医生说伤势太重，恐怕很难熬过今天！”
<br/>　　陈东也在一旁插口道：
<br/>　　“更可恨的是，唐威是在公共擂台上受的伤，我们就算报警恐怕也没用！”
<br/>　　为了适应如今的形势，提升华夏大学生的武道素养，这几个月来，各大国已经陆续通过了一项类似的议案：
<br/>　　但凡武道中人，只要登上公众擂台接受挑战，便不受法律保护，生死勿论！
<br/>　　因而，唐威即便是重伤致死，对方也不需要承担任何法律责任。
<br/>　　听完两人这话，叶尘的眉头顿时皱的更紧了，当下也不再继续隐藏实力，直接两手分别抓住李虎和陈东，然后身体微微一晃，便已经以最快的速度，直奔医院而去。
<br/>　　至于李虎和陈东两人，只感觉两眼一花，一阵头晕目眩，等到他们的身形重新稳定下来，抬头定睛一看，发现自己竟然已经来到了医院之中。
<br/>　　“这、这……什么情况？”
<br/>　　两人顿时又惊又骇，然而还没等他们回过神来，两眼再次一花，下一刻人已经来到手术室门口。
<br/>　　“咔！”
<br/>　　叶尘随手一挥，手术室的电子铁门，便已经自动弹开。
<br/>　　“小子！这里是手术室！不能乱闯！”
<br/>　　等到门口两旁的保安人员反应过来，想要上前阻拦，叶尘早已经迈入手术室中。
<br/>　　“没看到正在做手术吗？谁让你进来的！”
<br/>　　手术台上，正在为唐威做手术的一众医护人员，看到叶尘这个不速之客，顿时一个个全都炸了。
<br/>　　叶尘理都不理，脚下轻轻一跨，人便已经来到手术台前，
<br/>　　“这里没你们的事了，你们可以出去了！”
<br/>　　淡淡说完这句之后，直接随手一招，取出一枚丹药，塞入唐威的口中，然后猛然抬手，在唐威的胸口处一拍。
<br/>　　“你干什么！”
<br/>　　唐威的主刀医生看到这一幕，顿时忍不住大吼了一声，当即便要上前阻止。
<br/>　　然而，
<br/>　　接下来的一幕，却让他和其他众人全部目瞪口呆。
