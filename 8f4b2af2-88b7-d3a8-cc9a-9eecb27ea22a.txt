        李忆莎只身走了过去，见傅夜沉一个人睡在大床上，大半个麒麟臂露在了空气里，顿时让她松了口气。
<br/>
<br/>        床头柜上还有一瓶新开的威士忌，里面有未喝完的酒。
<br/>
<br/>        这个时候，李忆莎心头暗生一计。
<br/>
<br/>        如果，真如那个女孩所说的那样，明天早上，傅夜沉一醒来就不记得今晚所发生的事情的话，那么她……
<br/>
<br/>        想到这里，李忆莎拿起座机，叫了一名女服务员上来。
<br/>
<br/>        她给了女服务员一笔钱，两人躲进浴室里，让那个女服务员按她的指示，在她的身上吸了一身的吻痕。
<br/>
<br/>        脖子上和锁骨上的吻痕，她是照着刚刚那个女孩的位置，让女服务员帮她吸出来的。
<br/>
<br/>        事情办好后，李忆莎又给了女服务员一笔封口费。
<br/>
<br/>        女服务员拿着钱离开后，李忆莎关了门，回到了傅夜沉的大床边。
<br/>
<br/>        她慢条斯理地拉开了后背上的拉链，脱光了自己身上所有的衣物，然后捡起墙角地面上的一片玻璃杯碎片，轻轻地扎破了自己的手指。
<br/>
<br/>        李忆莎扔了玻璃碎片，回到床边，轻轻地掀开被褥，爬上床，躺在了傅夜沉的怀中。
<br/>
<br/>        为了嫁给自己爱的男人，她决定豁出去了！
<br/>
<br/>        她将手指上的那滴冒出来的血珠，抹在了自己屁股下压着的床单上。
<br/>
<br/>        明天早上起来，她就是他傅夜沉的新婚妻子。
<br/>
<br/>        李忆莎嘴角微扬，满脸洋溢着幸福的微笑，将脑袋枕在傅夜沉的胸膛上。
<br/>
<br/>        他的胸膛光滑健硕，八块腹肌透着十足的男人味。
<br/>
<br/>        傅夜沉平常很喜欢健身，所以他的体型几近拥有完美的黄金比例。
<br/>
<br/>        她爱着这个男人，爱了十四年，如今总算是有了个盼头。
<br/>
<br/>        李忆莎心里打着小鼓，脸颊也红了个通透。
<br/>
<br/>        上半夜，她一直未睡，直到下半夜，她才在傅夜沉的怀中渐渐地进入了梦乡。
<br/>
<br/>        她希望，自己能梦想成真。
<br/>
<br/>        翌日。
<br/>
<br/>        童筱颖一大清早，不跟夏凌焰打招呼，就自顾自地赶上了云城到西镇的早班车。
<br/>
<br/>        她到了西镇后，还遇上了赶集，于是买了一些菜种、花种、果种，带回了深山区。
<br/>
<br/>        童筱颖现在住的这个地方，可谓是世外桃源。
<br/>
<br/>        放眼望去，蓝天之下是一座座耸入云端的群山。
<br/>
<br/>        每年春季，烂漫的野生桃树上，桃花盛开时犹如一片粉色的薄雾，笼罩着整个半山腰，那样子分明像极了大山飘逸的裙带。
<br/>
<br/>        晨曦初照，群山像含羞的少女，在云雾间若隐若现；
<br/>
<br/>        日落西山，余光横照，群山又显得妩媚娴静。
<br/>
<br/>        这片深山大林区域，保留着原始森林的美丽与神秘。
<br/>
<br/>        小时候，童筱颖看它们的时候，不觉得它们是一种别致的美景。
<br/>
<br/>        现在长大了，对比着大城市的繁华，她才发现，这里才是一片心灵的净土。
<br/>
<br/>        她身世不详，遭诺母嫌弃，丢在这农村深山里跟着赤脚医生的外婆生活，被外婆抚养长大。
<br/>
<br/>        高中在镇上念的书，邂逅自以为是的美好初恋，但不被父母看好，却在念大专的时候被初恋劈腿抛弃。
<br/>
<br/>        她在夜店里卖过唱，在大企业里打过工，还去了国外念了最好的本科大学。
<br/>
<br/>        结过婚，可惜没有孩子，为老公献出了一颗肾，而老公却出轨了。
<br/>
<br/>        她被离婚，还被净身出户。
<br/>
<br/>        天下之大，竟然没有她的容身之地。
<br/>
<br/>        几经周转，最终，还是外婆这里肯收留她。
<br/>
<br/>        其实，连外婆都不是她的亲生外婆。
<br/>
<br/>        童筱颖跪坐在外婆的坟头，心里的酸楚，一一向外婆诉说。
<br/>
<br/>        以前，她没有学历的时候，总想着通过自己的努力赚钱，来改善自己和家里的生活条件。
<br/>
<br/>        她想在大城市立足，想留在大城市里闯出自己的一片天地，想让老家的乡邻，看到她的扬眉吐气。
<br/>
<br/>        可结果，她现在有了高学历，却不愿意再去大城市里拼搏了怎么办？
<br/>
<br/>        “外婆，是你的在天之灵，让我在大城市里与你的亲外孙相遇的对吗？我嫁给了小卫，想跟小卫厮守一辈子，但是到头来……我跟小卫已经不可能了。”童筱颖面对着外婆的墓碑，自言自语地说。
<br/>
<br/>        这段日子，她想通了。
<br/>
<br/>        既然傅昼景和季思妍有了孩子，那么她就退出好了。
<br/>
<br/>        哪怕她仍旧爱着傅昼景，但是，傅昼景和季思妍的事情，却永远成为了她心头的一根刺。
<br/>
<br/>        生了间隙的爱情，就算勉强维持在一起，日子过得一定也不会舒坦，就更别提会幸福了。
<br/>
<br/>        她可以和小卫同甘共苦，不畏生活的艰难，却终究还是做不到去包容爱情的背叛。
