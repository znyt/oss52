　　“不行，这一群魔族的攻势越来越猛烈了，快要扛不住了，在死撑下去，也就只有死路一条了！”
<br/>　　在这阵法之中，一名五灵宗圣子，拼命的在抵挡着魔族的轰击。
<br/>　　可是纵然是苦苦支撑，这一道大阵之上破碎的裂痕，也越来越多，这样下去，只怕是很快就要崩塌。
<br/>　　“木麟师兄，不能在这样撑下去了！”
<br/>　　又一名五灵宗圣子大声的喊道。
<br/>　　“木麟师兄，我们突围出去吧！”
<br/>　　一名名五灵宗圣子的目光，都看向了位于阵眼的木麟。
<br/>　　木麟是这一群五灵宗圣子中的领袖，一身实力已经是达到了八劫古帝，是这一群五灵宗圣子之中的最强者。
<br/>　　如今就是他作为阵眼，在保护这这一群五灵宗圣子。
<br/>　　“不行，现在突围，必然损失惨重！”
<br/>　　木麟听到众人的话，脸色也是阴沉无比，想要突围而走，他可以，但是这十几名五灵宗圣子，至少要死伤大半。
<br/>　　这种损失，五灵宗也承受不住。
<br/>　　“想要突围出去？”
<br/>　　“就凭你们几个小家伙？不用白费力气挣扎了！你们几个的精血，都会成为我们魔族的补品！”
<br/>　　几名魔族，听到木麟等人的话，哈哈大笑起来。
<br/>　　“夜叉皇，独魔皇，不用在跟这几个废物纠缠了，一起出手，打破了他们的大阵！”
<br/>　　在这无数魔族的背后，四道身影，正站在那里。
<br/>　　这四道身影，就是如今四位魔族的皇者，夜叉皇，独魔皇，阎魔皇，鬼魅皇。
<br/>　　“好，杀了这些人，不用跟这些蝼蚁在多做纠缠了！这九域遍地都是宝藏，都是天才，我们有的是目标可以猎杀，我先解决了他们！”
<br/>　　一尊十几米高，头生独角的魔族，一下子走了出来，盯着那木麟等人的大阵，突然独角变的巨大无比。
<br/>　　在这独角之上，一道血红光芒突然迸发而出，直指那一群五灵宗圣子的大阵之上。
<br/>　　这五灵宗圣子联手布置而出的大阵，本就是在无数魔物的围攻之下，摇摇欲坠。
<br/>　　在这血红光芒的轰击之下，当即破碎开来。
<br/>　　砰的一声。
<br/>　　大阵破碎，其中的五灵宗圣子都是被直接震飞出去，吐出了一大口的精血。
<br/>　　身为阵眼的木麟，更是一下子受了重伤，气息无比虚弱。
<br/>　　失去了大阵的保护，无数魔族都是大笑连连，一个个冲了上去。
<br/>　　“不好！”
<br/>　　“拼了！”
<br/>　　那些五灵宗圣子眼神之中都是露出了几分决断之色，落到这些魔族的手里，比起死了还要痛苦，一个个施展出最后手段，要做困兽之斗。
<br/>　　“死吧！”
<br/>　　一名古帝实力的夜叉，盯着受了重伤的木麟，咆哮一声，当即斩杀了过去。
<br/>　　就在这时，刹那之间一道剑光，突然破空而出，直接斩向了那夜叉的胸口。
<br/>　　这一名古帝实力的夜叉，在这一剑之下，当即被贯穿胸口，随即凌空爆炸，化为了一团血雾，被这一剑当即灭杀。
<br/>　　不止是这夜叉，随之而来的，还有数道剑光。
<br/>　　这每一道剑光，都准确的斩杀在一名魔族的胸口之上，将他们直接斩杀。
<br/>　　同时在那些魔族的背后，都是出现了一道漩涡，将这魔族的精血，尽数吞噬。
<br/>　　“什么人？”
<br/>　　那独魔皇一惊，就连他都没有想到，这个时候居然会有人出手相助。
<br/>　　尤其是楚风眠的到来，一出手，就直接斩杀了五位七劫古帝级别的魔族。
<br/>　　这些魔族的精血，全部都已经是被吞天秘术所吞噬，化为了神树的给养。
<br/>　　“有人来救我们了！”
<br/>　　“好强的剑光，悟剑堂的圣子，都没有这样的剑光，一剑斩杀古帝级别的魔族？这是什么人物？”
<br/>　　几名五灵宗圣子都瞪大了眼睛。
<br/>　　那四位魔族魔皇，可都没有出手，单单是凭借这些七劫古帝实力的魔族，都已经将他们逼入到绝境了。
<br/>　　同境界的魔族，几乎足以跟同境界的武者天才，相媲美。
<br/>　　而且拥有尊贵血脉，生命力强到吓人的地步，斩杀一名魔族，比起斩杀一名武者可是要难得多。
<br/>　　可如今一道剑光的到来，都足以斩杀一名古帝魔族。
<br/>　　“什么人？滚出来！”
<br/>　　看着这么多的魔族被斩杀，独魔皇当即忍不住了，咆哮一声，头顶之上的尖角开始颤动，一道红色光芒，凝聚而出，对准剑光到来的方向，轰击而来。
<br/>　　这一道红色光芒，便是独魔一族的特殊手段，毁灭之光，足以毁灭天地一切。

<br/>　　楚风眠看到这毁灭之光的到来，嘴角露出了几分冷笑，非但不退，反而是足以看见，一道身影，向着毁灭之光直接冲了过去。
<br/>　　“硬抗毁灭之光？”
<br/>　　“这人太自大了吧！独魔皇的毁灭之光，就连九劫古帝都不敢肉身硬抗！”
<br/>　　“他疯了不成？”
<br/>　　几名五灵宗圣子，见到这一幕，刚刚议论起来，话都没有说完，只见那毁灭之光，轰击在了楚风眠的身上。

<br/>　　“难道是一位九劫古帝？”
<br/>　　“九劫古帝，都未必做得到这一点！”
<br/>　　“可是此人身上的气息，并不强横，到底是怎么回事？”
<br/>　　许多五灵宗圣子也注意得到，在这一道身影上的天地之力，并不雄厚，甚至是就连古帝都不如。

<br/>　　一名五灵宗圣子，突然看清出身影的面孔，突然惊呼一声道。
<br/>　　“楚风眠？”
<br/>　　“怎么可能是他？”
<br/>　　听到这呼唤，一道道目光落在楚风眠的身上，脸上的惊讶之色更甚。
<br/>　　其中几名五灵宗圣子，都曾经在当初五灵殿的时候，见过楚风眠，对楚风眠并不陌生。
<br/>　　可他们本以为，来救他们的，至少也是一位九劫古帝才对，可没想到，居然是楚风眠，一个准帝。
