        崔素妮这回对答如流：“是用来防滑的！”
<br/>
<br/>        这回，她总没回答错吧！
<br/>
<br/>        而傅昼景却对她嗤之以鼻，铿锵有力地解释道：“诺筱颖在设计这套图纸的时候，曾经拿着这套图纸的初稿来请教过我。书柜的暗梯设计，是我给她的思路。她觉得暗梯的踏板光只设计防滑很单调，于是便在每个踏板上，按照人体足底穴位，设计了按摩点。所以，这些规律不齐的凹凸圆点，是按照针灸的方法，用来给足底按摩穴位的！崔素妮小姐，你还有什么话可说？”
<br/>
<br/>        崔素妮下意识地捂住嘴巴，顿时哑口无言。
<br/>
<br/>        傅昼景冷冷地盯着崔素妮，呵斥道：“设计图这种明摆着的东西，变着法子就可以抄袭到，但是，设计理念与设计师的心意，全在一个人的脑子里，是抄袭不到的！崔素妮小姐，我不知道你是用了什么法子去诬陷诺筱颖抄袭你。但是，事已至此，到底是谁抄袭谁，大家已经一目了然了。无需我多说了。接下来，这样的员工，要怎么处理。孙副部长，你应该无需我再教你了吧！”
<br/>
<br/>        “是，傅总！”孙莉莉额冒冷汗地颔首。
<br/>
<br/>        傅昼景不再二话，而是直接离开了会议室。
<br/>
<br/>        此时此刻，孙莉莉恼火不已地瞪着崔素妮。
<br/>
<br/>        在场的所有人，纷纷交头接耳，对崔素妮窃窃私语了起来。
<br/>
<br/>        大家都没想到，他们会误会诺筱颖，还给崔素妮这样的恶女人做了帮凶，那般驱逐诺筱颖。
<br/>
<br/>        就连孙莉莉的心里也对诺筱颖感到惭愧。
<br/>
<br/>        傅昼景离开后，崔素妮仍旧想替自己辩解，但是孙莉莉已经不再相信她了。
<br/>
<br/>        而接下来，崔素妮要面对的是比当初诺筱颖被驱赶还要严重的事情。
<br/>
<br/>        那就是，她被公司的法律部以涉嫌“剽窃与抄袭他人劳动成果”起诉了。
<br/>
<br/>        公司这么做，就是在杀鸡儆猴！
<br/>
<br/>        只可惜，诺筱颖看不到崔素妮这个坏女人的下场。
<br/>
<br/>        而与此同时，在医院里住了一个月的傅夜沉正式向上级提交了退役申请书。
<br/>
<br/>        那次绑劫爆炸案，傅夜沉用自己的身体护住了那个人质女孩季雅姗，自己却受了重伤，被掉下来的混凝土块砸到内出血。
<br/>
<br/>        而他们特种部队的作战小分队另外五个人也受了伤，有的虽然瘸了腿，但幸运的是，他们并无性命之忧。
<br/>
<br/>        只不过，那五个歹徒中却死了四个，只有一个因为脑部重创，至今躺在医院里昏迷不醒。
<br/>
<br/>        就连跟他一起死里逃生的战友韩剑锋不禁调侃：“四少，嫂子真是你的护身符！”
<br/>
<br/>        当初，烂尾楼坍塌后，如果不是傅夜沉左手腕上那条青丝手链将他挂在一根突出来的钢筋上，起了个缓冲作用，恐怕他和那女孩季雅姗早就被那些七倒八歪的混凝土块给埋没，砸成烂泥了。
<br/>
<br/>        傅夜沉会心一笑，脑海里不由自主地浮现出诺筱颖的样子：“当然！”
<br/>
<br/>        这个女人，真的令他又爱又愤。
<br/>
<br/>        没见面的时候，光听刘管家说的那些事情，他就气愤得脸红脖子粗。但一见面，他又对她有种“爱”不释手的感觉。
<br/>
<br/>        傅夜沉也无法说清楚，这是一种什么样的矛盾情感。
<br/>
<br/>        不过，现在离开特种部队，转战商界，最高兴的人，应该是他爷爷。
<br/>
<br/>        他重伤住院的这件事情，他是瞒着他爷爷的，就连弟弟傅昼景，他都没有告知。
<br/>
<br/>        “四少，那你打算什么时候正式跟嫂子领结婚证？”韩剑锋又问。
<br/>
<br/>        傅夜沉会心一笑：“拿下西镇度假山庄的那个项目后，等她从韩国旅游回来后，作为新婚礼送给她！”
<br/>
<br/>        因为，她的老家，是住在西镇。
<br/>
<br/>        他听刘管家说，他在医院住院的这段日子，苏漫雪中途有回来过，但只停留了一两天又跑去了韩国玩。
<br/>
<br/>        刘管家本想把他受了重伤的事情告诉苏漫雪，但是他不让。
<br/>
<br/>        他不想她替他担心，他只希望，她在韩国玩得开心就好。
<br/>
<br/>        只有她开心，他才会更开心。
<br/>
<br/>        而接下来的这段日子，诺筱颖一直被哥哥，哦，不，是表哥诺司峻安排相亲，只为早点把自己嫁出去。
<br/>
<br/>        与此同时，她还发现，童爸童妈对她总是客客气气的。
<br/>
<br/>        或许是因为彼此还不够熟悉的缘故，但是，诺筱颖又说不上哪里不对劲。
<br/>
<br/>        童妈偶尔还会念叨，想去梁家看看梦萌，童爸也只是默不吭声。
<br/>
<br/>        诺筱颖和他们相处，只有简单地几句语言，例如：爸妈，吃饭了；爸妈，我出去相亲了。
<br/>
<br/>        没错，自从她被冤枉诬陷，从公司被同事们赶出来后，她回到家的这些天，就一直与被哥哥介绍人的相亲。
<br/>
<br/>        她的梦想，她的理想，她的愿望……
<br/>
<br/>        一个个都无法实现！
<br/>
<br/>        诺筱颖每次在夜深人静的时候，总会无助地躲在被子里哭泣。
<br/>
<br/>        这日，一如既往，她又跟不同的男人在相亲。
<br/>
<br/>        “我身心干净，从未谈过恋爱，对婚姻忠诚。我懂勤俭持家，会洗衣服做饭。但我没车没房没工作没存款，而且还因为每月要偿还高利贷利息，所以结婚时的礼金最少都得二十万！这样的我……你愿意娶吗？”
<br/>
<br/>        诺筱颖对每一位来跟自己相亲的男士，都会如此坦诚地问这句话。
<br/>
<br/>        当然，得到的答复虽是五花八门，但大致的内容可以用“不愿意”这三个字来概括。
<br/>
<br/>        噢，对了！她现在已经不再叫“诺筱颖”了，而是叫童筱颖。
<br/>
<br/>        当然，表妹也不再叫“童梦萌”，而是叫梁梦萌。
<br/>
<br/>        童筱颖苦笑，只求上帝，原谅她如此现实地向“钱”看。
<br/>
<br/>        “叮咚，您有新的消息，请注意查收。”
<br/>
<br/>        一条短信息，将童筱颖的思绪拉了回来。
<br/>
<br/>        刚相完一个相亲对象以失败告终后，表哥诺司峻又发来了下一个相亲对象的手机号码。
<br/>
<br/>        童筱颖点开手机短信看了看，一如既往，她会主动拿起手机拨打号码过去，跟对方约见面的时间和地点。
<br/>
<br/>        是不是觉得她很“不要脸”？
<br/>
<br/>        是！
<br/>
<br/>        她是不要脸地想把自己早点嫁出去。
<br/>
<br/>        目标很简单，也很现实，就为了一个字——钱！
