　　毒龙山贼团，乃是破玄城外最臭名昭著的山贼团伙。
<br/>　　杀人越货，无所不作，要钱也要命。
<br/>　　偏偏，他们的势力极强，谁也奈何不了他们。
<br/>　　就算是一般的大型商队遇到他们，也得乖乖缴械投降，送出货物，以求活命。
<br/>　　故破玄城内的不少商贾和一些家族都对他们深恶痛绝。
<br/>　　曾有商贾和家族联合起来，连续两次高价在猎妖殿中发布过悬赏任务，但都以失败告终。
<br/>　　第一次，接任务的是一支破玄城颇负盛名的猎妖队，队长乃是先天九重武者，手下猎妖师数十人。
<br/>　　可却在清剿他们时，遭遇团灭，连那个先天九重的队长也没能逃出来。
<br/>　　第二次，赏金加高，三支当时在破玄城最强的猎妖队同时接任务。三个先天九重武者，近百名猎妖师联合出手。
<br/>　　可是，毒龙山贼团依靠山林之利，四处躲藏。硬是让这三支猎妖队无功而返。
<br/>　　之后，待得三只猎妖队无奈地去接其他任务，分散了之后，竟遭到他们一次次地报复，导致死伤惨重。
<br/>　　至此为止，已经有不下一千之数的猎妖师死在他们手上。
<br/>　　城内商贾们也是无奈，请的猎妖师少，不是他们的对手，请的猎妖师多，又会打草惊蛇，他们藏身深山老林，谁也奈何不了他们。
<br/>　　除非是洞玄境武者亲自出手，才能轻松灭掉他们。
<br/>　　可是，破玄城内就这么几个洞玄境，而且还是大家族的家主。他们自然不可能因为赏金而放下身份，去深山密林里对付一群山贼。
<br/>　　久而久之，毒龙山贼团仍旧盘踞着，再无猎妖师接这个任务。
<br/>　　当萧逸在猎妖殿看到这个任务时，毫不犹豫地接了下来，毕竟顺路，且又是四级任务，任务点足有30点。
<br/>　　猎妖殿的任务，一般因其难度而评出等级，也与武者的修为对应。
<br/>　　一级任务，凡境武者就能轻松完成。
<br/>　　三级任务，起码要先天境武者带领或者出手才能完成。
<br/>　　四级任务，一般需要洞玄境武者，或者多个先天境武者联手完成。
<br/>　　毫无疑问，对付毒龙山贼团，被列为了四级任务。
<br/>　　言归正传。
<br/>　　以萧逸的能力，直到潜入这伙山贼的老巢，仍旧没有人发现他的存在。
<br/>　　他就这般站在树上，淡淡地看着下方的山贼们饮酒作乐。
<br/>　　“再干几票，让大哥带兄弟们去放松放松吧。”一个山贼一口吃着肉，一边喝着酒，兴致勃勃地说道。
<br/>　　“啧啧，在山林里憋了那么久，好想回破玄城看看那相好。”
<br/>　　“切，那些风尘女子，有什么好想的。”
<br/>　　“我倒是知道，过几天又有商队经过，好像有不少女人。”
<br/>　　山贼们大口喝着酒，肆意地聊着下流之事。
<br/>　　忽然，一中年大汉龙行虎步地走到他们身前。
<br/>　　中年大汉，左脸有一刀疤，面容十分凶恶，他就是这伙山贼的首领，毒龙。
<br/>　　“大哥...大哥...”山贼们纷纷行礼。
<br/>　　“别太放松了。”毒龙沉声吩咐道，“我听说前阵子狂血玄君的洞府被发现了，很多猎妖师都在里面得到宝物，实力大增，可别出什么幺蛾子。”
<br/>　　“放心吧大哥，那些猎妖师还有商队护卫，在我们眼中，就是一头头肥羊罢了。”
<br/>　　“就是，一刀将他们杀了，宝物抢过来就是。”
<br/>　　山贼们恶狠狠地说着。
<br/>　　毒龙刚想呵斥几句，忽然，他耳朵一动，猛地转过身，对着不远处一颗大树怒喝一声，“谁？”
<br/>　　树上，萧逸一跃而下。
<br/>　　“不愧是毒龙，有些本事。”萧逸淡淡地说道。
<br/>　　“嗯？”毒龙双眼一眯，冷冷地盯着萧逸，“一个少年，竟然有本事无声无息潜到这里来，你也不错。”
<br/>　　幽魂面具并非是全覆盖式的面具，仅仅是遮盖眼睛之下的部分。
<br/>　　眼睛以及眼睛之上的部分却没有遮盖。
<br/>　　那种全覆盖式的面具，整张脸除了眼睛外，便全遮盖住了。这会遮到武者的眼角部分，使武者产生视觉盲区，极不利于战斗。
<br/>　　反倒是半覆盖式的，既能起遮盖样貌的作用，又不影响武者战斗，更为方便。
<br/>　　所以炎龙大陆上，大多是半覆盖式的面具，幽魂面具也一样。
<br/>　　毒龙看着萧逸那清澈的眼睛，还有白哲清秀的眉额，几乎已经能断定萧逸的年龄不大。
<br/>　　再听到萧逸的声音，已经可以确定萧逸是个少年。
<br/>　　当然，他也对一个少年胆敢闯入他的地盘而感到不可思议。
<br/>　　“你是何人？”毒龙警惕地问道，“破玄城中，没有你这样一号人物。”
<br/>　　破玄城中，除了那几位洞玄境外，有本事闯入他地盘的先天境不出十人。可这些人中，绝对没有一个少年。
<br/>　　“你就是毒龙吧。”萧逸没有回答，而是反问一声。
<br/>　　毒龙皱起了眉头，“明知故问，你到底想做什么。”
<br/>　　萧逸淡淡一笑，“没什么，只是想问清楚而已，免得杀错了人。”
<br/>　　毒龙脸色一冷，“你是猎妖师。”
<br/>　　“猎妖师？”周围的山贼也是脸色一冷，而后满脸戏谑之色，“又来一个送死的。”
<br/>　　“呵呵，这些年都不知道有多少猎妖师死在我们手上了。”
<br/>　　“破玄城的家族这次是出了多高的价格阿，竟连一个毛都没长齐的小子也敢来找我们麻烦。”
<br/>　　山贼们哄笑着，显然没将萧逸放在眼中。
<br/>　　“给我拿下。”毒龙冷喝一声。
<br/>　　十数山贼一拥而上，手中锃亮的大刀仿佛在告诉萧逸，小子，最好别反抗，否则一刀要你命。
<br/>　　“哼。”萧逸冷哼一声，不见他有什么动作，但那十数山贼却在一瞬间火焰加身。
<br/>　　轰，火焰爆炸，十数山贼甚至连反应过来的机会都没有便被炸成重伤，倒在地上，生死不知。
<br/>　　毒龙冷笑一声，“小子，别以为有些本事就可以来我这撒野。你这种不知天高地厚的家伙，我一年不知杀多少个。”
<br/>　　说罢，毒龙大手一挥，大喝道，“给我将这个臭小子射成蜂窝。”
<br/>　　嗖嗖嗖....
<br/>　　密布的林间中，百多个弓箭手应声而起，片刻之后，越来越多的弓箭手出现，观其数量，竟在五百人以上。
<br/>　　萧逸眉头一皱，看了眼其他山贼。
<br/>　　这时，竟又有不下数百人呈包围之势向这边靠拢。
<br/>　　萧逸微微一惊，“原来如此，竟有上千个山贼，而并非是传言中的数百。单单在外围警戒的弓箭手就有数百，难怪其他猎妖师数次对你们的大规模围剿都被你们事先发现被躲起来了。”
<br/>　　“而小规模的猎妖队，在你们的人数优势面前，简直就是送死。”
<br/>　　“哼。”毒龙冷笑一声，“小子，现在跪地求饶，我或许可以让你死得痛快些。”
<br/>　　“求饶？”萧逸怡然不惧，道，“毒龙，你听说过火烧连营吗？”
<br/>　　“嗯？”毒龙一惊，瞬间反应过来，大喝道，“放箭，杀了这小子。”
<br/>　　他的话语刚落，无数根箭矢飞速射向萧逸。
<br/>　　但同一时间，萧逸也是大手一挥，漫天火焰澎湃而出，席卷密林中的弓箭手。几乎只在一瞬间，整个密林化为一片火海。
<br/>　　绿色武魂的控火能力加上自身澎湃的真气，足以释放数量惊人的火焰。
<br/>　　这时，数道噬火百刃打出。
<br/>　　密集的箭矢还未接近萧逸，便已经被火焰和爆炸所吞噬。
<br/>　　就算有零丁几只箭矢能穿过火焰，但打在萧逸的真气外衣上，却毫无作用。
<br/>　　“真气外衣？”毒龙猛地一惊，“小小年纪，竟已经凝聚出真气外衣了，此子留不得。”
<br/>　　同时，当他看到火海中的弓箭手死伤惨重时，眼中的杀意更为浓郁。
<br/>　　那些弓箭手顶天了就凡境武者，在萧逸的火海中，只是秒杀的份。
<br/>　　“给我擒下这小子，我要将他拨皮抽筋。”
<br/>　　毒龙怒喝一声，身后五个武者瞬间袭来。
<br/>　　“两个先天六重，三个先天四重，你手下竟还有这么多强者，难怪这么多年来没有一支猎妖队奈何得了你们。”
<br/>　　萧逸脸色不变，手中凝聚出一股火焰。
