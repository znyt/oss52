　　第412章
<br/>　　挖掘工作还在进行中，宋伯康守在外面等待，大批士兵向此处集结，以求尽快把李行哉挖出来。
<br/>　　对于李行哉，他不想杀，也不能杀。但对于抢走李行哉的人，宋伯康觉得一定要让他们知道什么叫酷刑。
<br/>　　沙沙！
<br/>　　这是铁锹掘土的声音，在黑夜里响个不停，感觉像一群蚂蚁在吞噬一头野猪。宋伯康不由得紧张起来，但他也无法分辨这种情绪从何而来。伸手将额头的冷汗擦掉，他问道：“怎么，还没挖出来么！”
<br/>　　“大人，大人……”
<br/>　　手下的声音结结巴巴，这让李行哉觉得，似乎将有不好的事情发生。
<br/>　　“怎么了？”
<br/>　　“掘，掘到头了！”
<br/>　　“什么！”
<br/>　　宋伯康有些不敢相信，但潜意识中已接受了这事实。他快步走过去，果然发现地道在接近城墙时消失，眼前是坚硬的墙砖。
<br/>　　瞬间，宋伯康瞬间便意识到发生了什么。这条地道是假的，本就是为了吸引自己的注意力，而目的是让李行哉从真正的隧道逃生。
<br/>　　眼前是铁一般的事实，甚至在地道尽头，还用碳灰画着一只丑陋的蛤蟆。果然是蛤蟆城的作风，宋伯康怀疑，今天程大雷甚至会亲自出席在此地。
<br/>　　“快，找，找到真正的地道！”宋伯康大叫。
<br/>　　于此同时，李行哉终于爬出了地洞，眼前是一望无垠的荒原，头顶繁星漫天，李行哉跌在地上，大口大口的喘息，任冰冷的空气渗入肺中，平生第一次，他感觉仅仅是呼吸也是如此愉悦的一件事。
<br/>　　“喂，走啦，再不走就走不脱了。”白元飞拍拍他的肩膀。
<br/>　　“程大雷呢，程大雷在哪里？”
<br/>　　“嘿，大当家什么身份，这样的任务，还用他来。”白元飞不屑道。
<br/>　　事实上，白元飞只是嘴上痛快痛快，今天程大雷还真到了。只是，如果在凉州城附近出现，目标太大，容易被人发觉。所以今天程大雷只带了三千弟兄，在远处埋伏接应，只等白元飞把人带回去，大家就一起返回蛤蟆城。
<br/>　　白元飞带着李行哉往前走，从树林中牵出三匹事先藏好的战马。留给三人的时间并不多，那伪装的地道想也知道坚持不了多久，一旦被人发现，三人想走怕就来不及了。
<br/>　　三人刚走出树林，就和一队凉州兵正面碰上。对方一共五人，都骑在马上。
<br/>　　“什么人？”为首的伍长大声道。
<br/>　　“哈哈，我们是宋大人的队伍，大家自己人不认自己人了么。”白元飞马上道。
<br/>　　对方伍长一楞，只见三人渐渐靠近，立刻认出了白元飞。
<br/>　　“是抢走殿下的敌人，快去通知州牧大人！”
<br/>　　随之一柄扑面而来的匕首射入了他的喉咙，白元飞从马上跃起，扑向下一个敌人。
<br/>　　如此出奇手段，的确震惊了这支五人队伍，短时间，白元飞已经击杀三人，可还是被两个人逃跑。
<br/>　　事实上，不管是白元飞还是王德，都不是沙场厮杀的战将，他们一个飞贼，一个盗墓贼，能在江湖上活这么久，本事还都是有的，只不过不强。
<br/>　　假若是秦蛮、赵子龙在此地，碰上一支百人伍的队伍，也自信能够带人脱身。可二人都没有秦蛮那样的本事，真要和大军碰上，三人必死无疑。
<br/>　　“走喽！”
<br/>　　白元飞收回匕首，狠狠一鞭子抽在马身上。
<br/>　　三匹马在黑夜中狂奔，逃向蛤蟆城方向。此时，程大雷正带着手下弟兄，于山坳中等待。
<br/>　　程大雷身边的人是赵子龙，千军万马七进七出，如入无人之境，程大雷自然也希望他能在今天发挥些作用。
<br/>　　而手下带领的弟兄，都是赵子龙麾下的龙字队，龙字队完全由骑兵组成，是一支轻骑兵队伍，正适合这样的任务。
<br/>　　今天的任务并不轻松，程大雷的目的是接到人就走，最好不要和凉州兵发生正面冲突。当然，如果真正厮杀起来，程大雷也顾不得许多，自从得到黑牛后，程大雷还没有领着它上过战场。
<br/>　　“大当家，要不要我去看看，那边情况怎样了？”赵子龙。
<br/>　　时间一分一秒过去，拖得越久，证明白元飞那边的情况就越不顺利。
<br/>　　“不，我们还是等在这里吧，静观其变。”程大雷看了赵子龙一眼，道：“你现在也指挥着三千人，不要动不动就亲自出马，这不是你该做的事情。”
<br/>　　赵子龙把头垂了下去，但程大雷估计，下次的时候他还是会冲锋陷阵，毕竟江山易改禀性难移。其实蛤蟆城的五虎上将，大多是这种脾气，能动手的时候绝对不动脑子。
<br/>　　这时候，宋伯康刚刚得到李行哉的行踪。有手下纵马疾奔而来，气喘吁吁道：“大人，大人，找到殿下了，在城西方向。”
<br/>　　“找到了么！”宋伯康迫不及待道：“快追，今天一定要救回殿下，哪怕是……”宋伯康将牙一咬，口中吐出三个字：“杀了他。”
<br/>　　不能得到时，就亲手毁了他，宋伯康并不缺乏这方面的残忍。现在他心里已经对李行哉起了杀心。
<br/>　　大军向城西集结，宋伯康一马当先已率先追了出去。其实不成阵型，只要先赶到的，就第一时间追了出去。
<br/>　　李行哉三人纵马狂奔，只听闻身后的马蹄声越来越近，回头看一眼，都是黑压压的骑兵。
<br/>　　“殿下，留下吧，留下吧，哪儿干不是干呐！”
<br/>　　凉州兵一声又一声呼着，可在李行哉听来，却如同魔鬼的低吟一般。
<br/>　　老实说，在凉州城他待遇不错，要女人有女人，要美食有美食，许多愿望都可以在凉州城满足。但这两样东西之所以珍贵，是源于他们是自己亲手夺得的，而不是别人恩赐的。
<br/>　　况且，凉州城给不了他最重要的一件东西：
<br/>　　他要去长安，去解长安之围。
<br/>　　有斥候飞奔回来，道：“大当家，殿下救出来了。”
<br/>　　“是嘛，这可再好没有。”
<br/>　　“只是他们身后追着许多凉州兵，人数至少上万。”
<br/>　　“啊！”
<br/>　　程大雷一张脸瞬间变得锅底一样黑。
<br/>(本章完)
