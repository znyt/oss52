<br/>　　八十万里！

<br/>　　这龙脉内蕴含的灵气，简直是如同汪洋大海，无法形容了。

<br/>　　当然，这只是他的幻想而已，若是真的吞噬了，最有可能的是直接爆体而亡。
<br/>　　苏莫刚一接近龙脉，巨大的白龙顿时睁开了有些虚幻的眼眸，转头看了苏莫一眼。
<br/>　　“不好！”苏莫心中一惊，毫不犹豫，立刻就催动了吞噬战魂。
<br/>　　强大的吞噬之力，顿时横扫而出，瞬间便将龙脉笼罩住了。
<br/>　　昂！

<br/>　　“给我吞！”苏莫将吞噬战魂催动到了极致，并且用吞噬奥义加持，想要将龙脉吞噬擒拿。
<br/>　　唰！
<br/>　　然后，只见白龙的龙尾一甩，瞬间震散了吞噬之力，唰一下消失无踪了，速度快到肉眼难辨。
