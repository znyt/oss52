        “嗯？”
<br/>
<br/>        “刘婶请假了。”
<br/>
<br/>        “嗯。”
<br/>
<br/>        “家里的家务，谁做呢？”
<br/>
<br/>        “我会请钟点工过来。”
<br/>
<br/>        “可我不喜欢家里有外人了。”
<br/>
<br/>        “那我来做。”
<br/>
<br/>        “那你还不快去换衣服！”画风忽然一变，童筱颖玉腿一跨，从他身上下来，站在床边怀抱着双臂，像个女王一样发号施令，还滔滔不绝起来，“吃过早餐后，桌子要抹干净，碗筷收入厨房的洗碗机，灶台也要抹干净，打开扫地机器人，让它开始扫地，还有，家里的玻璃窗也要抹，浴缸和马桶都要洗……”
<br/>
<br/>        “好，全都没问题。”傅夜沉意味深长地盯着童筱颖，黑曜石般的眸子里，彷佛刮起了一阵暧昧的龙卷风，像是要将她直勾勾地卷入他的漩涡里。
<br/>
<br/>        “那……你还愣着做什么？”童筱颖心头突然生起有一种怪怪的错觉，总觉得自己这是在玩火自焚，“你先换衣服吧！我下去了！”
<br/>
<br/>        语毕，她想逃，如白玉般的手腕却被傅夜沉给紧紧地抓住。
<br/>
<br/>        童筱颖微蹙着柳眉，本能地侧过身时，嘴里接着说了一个“你”字，下一秒，一道黑影防不胜防地凑了过来。
<br/>
<br/>        “在做家务之前，我想先做你。”话音刚落，傅夜沉的唇印在她的眼睛上。
<br/>
<br/>        童筱颖瞠目结舌地看着傅夜沉凑近的俊脸，刚刚平复的心情，此刻又突然泛起了涟漪。
<br/>
<br/>        他温热的唇瓣从她的眼睛缓缓地移动到她的眼角。
<br/>
<br/>        他的唇很柔、很软，轻轻的、细细的、柔柔地贴在她的眼角时以一种极其缓慢的方式蹭着她，与此同时，他的唇也很烫，不断地磨着她，害她也跟着发热，脸颊飞入两朵红晕。
<br/>
<br/>        傅夜沉呼出的气息如热浪般一波一波地袭向她，就好像要将她淹没在他的浪潮之下，他的手抚着她的背，一下一下，宛若在她的脊椎骨那里弹奏着。
<br/>
<br/>        童筱颖感觉头有点晕，注意到他的睫毛比她还要长，一颤一颤的，好像一只飞舞的彩蝶。
<br/>
<br/>        她的呼吸变得紊乱，她微微眯着眼睛，双颊通红。
<br/>
<br/>        傅夜沉退开一小步，薄唇对着她轻语：“宝贝，如果你不是真的愿意，就不要试图在我身上点火，否则，后果会很严重。”
<br/>
<br/>        心跳好快好快，比坐云霄飞车还要快，童筱颖半眯着眼睛打量着这个男人，双手不知哪里来的力量，她推开他。
<br/>
<br/>        沉默就如杯中的茶叶，一叶一叶地在水中绽开舞姿。
<br/>
<br/>        “你……”童筱颖愣着，她不知道该说什么。
<br/>
<br/>        他居然看出来，她是在故意引诱他！
<br/>
<br/>        傅夜沉忽然伸出手，摸摸她的头：“早安。”
<br/>
<br/>        “早安。”童筱颖傻傻地重复。
<br/>
<br/>        傅夜沉的手伸向她，以迅雷不及掩耳之势在她的额上印下一吻，随即说：“今天我在家，做一天的家务，直到我的宝贝，满意开心为止。”
<br/>
<br/>        不对，好像有什么东西脱序了，童筱颖感觉自己的身体一阵一阵的火热，而这种火热很奇怪，好难形容。
<br/>
<br/>        她微微喘气，隐约明白他的吻挑起了她要作为女人时的觉醒。
<br/>
<br/>        她和昼景之间，也有过一段她以为是的爱情，但相比之下，却与大哥傅夜沉在一起时的感觉完全不同。
<br/>
<br/>        昼景给她一种柔情似水的温情，而大哥给她的却是一种情难自控的激情。
<br/>
<br/>        她像是要确定什么，小手扯了扯他的衣角，娇羞地对他一笑：“大哥，你为什么要对我这么好？”
<br/>
<br/>        傅夜沉的黑眸就如星光般闪烁，他的手抚着她的脸，轻轻呢喃：“因为，你是我的唯一。”
<br/>
<br/>        童筱颖明明看见傅夜沉的眼底蛰伏着某种异样的情绪，但他柔和的表情让她彻底为他心花怒放了。
<br/>
<br/>        好像有什么东西，在尘封已久的心田破茧而出，她不知道是什么。
<br/>
<br/>        她只是一头扎入他的怀中，紧紧地搂着他，感受着这个男人带给她的霸道宠爱。
<br/>
