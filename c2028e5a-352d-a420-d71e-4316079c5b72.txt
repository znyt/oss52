        “不要——不要——昼景救我！救我啊！”童筱颖忽然一声呓语，将傅昼景吓了一跳。
<br/>
<br/>        傅昼景连忙捧住童筱颖那双在半空中乱挥舞的手，紧紧地捂在自己的胸口上，担忧地唤了一声：“筱颖，不要怕，我在这里！”
<br/>
<br/>        童筱颖安分下来后，缓缓地睁开了眼帘。
<br/>
<br/>        就在她侧过头来，看到身边坐着的人是她心心念念惦记的男人后，立即坐起身来，扑入了他的怀中。
<br/>
<br/>        傅昼景欣慰地笑了笑，一边将童筱颖搂入怀里，一边轻轻地拍抚着她的背脊，安慰道：“筱颖做噩梦了也不要怕！有我在！我一直都在！”
<br/>
<br/>        而此时此刻，童筱颖却躲在他怀里，失声痛哭地道歉：“对不起！对不起！对不起！”
<br/>
<br/>        “筱颖这是怎么呢？”傅昼景心疼地皱起了眉头。
<br/>
<br/>        童筱颖这才意识到自己的失态，立即闭上了嘴，只是“呜呜”地抽泣着。
<br/>
<br/>        她不能告诉他，她被那个她所救过的一男人给强吻了，还差点被那个男人给强上。
<br/>
<br/>        这样的痛苦，她只能一个人默默地承受着。
<br/>
<br/>        她这样的遭遇，算不算是被人性侵未遂？
<br/>
<br/>        内心的无助，她又能向谁诉说？
<br/>
<br/>        傅昼景见童筱颖不吭声，便不再追问，只是静静地抱着她，任由她在自己怀里哭泣。
<br/>
<br/>        或许，就这样让她痛哭出来后，她的心情就会好起来。
<br/>
<br/>        傅昼景陪着童筱颖在医院里待了一夜，第二天早上，她总算是退了烧。
<br/>
<br/>        医生给童筱颖开了口服药，当天便可以出院了。
<br/>
<br/>        傅昼景带着童筱颖回到家后并未去上班，而是一直留在家中陪着童筱颖。
<br/>
<br/>        童筱颖自知他手里一定有一大堆的项目要做，故作坚强地告诉傅昼景，她一个人可以好好照顾自己。
<br/>
<br/>        但傅昼景没有依她，而是继续留在家里给她做饭做家务。
<br/>
<br/>        这样的好男人，天下少有！
<br/>
<br/>        正当童筱颖感到欣慰又幸福，觉得自己像是在做梦一样的时候，赵斌一个电话，还真就打破了她的这场美梦。
<br/>
<br/>        傅昼景一而再再而三地推辞后，发现这个工作推不掉后，一脸无奈地看着童筱颖。
<br/>
<br/>        童筱颖正盘腿坐在床上，手里端着他刚给她熬好的鱼片粥。
<br/>
<br/>        看到傅昼景薄唇紧抿，童筱颖咧嘴一笑：“你去工作吧！我真的不要紧！而且，我也不发烧了。我会记得按时吃药的！”
<br/>
<br/>        “我得出差。”傅昼景深沉地说道。
<br/>
<br/>        童筱颖下意识地问：“要去多久？”
<br/>
<br/>        “一个星期。”傅昼景微微耸了耸肩。
<br/>
<br/>        童筱颖虽然心有不舍，却仍旧强颜欢笑地释然道：“嗯，不久。我等你回来！”
<br/>
<br/>        “我还是等你病好了再去吧！”傅昼景莞尔一笑，刚准备拿起手机打电话给赵斌时，手机便被童筱颖给夺了去。
<br/>
<br/>        “我都这么大了，又不是小孩子。你放心去工作吧！不要紧的！再说了，我还有哥哥嫂嫂住楼上啊！他们虽然不是我的亲哥哥亲嫂嫂，但我哥，怎么着也是看着我长大的。不会不管我的！”童筱颖微笑着说道。
<br/>
<br/>        傅昼景犹豫了一下，还是点了点头，并抬起手来托住童筱颖的后脑勺，凑过去亲了亲她的额头：“等我回来。”
<br/>
<br/>        “嗯！”童筱颖乖巧地点了点头。
<br/>
<br/>        下午，傅昼景给童筱颖做好了晚饭，才收拾了行李，出了门。
<br/>
<br/>        而当晚，童筱颖也跟唐倩兮和“夜惑”的大老板费尔南请了假。
<br/>
<br/>        唐倩兮是批准了，但那位坐着收钱的大老板费尔南似乎不太乐意。
<br/>
<br/>        不过，庆幸的是有唐倩兮在一旁帮腔，童筱颖还是请到假了。
<br/>
<br/>        今晚童筱颖不来驻唱，打赏的那份收入也少了一大半。
<br/>
<br/>        傅夜沉并不知道“小谢”今晚请了假，所以他一个人在卡座里坐了许久，见“小谢”仍旧未登台，准备起身离开的时候，却被一行人给堵住了去路。
<br/>
<br/>        季运恒带着自己的这帮狐朋狗友，傲气十足地扬起下巴，拦在傅夜沉的面前。
<br/>
<br/>        这个男人是什么来路，他压根就没摸清楚，只知道对方叫“连少”，看样子也是个有钱的主儿，不然每次给“小谢”打赏，他都比他多出一倍。
<br/>
<br/>        很明显，他这是故意在跟他季大少爷找茬。
<br/>
<br/>        “兄弟，本少看上的女人，你每次都出来插一手，是几个意思？”季运恒抖着身子，吊儿郎当地围着傅夜沉转了一圈，并上上下下将他打量了一番。
<br/>
<br/>        这男人一米九的高个子，宽肩硕胸，身段完美。脸庞光洁，透着棱角分明的冷峻，似是一件名家精雕的艺术品。
<br/>
<br/>        而他身着的是现今最时尚的黑色西装，仍掩不住他天生的贵族气质。
<br/>
<br/>        看样子，这男人大有来头！
<br/>
<br/>        他手下的人说，这个男人的身手不错，估计以前也是在道上混的。
<br/>
<br/>        季运恒查不到这男人的底，暗地里又跟他较量了两次，现在只得跟他明着来了。
<br/>
<br/>        “小谢也是我看上的女人，你觉得，我能对我女人的事情袖手旁观？”傅夜沉黑眸微眯，深邃的眼眸里透着无尽的冷气，像似会把人给吸进去一样。
<br/>
<br/>        季运恒忽然一屁股坐在傅夜沉身后的卡座里，双手抱臂，翘着二郎腿，打算跟傅夜沉摊牌：“小谢这个女人我要定了！你开个价，你要怎样才能放弃小谢？再说了，这‘夜惑’里最不缺的就是美女。你争谁不好，跟我争小谢！更何况，小谢是本少先看上的！”
<br/>
<br/>        “所以，你这是打不过我，就来跟我拿钱谈判？”傅夜沉嗤之以鼻，剑眉微扬。
<br/>
<br/>        被说中了心思的季运恒放下腿来，从卡座里站了起来，大步流星地走到傅夜沉的跟前，接着问道：“你开个价！”
<br/>
<br/>        “呵！”傅夜沉不禁冷哼。
<br/>
<br/>        这男人把“小谢”当什么呢？
<br/>
<br/>        有价值物品？
<br/>
<br/>        可以放到秤上称斤按量地贩卖？
<br/>
<br/>        季运恒见这男人如此轻视自己，不禁十分不爽地怒吼：“别敬酒不吃吃罚酒！”
<br/>
<br/>        “又想打架吗？你可以再试试！”傅夜沉邪肆地勾唇一笑。
<br/>
<br/>        却让季运恒不寒而栗。
<br/>
<br/>        本来他是想带自己这帮兄弟来压压他气焰的，哪知道这男人完全不把他放眼里，真是太可恶了！
