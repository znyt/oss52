　　修罗殿总殿，闭关室内。
<br/>　　选好功法，萧逸便来了此地。
<br/>　　在周遭布下屏障，萧逸盘膝坐下。
<br/>　　看了眼手中功法，萧逸面露喜意。
<br/>　　能不必参加传承考核，直接便得到功法，自然是极好的。
<br/>　　虽然萧逸并不担心传承能难倒他，但能省下一月时间，这对他而言是再好不过的事了。
<br/>　　“帝岳不灭体，总殿最强体修功法之一。”
<br/>　　萧逸自语一声，翻开了功法。
<br/>　　之前那一本本功法武技，他只是看个大概。
<br/>　　特地选这一本，自然是因为这是其中最强的一门炼体功法之一。
<br/>　　帝岳不灭体，土属性炼体功法。
<br/>　　炼至小成，身躯坚如山岳；炼至巅峰圆满，千万山河，一拳碎尽；山川河岳，以之为帝。
<br/>　　此门炼体功法，堪称防御最强，力气最强。
<br/>　　另外，萧逸选这门功法的原因，除了这是总殿的最强炼体功法外，还有一个原因。
<br/>　　那便是他把自己的云生铠甲给烧了。
<br/>　　没了这件中品防御圣器，他自然要修些护身手段。
<br/>　　不过，说起来总殿主让他不必参加传承考核，倒也是他的意料之外。
<br/>　　不必参加传承考核，直接便从总殿所有传承，所有炼体功法中，任他挑选。
<br/>　　这样的待遇，萧逸也不禁感到惊讶。
<br/>　　或许，是修罗殿的人，都比较爽快吧。
<br/>　　毕竟，他在风刹殿已经过了传承考核。
<br/>　　而修罗殿与风刹殿乃是联盟殿，两殿之间，自有卷宗情报间的互通。
<br/>　　“算了。”萧逸摇摇头，没有多想下去。
<br/>　　翻开功法，萧逸细细看着，也认真参悟着。
<br/>　　数个时辰后，整本功法，萧逸已参悟一小半。
<br/>　　他的眼眸，不知何时起，缓缓闭上。
<br/>　　他的身体周遭，亦不知何时起，一缕缕深黄色的氤氲之气流转其中。
<br/>　　这些，便是修炼帝岳不灭体所产生的肉体之力。
<br/>　　炼体功法，虽修的是肉体。
<br/>　　但其本身，也会衍生出属于肉体的特殊力量。
<br/>　　如以前的修罗战体，修炼之，除了肉身惊人外，还衍生了修罗之力。
<br/>　　修罗之力，便是施展修罗变与崩界拳等体修武技的力量。
<br/>　　“呼。”
<br/>　　一个时辰后。
<br/>　　萧逸轻呼出一口气，眼眸睁开的瞬间，周遭流转的氤氲之气就此消散。
<br/>　　“勉强算小成了。”
<br/>　　萧逸自语一声，脸色开始变得凝重。
<br/>　　以他的天赋，要炼至小成，并不难。
<br/>　　但帝岳不灭体，这等绝世体修功法，岂会简单。
<br/>　　小成，不过是个开始。
<br/>　　真正的难度，现在才算出现。
<br/>　　萧逸翻阅着功法，这里，已接近书籍的中间部分。
<br/>　　同时，这里开始，出现了一门武技。
<br/>　　一份真正的传承，绝不仅仅是只有功法。
<br/>　　与之对应的，还有武技，以及属于这位前辈的感悟等等。
<br/>　　现今这本功法书籍，虽非传承，但绝不亚于传承。
<br/>　　“灭墟指。”
<br/>　　这是帝岳不灭体上的武技。
<br/>　　同时，书籍中，渐渐出现了一些总殿先辈强者的批阅与感悟。
<br/>　　这是前人的理解与感悟，供萧逸参悟所用。
<br/>　　“好厉害的武技。”萧逸眼中闪过一丝惊色。
<br/>　　单单他对这门灭墟指的现今理解来看，其程度和威力，就远远超过了星幻剑阵等剑技。
<br/>　　就他现今拥有的手段和武技来看。
<br/>　　唯独从破晓钟上悟得的感悟，能与之媲美。
<br/>　　其余的，星幻剑阵、血界斩等武技，远远不如。
<br/>　　当然，血界斩第四式他还未掌控，这一点，倒是还无法下定论。
<br/>　　但毫无疑问，帝岳不灭体上的这门武技，强悍得惊人。
<br/>　　萧逸参悟着。
<br/>　　整整数个时辰后，眼眸睁开，却是摇了摇头。
<br/>　　“看来是对整门功法的理解还不够，强行参悟其内武技，太勉强了。”
<br/>　　现今，他对帝岳不灭体的修炼，不过是刚刚开始。
<br/>　　单凭此，便想掌控里头的强悍武技？
<br/>　　不说痴人说梦，但起码是强行为之，自讨苦吃。
<br/>　　萧逸暂时未理会灭墟指，转而继续参悟里头功法。
<br/>　　又是一个时辰后。
<br/>　　萧逸皱了皱眉。
<br/>　　萦绕在他周身的深黄色氤氲气息再次凝聚，却又忽然溃散。
<br/>　　“不愧是总殿的最强体修功法之一。”萧逸微微皱眉。
<br/>　　小成之后，渐渐开始修炼中间部分的功法，萧逸明显感到一股后劲不足的味道。
<br/>　　体修功法，同样需要修炼之物。
<br/>　　帝岳不灭体也不例外。
<br/>　　且，其需要的修炼之物，乃是一个极其庞大的天文数字。
<br/>　　它是土属性功法，所需的修炼之物，自然也是蕴含庞大土属性力量的重宝。
<br/>　　萧逸的目光，看向了闭关室一旁。
<br/>　　那里，放置着一枚枚乾坤戒，不下百枚。
<br/>　　萧逸手中功法书籍挥手而出。
<br/>　　功法靠近一枚枚乾坤戒。
<br/>　　哗…一阵声响，百枚乾坤戒，凭空消失，只剩一枚。
<br/>　　剩下的一枚，便是与功法对应的修炼之物。
<br/>　　乃是总殿馈赠。
<br/>　　萧逸大手一挥，将功法与乾坤戒悉数吸摄而回。
<br/>　　放出感知，感知了一下乾坤戒之内。
<br/>　　下一秒，萧逸就此愣住。
<br/>　　“修罗太荒石。”萧逸的脸庞，抽了抽。
<br/>　　修罗太荒石，修罗殿特有。
<br/>　　传言，这种石头尽吸上古太荒之时的大地至纯之气，珍贵无比。
<br/>　　时至今日，早已全无。
<br/>　　唯修罗殿从上古之时，便开辟秘境，以修罗殿阵法困之；时隔千万年，方有了一块专产此石的宝地。
<br/>　　修罗太荒石，其性质，等同灵脉灵石。
<br/>　　不过，灵石之内，是天地间的精纯灵气。
<br/>　　修罗太荒石之内，则是精纯的大地土属性力量。
<br/>　　一块修罗太荒石蕴含之力量，不亚于一条灵脉。
<br/>　　而此时，乾坤戒内，整整有20块。
<br/>　　据他所知，即便是修罗殿的那块宝地，也产量极少。
<br/>　　故这些修罗太荒石，每一块，都珍贵无比。
<br/>　　萧逸再次感知了一下。
<br/>　　乾坤戒内，除了20块修罗太荒石外，还有大量土属性天材地宝。
<br/>　　这些修炼之物，若他悉数吸收之，以修炼帝岳不灭体，怕是修炼至圣皇境层次都足够了。
<br/>　　“修罗殿，好大的手笔。”萧逸面露惊骇之色。
<br/>　　萧逸皱了皱眉，按捺下心头惊骇，继续参悟功法。
<br/>　　数个时辰后，他的修炼，戛然而止。
<br/>　　原本打算吸收这些修罗太荒石，以作肉体力量突破的想法，亦忽然消散。
<br/>　　原本皱眉的脸色，忽然化作了一丝不可置信。
<br/>　　帝岳不灭体，修炼至现今这个地步，里头功法意味，竟让他莫名感到一股熟悉。
<br/>　　“怎么可能。”萧逸双眼一眯，紧紧盯着功法上的内容。
<br/>　　“这是修罗战体第七层巅峰上的内容…”
