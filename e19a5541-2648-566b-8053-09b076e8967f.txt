　　叶少阳缓缓点头，“说的是，我之前还怀疑她是假装不信，照你这分析，的确，除非找到她失去的记忆，否我看很难让她相信我们的话。”
<br/>　　“办法也是有的。给她刺激，让她先否定自己的记忆，然后才能相信我们的话啊。”
<br/>　　碧清露出邪恶的笑容，“我倒是有个办法，能找到一个强有力的证据，让她无法反驳。”
<br/>　　叶少阳被勾起了兴趣，忙问她是什么办法。
<br/>　　“你看她今年多大？”
<br/>　　“她……二十多？不不，应该就二十出头。”叶少阳回想了一下，因为她是自己“嫂子”，并且有孩子的缘故，乍一想感觉她比自己要大，而且她作为长嫂，谈吐举止也都比较成熟，不过，当认真思索她的容貌时，叶少阳才感觉她实际年龄应该更小一点，可能比自己还小。
<br/>　　“你问她年龄干什么？”
<br/>　　“她最多二十出头，在现实世界，这个年纪应该大学还没毕业吧，一般来说，应该还是处女吧，当然如果不是吗，那就没办法了。”
<br/>　　叶少阳做出笑哭的表情，“怎么讨论到这个了。”
<br/>　　“呆子，她如果是处女，怎么可能生出儿子来！”
<br/>　　叶少阳一下子怔住。
<br/>　　“卧槽，你这办法，简直诛心啊！”叶少阳感叹，的确，如果陈悦还是处，那就不可能有瓜瓜这个儿子，等于就是证明了他的记忆是假的。
<br/>　　真的没有比这更直接更诛心的证据了。
<br/>　　猛然，叶少阳想到一个关键的问题：“那个，办法是不错，但是怎么检验呢？”
<br/>　　“处女，当然有那一层膜……你不是连这个都不知道吧。”碧清微微脸红地说道。
<br/>　　叶少阳更是尴尬，“我是说，怎么检验，总不能让她自己用手指去试吧……她肯定不肯的。”
<br/>　　碧清眼珠一转，“不然你去勾搭她吧，跟她睡了，如果流血，那自然就是第一次……”
<br/>　　“靠！你去跟她睡吧！”
<br/>　　“我没这功能。”
<br/>　　叶少阳斜眼看着她，“我说，你现在怎么变得这么污啊，这么坑的办法你也想得到。”
<br/>　　“你以为我这些电视剧是白看的啊，电视剧里，比这夸张的计谋都多得是。”
<br/>　　叶少阳无语。
<br/>　　碧清笑道：“不逗你了，不过，关于处女这件事，的确是可以做证据的，算是最后的办法吧，如果某一天非得让她相信的话。”
<br/>　　“嗯，先想办法回人间去再说吧。”
<br/>　　第二天，大家吃饱喝足，然后上路，走到中午的时候，官道分成三条，边上有个茶棚，大伙停下来喝茶，叶少阳问老板长安怎么走，老板指了西边那条路。
<br/>　　上路之后，走到快傍晚，又看到一家驿站，于是停车住宿。吃饭的时候，叶少阳随口问小二，这里离京城有多远。
<br/>　　“京城啊，有七八百里地吧。”小二挠着头说道，“大家都这么说，小人没去过京城，并不清楚。”
<br/>　　叶少阳一听，眉头皱起来，“不对吧，这不是到河南了吗，怎么还有那么远？”
<br/>　　小二笑道：“客观，河南离京师还远着呢，从这里到河北就有几百里地，到京师当然得有八百里。”
<br/>　　“河北？”叶少阳虽然是个地理白痴，至少也是上过学的，知道河北还在北边，纳闷道：“我们去京城啊，关河北什么事。”
<br/>　　小二更懵。“京师就在北边啊。”
<br/>　　“你欺负我没读过书啊，长安，几百年后叫西安，自然是在西边。”
<br/>　　小二笑道：“客观你们到底是去长安呢，还是去京师。”
<br/>　　“京师不就是长安吗？”
<br/>　　小二彻底傻了，随即挠头笑道：“客官就不要取消小人了，小人没读过书，也知道京师就是北京，那自然是在北边啦。”
<br/>　　京师，在……北京！！！
<br/>　　叶少阳无力地趴在桌子上，有一种强烈的要吐血的冲动，边上碧清几人也是震惊不已。
<br/>　　“对啊！”碧清一拍大腿，“京城，明朝的都市起先是南京，后来永乐皇帝迁都北京……跟长安有什么关系啊……”
<br/>　　叶少阳转过头，一脸哭丧地看着她，“你既然知道，为啥不早说啊大姐？”
<br/>　　“是你们先说的长安，我也就没注意这个啊，毕竟长安的确是古代当国都最多的城市啊。”
<br/>　　“好吧，我……实在不知道说什么好了。”
<br/>　　他已经完全记不清最早是谁说长安是都城的了，反正这么简单的常识，自己居然走了两天的路才发现，真的是要吐血。
<br/>　　碧清安慰道：“不错了，好歹今天就发现了，假如真的走到长安才发现，那才真叫郁闷。”
<br/>　　也只能这样安慰自己了……
<br/>　　“对了，叶少阳，你发现没有，这个世界的地图，比真实的要小的多。”
<br/>　　“哦？”
<br/>　　“我们从南京出来，才走了两天，马车的速度，一天也有两三百里，现在已经到驻马店了……在我们的世界中，是不可能的事。”
<br/>　　朱智慧正在一边埋头吃肉馒头，听见碧清的话，连忙点头，“没错没错，老猪我经常各地溜达，这些地方我都来过，这里的确比现实世界小很多。”
<br/>　　叶少阳眉头皱起来，问碧清：“为什么会这样？”
<br/>　　“不知道，我想，也许这是一个微缩版的大明王朝，我们这一路走来，我也注意观察了，除了一些有驿站和隘口的地方，很多这一路上该有的城市都没有，更别说县城什么的，我猜测，这个空间只有几个主要城市，用来给人们生活，其余这些地方都没有开辟，没人住，也没任何东西。”
<br/>　　叶少阳缓缓点头，“要真是这样，那我们赶往京城，也用不了几天啊。”
<br/>　　“我们需要快马，赶紧过去，万一星月奴渡劫成功，大家都得死在这。”
<br/>　　驿站就有卖马的，这倒不是问题。
<br/>　　叶少阳想了一下，然后跟大家一商量，决定买跟碧清先骑马赶往北京，瓜瓜和朱智慧、还有那两个妖精，留下来跟着马车走，有瓜瓜保护陈悦，叶少阳也放心。
<br/>　　第二天一早，叶少阳从驿站买马，结果还只有一匹了，好在碧清是妖精，于是让叶少阳骑马，自己现出本尊，挂在他身上，朝着北方一路狂奔……
