　　却说人族部落之中，经过这些年伏羲的治理，人族已经越来越强盛了，武者虽然增多了，但毕竟还是人族之中的少部分，普通人还是占据着绝对的人数优势。
<br/>　　武者可以通过修炼，去病除灾，防止祸患，而普通人却是不能，他们需要强者的守护，否则会经常遇到各种天灾人祸。
<br/>　　伏羲这些年就一直在为这个问题困扰着。他虽然知道该怎么做，但是却是缺少最关键的机缘，无法悟出先天八卦。
<br/>　　这天，伏羲来到洛水河畔，端做在河边，在静静的思考着一些问题，也在静静地参悟这这方世界的大道。
<br/>　　一方世界越是完善，大道就越是难以触摸，难以参悟，但若是能够参悟透，也是最为强大的。如他之前所在的世界，便是天地残缺不全，大道不完善，缺失了好多大道，悟道就变得容易许多了，但那也会导致自己的大道不完善，存在众多缺点。如今来到此方世界，想要如之前那般随便悟道，却是不可能了。
<br/>　　但对于他来说，这也是一切全新的起点，天地大道越是完善，他的根基就越是深厚，悟出的大道也就越完善。他发现，如今这方世界的大道对比他之前世界的大道，就如同成年人对比孩童一般，简直不可相比。
<br/>　　对于这样一个道法完整的地方，他自然是要抓住机会，争取早日证道。
<br/>　　伏羲带着上一世的修炼经验，再加上这一世，他乃是天定的人族圣皇，可以说，他如今就是主角，或者一个时代的气运之子，身上带着浓厚的气运，仅仅数百年时间，他便已经修炼到了天地法相境巅峰，也就是相当于大罗金仙巅峰，这个修炼速度，简直是惊人。不过，一想到他此刻身上汇聚着的无尽气运，这点速度，也算说得过去。
<br/>　　伏羲一阵对比，他发现自己之前世界的至尊便相当于洪荒的金仙，也就是说，他如今早已经超越了前世的自己。
<br/>　　天地悠悠，三个月一晃而过。伏羲在这三个月中，却是一动也不动，他好似陷入了某种悟道之中。他时而看着天上的星辰轨迹，时而看着大自然中的一山一水，这些最为寻常的力量之中，却又仿佛是带着某种最不寻常的力量。
<br/>　　一点一滴，一山一水，一草一木，一沙一石，都好似在沿着某种轨迹在运转，他想要抓住它们，但却又时时差了那么一点。
<br/>　　“到底是差了什么呢？”
<br/>　　伏羲在默默的思考着，心中却丝毫不着急，因为他知道，自己着急也没用，反而会破坏这种心境。
<br/>　　这一日，天朗气清，昊阳当空，阳光明媚。陡然，落水中传来流水之声。
<br/>　　“哗啦哗啦哗啦……”
<br/>　　那声音之中有但随着兽吼之音，听之好似那洛水之中有某种水兽出没一般，伏羲神情一动，连忙望了过去。
<br/>　　这一看，却看到，在自己的前方不远处，有一头异兽，分水踏波而来。
<br/>　　他自己打量一番，发现此兽并不是凶戾之兽，而是性子温和之兽。它高大八尺五寸，体型似马非马，生有龙头，龙爪，身上更是有龙鳞，那龙鳞散发着阵阵白光，充满一种神圣的气息，神骏无比，如同祥瑞。
<br/>　　“此乃瑞兽，莫非是那传说中的龙马？”
<br/>　　伏羲也算是见识渊博，轻易认出了这头异兽的名字。
<br/>　　看着向自己走来的龙马，眼中闪烁着智慧之光，似乎极为温和，一看就是祥瑞之兽，伏羲心中欢喜，却不知道这龙马来找自己干什么。
<br/>　　“咦！这龙马背上，居然有东西？好像是一本书与一幅图？嗯，这股宝光波动绝对是先天灵宝，而且还不是普通的先天灵宝，莫非这是我的机缘？”
<br/>　　伏羲惊讶的看着龙马匍匐在他面前，将脊背展现给自己，而且在它背上，有着一图一书，俱皆散发着璀璨宝光，颇为神异。
<br/>　　“听老师说，我有一番机缘在此，能不能悟道就看这一次了，莫非，这机缘就是这两件先天灵宝？”伏羲惊异道。
<br/>　　“看着两件灵宝的模样，莫非就是当年巫、妖、人三族大战之中的河图与洛书？妖族便是靠着这两件极品灵宝，摆下周天星辰大阵，纵横三界，无敌天下。但据说后来此两件灵宝失踪了，莫非，眼前的就是那妖族的河图、洛书？”
<br/>　　“若真是如此，那绝对是大机缘，说不定我的八卦就要在这机缘上，若是我能够参悟透此两件灵宝，定然能够完善大道，再次悟出先天八卦。”伏羲眼中有所明悟。
<br/>　　伏羲脸色狂喜，取下龙马背上的河图与洛书两件极品先天灵宝，他来到河边，盘腿坐下，眼中带着思索之芒，开始仔细研究这两件灵宝。
<br/>　　而那龙马，则是没有离开，它带着灵动的双眼，看着河边的伏羲，静静地伏在地上。
<br/>　　“嗡嗡嗡……”
<br/>　　道韵流转，玄之又玄，一道道天地法则，扑面而来。这河图与洛书，仿佛天生就是伏羲的证道之物一般，刚一获得，很快便沉浸其中。
<br/>　　“果然，此子好悟性，居然这么快就沉浸在其中，陷入顿悟之中，不愧是天定的人族天皇。”
<br/>　　无尽虚空之中，有一道姑与一道人站在一起，静静地关注着河边伏羲的一举一动。
<br/>　　“这就是天皇吗？果然不凡。”男子开口说道。
<br/>　　这两人自然就是玄都与伏羲的老师龟灵圣母了。一个为人教大弟子，一个为截教真传弟子，都属于三清一脉。
<br/>　　“嗯，伏羲不愧是上古妖族羲皇转世，这天资与悟性确实非凡。”
<br/>　　龟灵轻声说道。说着，脸上还流露出一丝复杂之色，虽说自己是伏羲的老师，但她却发现自己一点也没有看透伏羲。她隐隐觉得伏羲身上还有更大的秘密，不过，她懂得分寸，并没有深究，她将这些都归咎于上一世的原因。
<br/>　　“嗯。”
<br/>　　玄都点了点头，同意龟灵的说法。
<br/>　　岁月匆匆，就这样，道韵流转之间，三个月转瞬即逝。这盘坐在河边的伏羲，身上的道韵却是越来越盛，他周身的气息，仿佛是在进行某种蜕变，好似有无数智慧之光，从天灵之中冒了出来。
<br/>　　那智慧之光，照亮大千，犹如日月之明。他周身气息翻腾，道韵运转，向着四周扩散开来。
<br/>　　那道韵奥妙，却是伏羲借助河图、洛书，将自己上一世的八卦给彻底完善了，而且还弥补了许多缺陷，加入更多奥妙，使之更加适合此方世界的大道。
<br/>　　他把自己的所悟，融入到这一世的武道修炼之上，实力、道行更进一步。
<br/>　　“轰！”
<br/>　　……
