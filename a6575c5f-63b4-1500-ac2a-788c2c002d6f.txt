        童筱颖顿时恢复了平静，意识到自己现在是真正地处于危险之中后，故作淡定地对傅昼景眯眼一笑：“你不是说你‘爱’我吗？你爱我的话，是不会做出伤害我的事情，对不对？”
<br/>
<br/>        “你怕我？”傅昼景勾了勾唇。
<br/>
<br/>        童筱颖眼珠子一转，微微摇了摇头：“不是怕你。只是觉得，其实，你并不坏。”
<br/>
<br/>        傅昼景突然间苦笑，松开了童筱颖的手。
<br/>
<br/>        “你乖乖在这里待几天，我并不会把你怎么样。等十爷他把事情搞定了，你便可以回到大哥身边去了。”
<br/>
<br/>        “你……”听着傅昼景的这句话，童筱颖欲言又止。
<br/>
<br/>        而接下来的这些天，傅昼景与童筱颖相处之后，发现她真的已经不再是以前的那个筱颍了。
<br/>
<br/>        就好像只是一个与他的筱颍长得一模一样的女人，但其实却不是筱颍她本人。
<br/>
<br/>        “如果，当初，我没有选择跟你离婚，我觉得，你过得一点比现在好。”饭桌前，傅昼景一边给童筱颖夹菜，一边感慨道。
<br/>
<br/>        至少，她不会经历这么多风风雨雨，也不会变成现在这个样子。
<br/>
<br/>        他也不知道自己是不是疯了，为了这个女人，陪着她在这里住了四天，今天是第五天了。
<br/>
<br/>        筱颍以前会下厨，而现在坐在他面前的女人，在厨艺方面几乎是一片空白。
<br/>
<br/>        童筱颖无法理解傅昼景的感慨，只是应付性地回答道：“不一定。婚姻讲究的是忠诚，如果没有了忠诚，再相爱的两个人在一起也是互相折磨。”
<br/>
<br/>        突然，“嘭”地一声，那边大门的门板，撞向了地面。
<br/>
<br/>        几个全副武装的黑衣人，持枪闯了进来。
<br/>
<br/>        傅昼景下意识地起身，想要将童筱颖护在自己身后，岂料，他刚起身，就只觉额头上瞬间传来一阵刺痛感，紧接着他只觉眼前一阵天旋地转，浑身酥麻瞬间失去了知觉。
<br/>
<br/>        童筱颖看着傅昼景忽然倒地，吓得从椅子上起身，直接躲在了餐桌底下。
<br/>
<br/>        “出来，别躲了。”外头传来一个男人的呵斥声。
<br/>
<br/>        童筱颖拧起眉头，不得不从桌子底下爬出来。
<br/>
<br/>        她看了一眼那边的傅昼景，见他只是中了麻醉枪晕了过去，心里顿时松了口气。
<br/>
<br/>        随后的这几个黑衣人，拿得拿绳索，拿得拿眼罩，不但将她五花大绑，还蒙上她的眼睛，将她给带走了。
<br/>
<br/>        一阵舟车劳顿之后，童筱颖终于见到了光明，同时也见到了不想见到的人。
<br/>
<br/>        连华生的父亲连中凯！
<br/>
<br/>        这里并不是那个实验基地，反倒像是另一处机密的城堡，因为这里四壁都是刀枪不入的铜墙铁壁。
<br/>
<br/>        “我命令你杀了傅夜沉，你反倒跟傅夜沉谈起了恋爱，呵，你可真是继承了她的‘好’基因！”连中凯不禁呵斥道。
<br/>
<br/>        童筱颖却偏过头去，不想理会这个老男人。
<br/>
<br/>        不过，一直落在她心头的疑惑，现在总算是解开了。
<br/>
<br/>        原来是这个连老头在她的潜意识里发号施令。
<br/>
<br/>        只是，他为什么要杀傅夜沉？
<br/>
<br/>        他的儿子连华生还是傅夜沉的挚友。
<br/>
<br/>        连华生到底知不知道这件事情？
<br/>
<br/>        童筱颖正若有所思的时候，连中凯突然间凑到了她的跟前。
<br/>
<br/>        “虽然，我让你继承了她的一切特性，但是，我发现，你和夏玄烟一样，终究不是她本人。”连中凯忽然感慨起来，顿了顿后，下一秒，他又一脸严肃了起来，“来人，把这个失败品，冷冻了。”
<br/>
<br/>        “是。”一旁，立即有两个戴着白色口罩穿着白大褂的男人走上前来，将童筱颖带了下去。
<br/>
<br/>        童筱颖被他们强行放倒在一副水晶棺材里，另一个带着白口罩的女人拿着注射器，慢条斯理地扎入了她臂弯的肌肉。
<br/>
<br/>        没过一会儿，童筱颖便彻底失去了知觉。
<br/>
<br/>        “你不打算拿她的血制药了吗？”裴金澈的生母裴医生凝眉问道。
<br/>
<br/>        隔着一张厚重的钢化玻璃，四面泛白的密室里放着三副连通着各种透明管道的水晶棺材。
<br/>
<br/>        而棺材里的三个女人，几乎长得一模一样。
<br/>
<br/>        连中凯看着里面的那三个女人，冷哼：“我们的实验，这么多年来，压根就没有成功。你儿子阿澈变成那个样子，就已经是最好的证明了。基因是复杂，我们谁都无法去控制。哪怕制造了一个又一个同样的人来，甚至通过催眠的方式将她们的性格同化，可结果，她们还是有自己的主观意识。”
<br/>
<br/>        “你真的要杀了童筱颖吗？”裴医生深沉地问道。
<br/>
<br/>        连中凯云淡风轻地回答道：“她本就已经死过一次了，如果不是我，她能有今天？更何况，我不能让中央的人知道，我们在做这种违背天理的科研。”
