　　这一切的发生，只是瞬息之间的事情。
<br/>　　天空之上的裂痕，也已经是消失了，看起来所有事情都没有发生。
<br/>　　不过楚风眠手中，本来即将镇压的那一缕残魂，却是的确被毁了。
<br/>　　这齐林山创始人，的确是够狠。
<br/>　　他也知道，如今他身处于亿万里之外，传递过来的力量，根本不是楚风眠的对手，甚至是都难以救下那一缕残魂，所以索性将那一缕残魂毁去。
<br/>　　这一缕残魂，如果是被楚风眠得到，楚风眠便是可以从中抽取记忆，甚至是知晓这齐林山创始人的无数隐秘。
<br/>　　可现在却是被这齐林山创始人亲自毁去了。
<br/>　　现在的楚风眠，就连这齐林山创始人到底是什么名字都不知道。
<br/>　　“可惜了。”
<br/>　　楚风眠摇了摇头，可他也知道，那个时候楚风眠也无法阻止这齐林山创始人。
<br/>　　这齐林山创始人虽然是将灵力从亿万里的地方传递了过来，可如果真的轰击楚风眠，楚风眠哪怕是全力之下，都可能受伤。
<br/>　　毕竟这可是真正天人境强者的出手。
<br/>　　那样的情况之下，楚风眠想要护住这一缕残魂，都是心有余力不足。
<br/>　　不过楚风眠通过这一次交手，倒是确定了一点，那就是大罗天星域之中，曾经的那些天人境强者，的确都去了其他的地方，无比遥远的地方。
<br/>　　那五行天的创始人只怕是已经陨落了，暂且不论，林五行，还有如今这齐林山创始人，都是最后没有本体前来。
<br/>　　便是极有可能他们都在无比遥远的地方，根本无法赶过来。
<br/>　　所以短时间内，楚风眠应该是不担心有天人境的强者来找麻烦。
<br/>　　不过对楚风眠最为关键的，还是提升实力，这些天人境强者，短时间内不会来，但是未来，楚风眠终究是要自己去应对。
<br/>　　只有拥有足够的实力，楚风眠才可以跟那些天人境强者，真正的抗衡。
<br/>　　等楚风眠将三大势力全部降伏之后，楚风眠就要去寻找突破九劫古帝，踏入半步天人境的机会了。
<br/>　　现在楚风眠的境界已经是达到了九劫古帝，距离半步天人境，只有一步之遥。
<br/>　　虽然是一步之遥，可楚风眠想要突破，谈何容易？
<br/>　　哪怕是寻常的九劫古帝，想要踏入半步天人境，都无比困难。
<br/>　　数十名九劫古帝之中，也就只有一人，可以踏入半步天人境。
<br/>　　其中的艰难，不言而喻。
<br/>　　像是青梦这样的天才，都没有绝对的把握，可以踏入半步天人境。
<br/>　　更何况对楚风眠来说，突破的难度，更是要远超其他人千倍，万倍。
<br/>　　楚风眠都有些无法想象，他突破半步天人境的难度，只怕是必须要有着大机缘才可能突破。
<br/>　　甚至可能楚风眠要离开大罗天星域，前往其他星域，去寻找机会，才可以突破到半步天人境。
<br/>　　不过在此之前，这三大势力，楚风眠都要挨个降伏。
<br/>　　楚风眠看着眼前的齐林大陆，整个齐林大陆之上，已经是没有了任何的生灵了，只剩下寥寥几名齐林山长老还活着。
<br/>　　所有的齐林山弟子，都被那林书古帝血祭了，已经是彻底的毁了。
<br/>　　“我们愿意臣服。”
<br/>　　那几名齐林山长老，见到楚风眠的目光，急忙的跪在地上说道。
<br/>　　齐林山创始人出手，都奈何不了楚风眠，现在他们又怎么敢还跟楚风眠抗衡？
<br/>　　“好，日后齐林山，也不复存在了，你们都加入到剑道门之中去，作为剑道门中最为普通的弟子，只要是你们好好为剑道门做事，立下功劳，你们的地位也可以提升。”
<br/>　　楚风眠凝聚灵力，在那几名齐林山长老的身上种下禁制之后，便是直接让他们去圣龙秘境了。
<br/>　　同时楚风眠跟青梦也是向着三大势力之中，最后一个势力，沧海都飞去。
<br/>　　对沧海都的仇怨，楚风眠倒是没有齐林山，五行天那么大。
<br/>　　很大原因便是因为海公主。
<br/>　　之前海公主，提醒了楚风眠五行刀圣即将偷袭的事，算是救了楚风眠一命。
<br/>　　虽然楚风眠后来也放过了海公主的父亲，碧云海王一命，不过总归倒是有些交情。
<br/>　　如果沧海都识相一点的话，楚风眠倒是不准备出手。
<br/>　　不过如果沧海都，还敢跟楚风眠为敌的话，那楚风眠也不会对敌人，手下留情。
<br/>　　楚风眠要将整个大罗天星域，全部整合成剑道门，觉不允许存在还有任何敢反抗的势力。
<br/>　　二人的遁光，在星空之中飞行，一个时辰的时间，在楚风眠的目光之中，突然一座碧蓝的大陆，映入到了楚风眠的双目之中。
<br/>　　不是碧蓝的大陆，应该是眼前的这一座大陆，九成九的地方，都是无尽的汪洋，真正的陆地，只有一点点，还被分割成一座座小岛。
<br/>　　看起来，就像是一座海洋，漂浮在无尽星河之中。
<br/>　　“沧海大陆，这里就是沧海都的老巢所在。”
<br/>　　楚风眠早就听说，沧海都所在的大陆，是一个几乎全是汪洋的大陆，现在终于可以一见。
<br/>　　在这样的环境之下，沧海都一代代修行的，几乎都是水系功法，而且许多人的血脉，都是跟海族有些关系。

<br/>　　不过后来荒古大变，一场场争锋之中，海族逐渐没落。
<br/>　　如今一些大陆周围的海洋深处，还会有着一些海族的存在，九域的海洋之中，也有着海族的存在，不过楚风眠却是了解不多。
<br/>　　可虽然有着零星一些海族的存在，可已经是很少有着以海族为主的大陆了。
<br/>　　这沧海大陆，便是一个异类，以海族为主。
<br/>　　沧海都，也是人类武者跟海族一同建立起来的，应该说沧海都之中的许多人类武者，其实都或多或少有着海族的血脉，只是看起来跟人类一样。
<br/>　　海族是海洋霸主，在修行水系功法的时候，有着得天独厚的优势，蕴含海族血脉，同样如此，这也是沧海都可以成为大罗天星域四大势力之一的资本。
