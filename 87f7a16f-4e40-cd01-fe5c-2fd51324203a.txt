　　“哦？”
<br/>　　叶尘听到这话，顿时眉头微微一挑，脚上顿时便收回了几分力道，
<br/>　　“说下去！”
<br/>　　老者两眼鼓溜溜一转，
<br/>　　“我若说出来，你能饶我一命吗？”
<br/>　　叶尘冷哼一声，
<br/>　　“少废话！不说的话，现在我就让你死！”
<br/>　　老者顿时怂了，
<br/>　　“好好！我说！我说！”
<br/>　　微微顿了顿，老者似乎是在整理思绪，过了片刻之后，方才幽幽开口道：
<br/>　　“据说，在千年之前，地球进入末法时代，而且四大族最后一次大战之后，虽然魔族和妖族战败，妖族被封印，魔族也自动隐退……”
<br/>　　“不过，四族都意识到一个问题，地球上的灵气越来越稀薄，根本容不下那么多修真者！”
<br/>　　“于是，四族高层之间，秘密立下了一个约定，四族开辟出各自的秘境之后，还加持了庞大的聚灵法阵，然后全部转移进去，和世俗界的凡人划清界限，以此来限定修真者的数量！”
<br/>　　“即便是有少数，遗留在世俗界的修真者，比如华夏境内的逍遥山庄，欧洲的血族和教廷等等，这些势力同样要遵守四界的约定，不得将修真之法，传授给世俗界的凡人……”
<br/>　　听完这老者的一番解释，叶尘也顿时明白了许多，之前他一直想不通的事情。
<br/>　　难怪地球世俗界，修真传承几乎完全断绝，原来竟然是这个原因！
<br/>　　“你刚刚说，我已经成为了四界公敌，你这话是什么意思？”
<br/>　　叶尘继续追问道。
<br/>　　老者趴在地上，眼眸中闪过一抹阴冷之色，缓缓道：
<br/>　　“我听说，你动用修真道法，逼的当世第一大国投降，后来又创立了狂仙门，公开收徒，我说的可对？”
<br/>　　叶尘点了点头，
<br/>　　“那又如何？”
<br/>　　老者冷冷一笑，
<br/>　　“若是在千年之前，自然没有人会多说什么，可是现如今，就连四界之中的天地灵气，都开始日渐稀薄，你这么做，日后将会有多人踏入修真之路，这等于是在和四界争夺资源，他们又岂会容你？”
<br/>　　听完老者的一番解释，叶尘顿时一阵默然，
<br/>　　不得不说，这老头说的还挺有道理的！
<br/>　　老者似乎也看出，叶尘已经被他的话说动，当即两眼又一转，嘿嘿笑道：
<br/>　　“小兄弟，倘若你放了我的话，我倒是有个你办法，可以帮你解决这个麻烦！”
<br/>　　叶尘两眼微微一眯，
<br/>　　“哦？那你倒是说说看！”
<br/>　　老者顿时精神一震，嘴角露出一抹笑意，道“你现在最缺的，无非就是一做靠山而已！我药王宗，乃是蓬莱仙境五大派之一，只要你肯加入我药王宗，由我药王宗罩着你，自然没有人敢再找你的麻烦！”
<br/>　　叶尘冷冷一笑，
<br/>　　“你的意思，是要我给你们药王宗当奴才喽？”
<br/>　　老者脸色微微一变，强笑道：
<br/>　　“小兄弟这么说就不对了！正所谓，识时务者为俊杰！再者说，以小兄弟的绝顶天资，只要加入我药王宗，必然会被宗主大人收为亲传弟子，将来委以重任，甚至成为我药王宗下任宗主，也不是没有可能啊！”
<br/>　　叶尘又道：
<br/>　　“如果我没有记错的话，之前你们药王宗有一位长老，就是死在我的手上，而且我还从你们手上抢走了逍遥山庄，你们会就这么轻易放过我？”
<br/>　　老者心中暗暗冷笑，嘴上却继续甜言蜜语道：
<br/>　　“叶小兄弟说笑了！夏侯长老不过就是门中一普通长老罢了，如何能和叶小兄弟相提并论？小兄弟尽管放心！我公孙林，以我的项上人头担保，肯定不会对小兄弟秋后算账！”
<br/>　　不得不说，这老头蛊惑人心的本事，倒也着实不小，倘若是那些没见过什么世面的菜鸟，被他一通连哄带吓，说不定还真就同意了。
<br/>　　可惜，他面对的叶尘！
<br/>　　“嗯，你说的好像很有道理！”
<br/>　　叶尘悠悠开口道，似乎已经为之心动。
<br/>　　公孙林顿时大喜，
<br/>　　“这么说，叶小兄弟，你同意加入我药王宗了？”
<br/>　　嘴上这么说着，公孙林心中，已经开始暗暗寻思起来，
<br/>　　‘哼！果然是年轻啊！被老夫三言两语就给哄住了！等老夫先将你诓回药王宗，到时候老夫一定要将你千刀万剐，以报今日之辱！’
<br/>　　甚至，公孙林心中，已经开始暗暗寻思着，到时候给叶尘用什么酷刑折磨他了。
<br/>　　然而，叶尘一开口，公孙林顿时傻眼了，
<br/>　　“抱歉，我拒绝！”
<br/>　　公孙林大急，脸色浮现出一抹狰狞之色，
<br/>　　“为什么？小兄弟，你可要三思啊！没有我们药王宗的庇护，那你就是四界公敌，你难道真的以为，凭你自己一个人，可要承受四界的怒火吗？”
<br/>　　叶尘冷冷一笑，
<br/>　　“第一，我不喜欢给人当奴才！”
<br/>　　“第二，就算和四界为敌，我叶狂仙又有何惧？”
<br/>　　“第三，也是最重要的一条，胆敢对我朋友出手者，杀无赦！”
<br/>　　“不过，看在你说了这么多机密的份上，我可以给你一次逃命的机会！”
<br/>　　叶尘说到这里的时候，公孙林早已经脸色惨白无比，不过在强大的求生欲之下，还是连忙哆里哆嗦道：
<br/>　　“什、什么机会？”
<br/>　　叶尘嘴角露出一抹嘲弄的笑容，
<br/>　　“我可以给你三十秒钟的逃命时间，三十秒之内，你若是能从我的手下逃走，我便饶你一条性命！”
<br/>　　说完这话，叶尘缓缓抬脚，同时开始数数，
<br/>　　“1……2……”
<br/>　　公孙林先是一愣，继而哪里还敢犹豫，当即脚下猛然一蹬，直接冲破周围的空间禁制，飞速朝大学外面爆射而去。
<br/>　　叶尘却一脸悠然，继续慢慢数着数，
<br/>　　“28……29……30！”
<br/>　　等到第三十个数数完，叶尘整个人已经在原地消失不见。
<br/>　　此时，
<br/>　　已经狂奔到数十里外的公孙林，正暗自松了一口长气，本以为自己已经捡回了一条小命。
<br/>　　“小子！等我回到药王宗，请太上长老出马，再来报今日之仇！”
<br/>　　然而他这话才刚刚落下，一道充满戏谑的声音陡然响起，
<br/>　　“唉！我给了你机会，可惜你不中用啊！”
<br/>　　公孙林顿时瞳孔一缩，眼前已经凭空出现了一道身影，挡住了他的去路。
