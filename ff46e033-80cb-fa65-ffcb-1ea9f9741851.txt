　　这两人顿时面色煞白，“我，我们没有强迫妇女卖淫啊，我……”

<br/>　　要知道，三十晶石，如今的市场价，是可以玩一个三线女明星一晚上的。

<br/>　　林峰没有多说什么示意她跟着自己来。
<br/>　　回到小区的屋子里。
<br/>　　林峰道：“这里就是我的家了，北边那个副卧室你可以收拾一下然后和你的孩子住在这里。以后这个屋子里的保洁问题就交给你来了。”
<br/>　　妇女二话不说连忙道是。
<br/>　　“你放心吧，我之前就是保洁公司的保洁阿姨，我肯定给你弄的干干净净的。”
<br/>　　妇女二话不说放下自己两个娃娃后就去拿抹布开始帮林峰清理了。

<br/>　　“也不知道苏老师在干嘛呢。”
<br/>　　林峰寻思着就给苏云打了个卫星电话。
<br/>　　“干嘛呀，不是前天才打了个电话吗？”

<br/>　　林峰一听好奇道：“苏老师你干嘛在呢。”
<br/>　　“啊啊啊啊啊啊。”
<br/>　　电话那头居然传来非常有节奏的声音。
<br/>　　林峰一听，这尼玛不对劲啊，这不就是在干那啥时候发出的吗？
<br/>　　“妈的，苏老师给我戴绿帽子了？”
<br/>　　这是林峰本能想到的一个念头。
<br/>　　不过转眼一想，不可能啊，苏云老师很保守的，这才没见几天就找了个野男人。

<br/>　　随后传来了苏云有些怪罪的声音。
<br/>　　“苏云，你可真有钱，卫星电话都买得起，这东西还是军部推出的最新款，属于限量款的，一百部iphonex都不见得可以换一部。”

<br/>　　林峰笑着说道。
<br/>　　“是啊，她叫郑梦，和我一样都是老师，不过她是小学音乐老师。今年才入职的，现在我和她都在白云镇准备做任务呢。”
<br/>　　苏云的声音很欢快。
<br/>　　林峰也为她感到高兴，看来苏云老师的确找到了自己喜欢做的事情。

<br/>　　苏云说道。
