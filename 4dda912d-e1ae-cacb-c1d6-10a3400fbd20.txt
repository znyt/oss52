　　第73章
<br/>　　快马在城门前停下，如今风声依旧很紧，城门上就贴着程大雷、秦蛮、林少羽三人的画像。
<br/>　　不过，除程大雷外，其他两人的画像依旧不像。
<br/>　　今天秦蛮将脸用锅底灰抹黑了，在塞给卫兵一把铜板后，顺利混入落叶城。
<br/>　　“二当家，你大哥和杜茂的交情怎样，会不会有什么问题？”
<br/>　　“军师你就放一百个心，大哥和杜老大是磕头拜把子的弟兄，要我说，这次根本不用我亲自来，递句话就把事情办了。”
<br/>　　三人牵着马前行，在大槐树下停住。高飞豹把马拴了，上前拍门。
<br/>　　“敲什么敲，报丧么！”
<br/>　　门内探出个脑袋，谨慎的盯着三人。
<br/>　　“去里面说一声，飞虎寨高飞豹来了，要见你们杜老大。”
<br/>　　“等着。”
<br/>　　这人撂下两个字，不多时重新把门打开：“杜老大在里面歇着呢，你们进来等吧。”
<br/>　　三人走进院子，徐神机目光转了一圈，看到几条大汉在院子里喝酒说笑，地上摆着几个喝空的酒坛。
<br/>　　“豹子，你小子怎么过来了，又想和花酒了。”
<br/>　　“小心我们弟兄拿你见官，你小子的人头现在可值一百两银子。”
<br/>　　有两个和高飞豹认识的人，笑骂着和他打招呼。
<br/>　　“别他娘的废话，杜老大呢，我找他有点事。”
<br/>　　“大哥喝醉酒在里面眯着呢，你先等一会儿。”
<br/>　　终于，大家的目光落在徐神机和秦蛮身上，问：“这两位弟兄看着眼生，什么来路，你们飞虎寨新来的弟兄。”
<br/>　　“还没介绍，这二位是蛤蟆寨的弟兄，这就是蛤蟆寨军师徐神机。”
<br/>　　这话说出来，院子内突然静下来。喝酒的也不再喝酒，举石锁的也把石锁丢在地上，都站起来，眼神盯着徐神机。
<br/>　　“莫非……就是那位外号玉面书生的徐神机？”
<br/>　　“据说蛤蟆寨接连几场硬仗，像击溃黑石城一千精兵，都是这人在背后出谋划策。”
<br/>　　有人盯着徐神机上下打量，只见他身材佝偻，面目可憎，书生二字说不上，玉面更是不知从何谈起。
<br/>　　“说是说得这么邪乎，谁知道真的假的，要我说，弄虚作假居多。”
<br/>　　如今蛤蟆寨的名头实在响，就连徐神机的声望也是水涨船高。大家都传这玉面书生如何如何神机妙算，可看见徐神机后，却是这幅样子……
<br/>　　老实说，有种偶像崩塌的感觉……
<br/>　　“嗨，你们蛤蟆寨真这么大本事，那你看看，我们弟兄能打你们蛤蟆寨几个？”
<br/>　　“哈哈，这个不好说呐，不好说呐。”徐神机弯着腰，陪着笑。
<br/>　　“有什么不好说的，大胆说，你担心什……”
<br/>　　“我担心的是你们人太少……”徐神机抬起头，眸子底杀机毕现：“不够杀啊。”
<br/>　　鸦雀无声！
<br/>　　院子里的视线都凝聚在徐神机身上，眼底有警惕、愤怒以及不安。
<br/>　　徐神机冲着屋子，高声道：“蛤蟆寨徐神机，特来拜见杜老大。”
<br/>　　屋门缓缓打开，一袭白袍的中年男人跨出屋子，站在太阳底下。
<br/>　　好俊一条大汉！
<br/>　　秦蛮心底也忍不住赞了一声，只见他身高八尺有余，猿背蜂腰，脸上懒洋洋的样子，看上去像个读书的儒生，可眸子中流露出的光芒却如刀一般锋利。
<br/>　　徐神机知道这位就是正主，他冲秦蛮递个眼色。秦蛮会意，将背着的包袱皮打开，取出一尊纯银佛像，约莫百两重。
<br/>　　“杜老大，我们大当家让我过来，想和杜老大交个朋友。”
<br/>　　杜茂将那佛像接过来，放在手中掂了掂。
<br/>　　“杜某不念佛，不烧香，程当家送我一尊佛像干什么，想杜某以后改吃素嘛。”
<br/>　　院内响起哗然大笑，众人都戏谑的看着徐神机。
<br/>　　“拿回去吧，我杜茂不缺朋友。”杜茂挥手将那佛像丢过来：“也回去转告你们大当家，狗有狗路，虎有虎道，我和你们蛤蟆寨不是一条路上的，劝你们程当家，手也不要伸得太长，这里，不是青牛山。”
<br/>　　高飞豹没想到杜茂这么傲，当场就碰个大钉子。他尴尬道：“杜老大，这次可是我大哥的面子……”
<br/>　　徐神机一摆手，阻止高飞豹继续说下去。他抱拳道：“杜老大请了，虽说狗有狗路，虎有虎道，可山水有相逢，路上道窄，免不得有碰上的时候，谁是狗谁是虎，到时候再见分晓。”
<br/>　　三人转身往外走，两条大汉堵在门口。
<br/>　　“兄弟，来了别空手回去，带上点东西。”
<br/>　　说着，杜茂忽然提起地上石锁，兜头砸了过来。听得脑后风声震响，秦蛮猛地回头，跨步拦在徐神机面前，往前进了一步，将那石锁接住，退步卸了石锁的冲力。
<br/>　　徐神机和高飞豹这时才转过头来，高飞豹怒道：“杜老大，你做什么！”
<br/>　　“路上道远，不方便带重物。”徐神机冷冷道：“秦蛮，还回去。”
<br/>　　石锁再一次飞在空中，砸向杜茂，杜茂往后退了三步接住，放在脚边。眼神在秦蛮身上打量了一遍。
<br/>　　“送客。”
<br/>　　三人混出城，高飞豹气得哇哇直叫，本来想仰仗面子摆平这件事，如今这面子是踏踏实实掉在地上。
<br/>　　“杜老大太狂了，真以为他自己能只手遮天了！不过老徐，平日看不出来，你也有两下子嘛。”
<br/>　　“好说，好说。”徐神机笑笑，忽然发现一旁秦蛮脸上的表情不对。
<br/>　　“秦老弟，你怎么了？”
<br/>　　秦蛮脸上露出苦笑：“大意了，伤着胳膊。”
<br/>　　扯开袖子，手臂已肿起来。杜茂背后偷袭，秦蛮防得急，闪住了胳膊。
<br/>　　回到蛤蟆寨，程大雷看到这一幕时，眉头立刻皱起。
<br/>　　“你和杜茂交过手，对方本事怎样？”程大雷问。
<br/>　　“不在我之下，确实有打虎之能。”秦蛮叹了一口气。
<br/>　　“这次亏得秦蛮有本事，不然我们可能就栽了。这杜茂忒也目中无人，根本不把我们蛤蟆寨当会事，说您是狗，他是老虎，老虎根本不会和狗交朋友。”徐神机。
<br/>　　“我说什么来着，我说什么来着……”程大雷一股火往天灵盖撞：“就不能惯着他们，真真是给他脸了。”
<br/>　　
<br/>(本章完)
