　　齐昊心急林惊羽安危，这可是自家师傅心头肉。
<br/>　　正因为关心，所以这一剑打过去完全没有留手。刚施展飞剑，齐昊心里就后悔了。
<br/>　　这里是大竹峰，自己出手过重，他和林惊羽都不会有好果字吃！
<br/>　　可到了这时，想收手已不可能！
<br/>　　“大不了先带着师弟逃回龙首峰，一切由师傅出面调和。！”电光火石之间，他就打定主意。
<br/>　　飞剑去势不减，齐昊用尽全力稍微偏移方向就不在关注。反而酝酿法力，准备一会儿见势不妙带着林惊羽逃跑。
<br/>　　苏重察觉身后攻击，几乎本能的一道剑气就飞了出去。乳白色剑气撞上飞剑，仅仅叮的一声响。飞剑就毫无阻碍斩破剑气朝他袭来。
<br/>　　田不易脸色大变，这里可是大竹峰，竟有人敢对自己弟子下狠手！
<br/>　　电光火石间，即便是他想出手都来不及。
<br/>　　苏重却面色平静，中食二指并成剑指立在胸前。一声低喝，身后顿时冲出十二道剑气。
<br/>　　好似孔雀开屏，乳白剑气从身后冲出，划个弧形绕到身前，砰的一声撞在一起。一道更加明亮剑气骤然出现在苏双指之上。
<br/>　　宛若黄豆大小剑气晶莹剔透，乳白色光芒毫不刺眼。可却锋锐惊人，只看一眼就有种被阳光刺痛想要流泪的感觉。
<br/>　　“去！”苏重冷喝一声。
<br/>　　聚合剑气骤然飞出，几乎瞬间就撞上白色飞剑。
<br/>　　砰！
<br/>　　一声巨响，大片白色冰晶在半空炸开，太阳照射下，竟折射出七彩虹光。
<br/>　　齐昊脸色一白，急忙调动法力牵引飞剑收回手中。看着气定神闲苏重，眼中满是不可置信。
<br/>　　虽然刚才仓促出手，力量可不小，苏重竟能完好无损接下！
<br/>　　这家伙也是和林师弟一样的妖孽吗！
<br/>　　看着苏重背后再度亮起道道剑气，而且比刚才足足多了两倍，齐昊有些发毛。他倒是不怕，可这眼看就要拼命，他实在不好招架！
<br/>　　“好了尔旦，这是你龙首峰的齐昊师兄，不得无礼！”田不易刚才怒气冲冲，现在眼里却满是得意。
<br/>　　你齐昊是龙首峰大师兄，每次会武都压住我大徒弟，现在却在我小徒弟手里吃瘪，嘿嘿……田不易心里乐开了花。
<br/>　　刚才关心则乱，这会儿才想起，自家小徒弟可不一般，是能自创武功，敢拿自己试验庚金剑气这种酷烈功法的狠主！
<br/>　　“见过齐昊师兄。”苏重不咸不淡的行了一礼！
<br/>　　刚才虽情有可原，可一出手就下狠手，苏重可不待见他。
<br/>　　“二蛋！你怎么变得这么厉害！？”林惊羽从地上爬起来，满脸的不甘心。
<br/>　　刚才还一身白衣，背负斩龙剑的偏偏美少年，现在却滚得满头满身的灰土，眼上还带着个黑眼圈。
<br/>　　田灵儿噗嗤一声就笑了。
<br/>　　齐昊满脸通红，自家师弟可真惨啊。
<br/>　　“二蛋一直都很厉害的！”张小凡满脸自豪。
<br/>　　“好了，七脉会武的事我们知道了，你去其他地方送消息吧！”田不易大袖一挥，直接对齐昊下了逐客令。
<br/>　　齐昊也不想多呆了，拉上还耿耿于怀，不时看向苏重的林惊羽，直接腾空而起快速消失不见。
<br/>　　“二蛋，可以啊，竟然连齐昊的飞剑都接的住！”大师兄宋大仁拍着苏重肩膀大声赞叹。
<br/>　　苏重谦逊一笑：“都是师傅教的好！”
<br/>　　田不易顿时笑的合不拢嘴。真是苍天有眼，给自己送来个妖孽徒弟。不仅自己是修炼奇才，还能带动其他师兄弟提高修为。
<br/>　　他已经决定了，黑竹林阵法、《聚元功》，都要列入大竹峰机密，作为传承的一部分，自家徒弟的名字，也要记入大竹峰历代纪要，保存在祠堂内！
<br/>　　“尔旦，过多时间就要七脉会武，你回去准备一下，到时候去涨涨见识，对修炼大有好处！”田不易和蔼道。
<br/>　　转过身来看着几个笑嘻嘻的徒弟，立刻冷声呵斥：“都还傻站着干什么！过段时间就要七脉会武，还不滚去黑竹林修炼，难道往年输的还不够难看吗！”
<br/>　　几个弟子浑身僵硬，顿时如鸟兽散。
<br/>　　看着悠闲踱步的苏重，众人心里不住的嘀咕，师傅真偏心！就连田灵儿都不满意了，我可是您亲闺女哎，怎么对她比对我还好！
<br/>　　……
<br/>　　后山竹屋，苏重回到这里就开始继续琢磨改进剑气。刚才的插曲根本没在苏重心理留下痕迹。他更在乎庚金剑气威力，这是他将来横行的根本，怎能不重视。
<br/>　　“十二道剑气聚合，竟能接下齐昊一剑。虽然那不是他全力，也没施展道法，仅仅催发法力。但足以看出剑气聚合的威力！”他聚合剑气的路子算是走对了！
<br/>　　真要走一剑生万法的路数，以他如今三百来枚剑气符，顶多也就能布置一座简陋阵法，威力不好说，但爆发一波后，就会直接陷入疲软状态任人宰割。这可不是苏重想要的。
<br/>　　“可惜，庚金剑气没有完整体系。”苏重想想也就摇头。
<br/>　　这想法太贪心。剑气聚合威力强大，真要被彻底完善，《庚金剑气》就不只是秘法，而是门派核心！就不可能落到田不易手里。
<br/>　　“一方面加快剑气聚合速度，最好能和剑气一样，心念一动立刻放出才好。再一个就要尝试融合更多剑气在一起。提高单方面攻击威力。”苏重心理暗自琢磨。
<br/>　　总的来说就是提高攻击速度和攻击强度。
<br/>　　“单道剑气激发迅速，是因为剑气符。如果把剑气聚合阵法，也放在体内呢！”
<br/>　　想到就干，苏重从来不会犹豫不前。聚合剑气阵法，已经被证实可用。他根本不用像当初融合天机印时一样犹豫，直接就在体内开始融合。
<br/>　　调动剑气符，十二个一组形成一条长鞭模样。丹田内顿时大变模样。
<br/>　　以前丹田内，中心处一颗道种，周围散布剑气符旋转。现在剑气符凝聚成阵宛若飘带，真就有点儿像银河了。
<br/>　　心念一动，一条剑气飘带猛然放光，苏重食指顶端，顿时出现一粒黄豆大小晶莹光点。
<br/>　　成了！
