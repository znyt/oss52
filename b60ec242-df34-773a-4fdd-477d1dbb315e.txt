夏夕绾是让婶儿们直接叫她名字，叫绾绾的，但是她也不知道婶儿们为什么总是喜欢叫她绾绾小姐，姑爷这个称呼的确有点怪怪的。
<br/>
<br/>        夏夕绾肤如凝脂的娇肌上染上了一层蔷薇花般娇艳的红色，她转过身，纤白的手指伸出去，大胆的勾住了男人英俊坚毅的下颌，“那是，陆先生，我在这里可是有好多地的，女土财主一个，陆先生，不如你留下来做我的压寨相公吧，我养你呀~”
<br/>
<br/>        陆寒霆的人生字典里还没有被“包.养”这个词，他低下眸，干净清冽的男人气息压进她，“想让我吃软饭啊，原来你图我的色。”
<br/>
<br/>        夏夕绾忍住青涩的娇羞，挑衅的望着他，“胡说，我怎么能是图你的色呢，我明明是…馋你的身子！”
<br/>
<br/>        陆寒霆眸色一沉，直接往她娇艳的红唇上欺去。
<br/>
<br/>        夏夕绾吓得轻呼了一声，连忙抓起一个红颜草莓塞到了他的嘴巴里，娇软的身体也从他的怀里退了出去，一本正经的开始煮面，“陆先生，你给我老实一点，在我的地盘上，你必须听我的！”
<br/>
<br/>        陆寒霆嚼开了那颗草莓，奶奶甜甜的草莓味道迅速在口腔里蔓延开，他伸手来到女孩绝丽的小脸上捏了一把，“真想养我，那点地可不够，明白么陆太太？”
<br/>
<br/>        “…”
<br/>
<br/>        行了，她被赤裸裸的鄙视了！
<br/>
<br/>        ……
<br/>
<br/>        两个人一起吃了面，赵婶的手擀面加独家酱汁，怎么吃都好吃，夏夕绾站起身，“陆先生，你去洗碗。”
<br/>
<br/>        陆寒霆收拾碗筷，然后看了她一眼，“你干什么？”
<br/>
<br/>        “我去给你洗衣服。”夏夕绾跑进了房间里。
<br/>
<br/>        这时周婶走了进来，周婶手里捧着几套新衣服，看着陆寒霆恭敬道，“姑爷，我给你和绾绾小姐做了几身新衣服，你们看合身不合身。”
<br/>
<br/>        陆寒霆看着那上等衣料，还有上面纯手工的精致走线，这是一位江南绣娘。
<br/>
<br/>        “先放着吧。”
<br/>
<br/>        “是。”
<br/>
<br/>        周婶将衣服放了下来。
<br/>
<br/>        这时里面传来了夏夕绾的惊呼声，“李婶，你怎么又把我的衣服给洗了？”
<br/>
<br/>        李婶拿着抹布正在擦玻璃，所过之处，擦的蹭光瓦亮，闻言她十分委屈的看着夏夕绾，“绾绾小姐，我没有洗啊，你们的贴身内衣我可没有动。”
<br/>
<br/>        “…”
<br/>
<br/>        夏夕绾抓了抓自己的头发，这些婶儿们总是这样，煮面把烧开水留给她，洗衣服将内衣留给她，反正总要留一点事情告诉她，这些活儿都是她自己做的！
<br/>
<br/>        “李婶！”
<br/>
<br/>        “绾绾小姐，你是生气了吗，你莫非觉得婶儿们年纪大了，收拾的不干净？”
<br/>
<br/>        夏夕绾，“…”
<br/>
<br/>        婶儿们每次都玩这一招，啊，她又输了。
<br/>
<br/>        ……
<br/>
<br/>        陆寒霆洗好了碗筷走进房间，夏夕绾已经洗好两个人的贴身内衣了，衣服上散发着金纺柔顺的淡淡香气，她纤柔的站在阳台上，将两个人的内衣挂上去。
<br/>
<br/>        他是黑色条纹的内裤，旁观是她的，薄薄的衣料，粉色卡通的，随风飘啊飘。
