　　“对了，接下来尸傀门的事情你别管了，你去管理骷髅会，骷髅会是我们王家在俗世一手栽培起来的势力，如今总算是发芽开花了，混到了基地内地下实力的第二，你去联络他们，一来是调查曹龙的事情，二来是继续找一些处女秘密押送到这里来。”王梅道。
<br/>　　“是，梅姐。”
<br/>　　寡妇道玩就转身离去了。
<br/>　　“战门，就算你厉害又如何。我王家和尸傀门也不是好惹的。”
<br/>　　王梅眼中露出一抹寒意。
<br/>　　战门自然不知道自己无缘无故背了黑锅。
<br/>　　话说林峰回到宿舍区和陈勾他们简单说了下寒意交代的事情。
<br/>　　一行人吃了点饭填饱肚子然后就出发了。
<br/>　　任务叫做明月湖。
<br/>　　是郊区一个不大不小的湖泊，之前附近有几个养鱼场。
<br/>　　林峰的任务就是拿着记录仪绕着这湖泊走一圈，确认这个地方的环境，然后把记录仪上交给军部就可以了。
<br/>　　“绕着走一圈？这么简单？林峰你可真会挑。”陈勾道。
<br/>　　“你想得到是简单，若是真的很简单，肯定也不会留给我们了，这个明月湖我估计附近都是丧尸之类的。”
<br/>　　林峰道。
<br/>　　一行人很快抵达了明月湖。
<br/>　　果然还真被林峰猜了个八九不离十，丧尸到没有多少，顶多五六十个，最主要的是，怪物太多了。
<br/>　　一眼望过去就可以看到三十多只变异狗。
<br/>　　这群变异生物显然把这个明月湖当做了巢穴来安家了。
<br/>　　“我去，这么多？”
<br/>　　陈勾李广等人趴在一边观察着。
<br/>　　“少说三十多只变异狗了，其中还有一个大家伙看起了似乎是这个族群的老大，实力也在六阶。”
<br/>　　“还不止变异狗，那边还有五十多头变异野猪，看到那边停在树上的鸟没有，一个个都有鸵鸟那么大。也最起码七八十个了。怪不得有人说水是生命之源。”
<br/>　　林峰道。
<br/>　　“这个鸟就是变异鸟，昨天袭击基地的就是这玩意，妈的这么多，其中还有好几个六阶的，握草，林峰，这个任务真特么坑啊，我们干脆放弃了算了，这简直就是去送人头。”陈勾道。
<br/>　　“是啊峰哥，我们也要量力而行啊，这几十头丧尸也就罢了，问题太多变异怪物了，其中不少等级比我和狗子狗还要高，这怎么打。”李广也胆怯了额。
<br/>　　“的确有点麻烦，不过这个任务有七天的任务时间，而且任务奖励很丰厚，除了一想老坛酸菜面之外，还会送我们十把步枪外加十颗手榴弹，我们先回去准备充分了再来。”
<br/>　　林峰道。
<br/>　　“我去送步枪？”
<br/>　　陈勾和李广震惊了。这可真的是好东西。
<br/>　　步枪现阶段威力还是很不错的，尤其是打丧尸和打人时。
<br/>　　一群人立即又是返回。
<br/>　　林峰之所以说准备，实际上也就是等几天空投看看有没有好东西。
<br/>　　第二天。
<br/>　　空投刷新。
<br/>　　直接十个D级空投。
<br/>　　林峰直接全部保存没有开启，准备留着融合升级更高档次的空投。
<br/>　　第三天。
<br/>　　这一次刷新了五个D级空投外加一个C级空投。
<br/>　　林峰直接把C级的空投开了。
<br/>　　得到了足足十发火箭筒，外加一箱子老干妈一箱子农夫山泉一箱子雪碧和一箱子红烧牛肉面。
<br/>　　“我去，C级的空投果然给力。”
<br/>　　林峰直接全部收走。
<br/>　　至于D级的，还是留着。
<br/>　　第四天。
<br/>　　运气一般般，只有五个D级空投。
<br/>　　第五天，十个D级空投。
<br/>　　第六天还是十个D级空投。
<br/>　　到了任务期限的倒数第二天下午，林峰开始开空投了。
<br/>　　林峰把足足四十个D级空投全部拿出来合成C级空投。
<br/>　　“系统，这玩意没有什么成功率的东西吧？你可别玩游戏那一套啊，合个东西还成功率，百分之七八十的成功率和假的一样。”
<br/>　　林峰道。
<br/>　　系统：放心，本系统不是腾讯出品。放心合成，都是百分百的。而且值得一提的是，若是你人品不错的话可能会出现跳级的小概率事情哦，也就是说你十个D级空投合成C级空投时，有小概率可以得到B级空投，当然这个概率和你的实力有关系，你实力越高，这个概率越大。
<br/>　　“还有这种好事？”
<br/>　　林峰直接开始合成。
<br/>　　四十个D级空投，妥妥的四个C级空投。
<br/>　　第一个C级空投就给林峰一个大惊喜。
<br/>　　《影分身之术》火影忍者木叶忍者村。
<br/>　　“影分身之术？我去，这不是鸣人的那一招吗？？”
<br/>　　林峰震惊了，还有这种骚操作？
<br/>　　“这一招好，特么的，无敌了我。”
<br/>　　叮！
<br/>　　恭喜宿主成功领悟《影分身之术》！
<br/>　　林峰继续开启第二个。
<br/>　　咔擦。
<br/>　　“这尼玛是无人机？？”
<br/>　　这一次空投箱内躺着个大家好，美国货，全球鹰军用无人机，世界上最先进的军用无人机，到现在都保持着许多军用无人机的世界纪录。
<br/>　　“牛逼，还挂了八发导弹，厉害了，有这空战幽灵，明天的任务绝对手到擒来了。”
<br/>　　林峰直接收入物品栏，无人机特别适用于去危险地区侦查或者轰炸的军事行动，也之所以被称之为空战幽灵。
<br/>　　第三个空投箱。
<br/>　　《天上地下唯我独尊八荒六合神功第二重。》
<br/>　　“第二重的？好玩意。”
<br/>　　叮！
<br/>　　恭喜宿主成功领悟《天上地下唯我独尊八荒六合神功》第二重！
<br/>　　“看看我现在力量多大了！”
<br/>　　林峰一拳轰了出去，一股比之前更强的力量打了出来，足足两万斤。
<br/>　　“两万斤？岂不是代表，在力量上，我现在媲美七阶的存在了？”
<br/>　　力量不是实力的全部，除了力量，速度和神经反应速度同样很重要。
<br/>　　但是在双方单挑的时候，力量还是起了决定性的作用，毕竟这年头不是谁都会八极拳不是谁都练过拳法刀法之类的，异能者也没那么多，最主要的还是纯粹靠蛮力解决问题，那么力量高不高，大体上就代表了一个人的战斗力高不高。
<br/>　　除此之外，蜜桃罐头十罐。香辣牛肉面一箱。红烧排骨面一箱。
<br/>　　“我去，这么多泡面？这年头，吃得起饭的人不多，但是吃得起泡面的还是有几个，只要用热水一泡就行了，顶多需要花费一些热水钱。”
