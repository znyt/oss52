　　苏莫离开大殿之后，很快便离开了内府，回到了外府。
<br/>　　他本想将自己的阁楼直接搬进内府，但想了一番，便放弃了这个想法。
<br/>　　火元峰这么大，总共也没几个人，直接在山上开出一座洞府，岂不是更为美哉。
<br/>　　随即，他便直接去寻找聂乾和韩盈，让两人一起随他到内府居住。
<br/>　　聂乾和韩盈闻听此事，顿时大喜过望，能留在内府修炼，这对于他们来说，当然是天大的好事。
<br/>　　不仅不用再担心廖云剑一事，还能得到火元圣师的庇护，对于修炼也是好处无穷，可谓是一举三得。
<br/>　　一般的外府弟子，虽然也可以进入内府，但并不能在内府久留。
<br/>　　除非那些内府的导师，向学府报备，招收弟子，才可以在内府常住。
<br/>　　可以说，火元圣师，虽然没有明说收苏莫为徒，但让他留在火元峰，便已经是个挂名的弟子了。
<br/>　　聂乾和韩盈，没有浪费时间，立刻便收拾一番，随苏莫进入了内府，来到了火元峰。
<br/>　　“苏兄，我们住在哪里？”三人站在火元峰上，聂乾满脸激动的问道。
<br/>　　这内府的天地灵气，比外府浓郁太多了，简直是修炼圣地。
<br/>　　就算不用灵石和灵晶，吸收天地灵气修炼，修炼十日，也抵的上外界一个多月。
<br/>　　“我们也不用建住所了，直接在山上开几座洞府吧！”苏莫说道。
<br/>　　“洞府也不错，有个容身之地便可！”韩盈说道。
<br/>　　随后，三人进入了火元峰的山腰部位，在一片草木茂盛的山谷中，开凿了三座洞府。
<br/>　　火元峰与其名字倒是颇为吻合，山上的草木，均是呈现赤红之上，蕴含浓烈的火属性精华。
<br/>　　山体之内火属性灵气更为浓烈，苏莫三人开凿出的山洞，都有滚滚热浪冲出。
<br/>　　开凿三座洞府，对于苏莫三人来说，实在是再简单不过了，只是片刻功夫，便完全搞定了。
<br/>　　苏莫的洞府，开凿的极大，方圆百余丈的空间，非常的宽敞。
<br/>　　除了一间宽敞的大厅之外，里面还开凿除了一座修炼室。
<br/>　　苏莫盘坐在修炼室中，感受着周身浓郁的火属性灵气，不禁暗想，若是司空炎来了此地，绝对不愿意走了。
<br/>　　少倾之后，苏莫抛却了心中杂念，开始了修炼，他要尽快突破到武帝境二重境界。
<br/>　　运转大混沌五行神诀，苏莫渐渐沉浸在了修炼之中。
<br/>　　所谓修炼无岁月，一个月的时间眨眼即过。
<br/>　　经过一个月的修炼，苏莫的根基更加的扎实稳固了，他便想冲击境界瓶颈。
<br/>　　他的修为，已达武帝境一重的巅峰，距武帝境二重之境，也就差临门一脚而已，冲过去便能突破修为。
<br/>　　不过，他目前下品灵晶可是不多了，只有五百余块，魔晶则是只有两三百块了，怕是很难一句冲过去。
<br/>　　不过，能不能冲过去，还要一试才能知道。
<br/>　　意念一动，苏莫进入了玉镯空间之中，盘膝坐在一堆乱石堆中。
<br/>　　手臂一挥，五百余块下品灵晶，以及七、八百块魔晶出现在了他的身前。
<br/>　　除此之外，还有几株灵草，和数十枚增加修为的丹药。
<br/>　　这些，已经是他全部的修炼资源了，全部是上次魔云海的战利品。
<br/>　　待这些资源用完之后，他就要再想办法收集资源了。
<br/>　　“吞噬！”毫不犹豫，苏莫立刻催动了战魂，开始吞噬这些资源。
<br/>　　霎时之间，所有灵晶、魔晶、灵草之内的灵气全部爆开，疯狂的向苏莫身上涌去。
<br/>　　轰隆隆！！
<br/>　　很快，苏莫体内便轰鸣不断，五枚神丹不断的震动，磅礴无比的灵气，被他炼化成了五行玄力，分别汇进五枚神丹之中。
<br/>　　这个过程在持续，虽然苏莫的修为已达武帝境一重的巅峰，但境界瓶颈却也坚固异常。
<br/>　　轰隆隆！！
<br/>　　轰鸣之声不断，五枚神丹均是光华璀璨，耀眼夺目。
<br/>　　冲击境界的过程，持续了约莫两个多时辰。
<br/>　　两个多时辰之后，灵气消耗一空，苏莫睁开了眼眸。
<br/>　　他，依旧没有突破到武帝境二重的境界！
<br/>　　这让他心中非常无奈。
<br/>　　不过，他也没有气馁，毕竟，修为达到了武帝之境，可不再是以前了。
<br/>　　像他这个级别的武者，在一个境界之内动辄都是一卡数年，甚至是数十年或数百年。
<br/>　　他想在一个多月之内，就突破境界，倒是有些好高骛远了。
<br/>　　虽然战魂可是不停的吞噬，但修为总要有一个沉淀的过程。
<br/>　　接下来的时间，他要多多赚取灵晶了，为接下来的突破修为囤积资源。
<br/>　　不过，该到哪里赚取大量的灵晶呢？
<br/>　　这个问题，倒要好好的问问聂乾和韩盈，想必两人定然比他清楚。
<br/>　　叹了口气，苏莫正欲离开玉镯空间，眼角余光便瞥见了不远处的巫娴三人。
<br/>　　上次苏莫突破修为，险些将三人吞噬。
<br/>　　现在，巫娴三人已经有所恢复，均是在远远的望着苏莫，个个眸中露出恐惧之色。
<br/>　　苏莫的修炼情况，他们都看在眼里，心中的震撼早已无以复加。
<br/>　　吞噬灵晶？
<br/>　　她们哪里见过如此的情况！
<br/>　　“你们过来！”苏莫没有急着离开，向三人呼喊了一声。
<br/>　　看到三人，他不禁动了心思，赚钱最快的办法莫过于杀人夺宝了。
<br/>　　在神武学府自然不可能随意的杀人夺宝，但去猎杀巫族之人，倒是个不错的选择。
<br/>　　巫娴三人闻言，身形均是颤.抖了一下，互相凝视一眼，便缓缓的飞了过来。
<br/>　　他们不敢反抗！
<br/>　　很快，三人便落在了苏莫的身前。
<br/>　　“问你们几个问题，希望你们如实回答，不然后果你们自己掂量！”苏莫扫视了三人一眼，冷漠的说道。
<br/>　　“你……你问吧？”其中一名个子略矮的巫族青年颤声说道。
<br/>　　“告诉我巫天御的一切情况！”苏莫道，他想要弄清巫天御的情况，才好出手救宏青璇。
<br/>　　若是宏青璇已经遭难，他会让巫天御后悔来到这个世上。
<br/>　　三人闻言眼眸一凝，不过，三人也知道苏莫打听巫天御是何意，就是因为巫天御抓了苏莫的一个朋友。
<br/>　　三人沉默了片刻，另一名略胖的巫族青年开口了，道：“天御大哥是我们巫族天冥星支脉的族人，从小天赋不凡，十八岁就达到了武王之境……！”
<br/>　　“我不想知道他的天赋！”
<br/>　　不待青年说完，苏莫立刻摆了摆手，打断了对方的话，继续问道：“巫族共有几脉？”
<br/>　　既然有天冥星支脉一说，那想必巫族还有其他支脉，以及核心的主脉。
<br/>　　巫族是苏莫的大敌，他想要问清巫族的具体情况。
