　　现在涂小安烦恼的是，灰妖王偏偏不出来跟他打，这让人很着急。
<br/>　　他就搞不懂，你堂堂妖王在怕什么，自己真的那么牛逼？将妖王吓得胆战心惊都不敢出来吗。
<br/>　　其实涂小安想法很简单，要是灰妖王都能收拾的了，那么白妖王那只刺猬就手一把掐了。
<br/>　　到时候杀过去抢亲，岂不是快哉而过瘾吗。
<br/>　　就算打不过，他也能衡量一下自己跟真正妖王之间的差距。
<br/>　　此时，忽然...涂小安脑海中系统的声音出现了。
<br/>　　“叮...恭喜宿主的战宠超等变异体三头邪蛇实力修炼圆满，三天后渡劫”
<br/>　　涂小安看到这条信息，古怪的看向九点钟方向，几十米通灵摄厉充满邪气的三头邪蛇在大吃四方，那片地带简直是鼠兽的人间地狱。
<br/>　　这小家伙是硬生生吃鼠兽吃到渡劫了？
<br/>　　而且还是三天后，比树精还抢先了一步。
<br/>　　鼠兽虽然不是每一只都是觉醒生物，但就算不是觉醒生物，也不是一般的老鼠，毕竟活在妖灵山脉，能普通的了吗。
<br/>　　这邪蛇，三颗蛇头大块朵颐的吃，吸收鼠兽的能量，成功把自己吃到渡劫了，当然，这要归功于它本身就已经是二十九级了。
<br/>　　这种战争，简直是邪蛇的福地，吃的越多，实力暴涨的越快，底蕴加深，
<br/>　　没多久，又是一道系统信息传来。
<br/>　　“叮...恭喜宿主的战宠超等生物黄金瞳修炼圆满，四天后渡劫”
<br/>　　涂小安愣了下，敢情那只凶煞的猫也没少吃鼠啊。
<br/>　　蛇是老鼠的天敌，猫更是老鼠天敌中的天敌。
<br/>　　黄金瞳的老虎之威，狮子之威更是一大利器，鼠兽在它的面前简直动都不敢动。
<br/>　　涂小安表示很欣慰，一个三天后，一个四天后，这天劫是轮番的来啊，被弄的挺忙的。
<br/>　　接下来他就等待自己的宝贝青枭传来渡劫的消息了。
<br/>　　但涂小安等了半天却没等到青枭要渡劫的消息，看来这孔雀是对老鼠肉不感兴趣啊，不和它的胃口。
<br/>　　无妨，涂小安给它弄一颗妖灵的金丹一吃，渡劫就可以来了。
<br/>　　如今涂小安证道后实力暴涨，就连深山大妖都看不上眼了，他的战宠若还在二阶，怎能跟的上他的节奏。
<br/>　　不过，这几个超等生物要是渡劫成功，也是实力暴涨，恐怕可以一跃成为深山大妖的级别了。
<br/>　　超等生物本身就是起点高。
<br/>　　蛇鼠大战依然还在如火如荼的继续，这种大规模的种族厮杀，没有几天几夜是结束不了的。
<br/>　　除非有一方主动的撤退。
<br/>　　不过看如今的局势，灰妖王是真的不准备露面了，他难道不要面子嘛。
<br/>　　这场世纪大战可是被妖灵山脉无数种族关注，若是败了，代价可是非常非常的严重。
<br/>　　灰妖王越是不露面，涂小安心中反而越是没底气，因为看不到的敌人才是最可怕的。
<br/>　　你永远不知道对方心里憋着什么坏，在准备什么大招。
<br/>　　隐隐间，涂小安有种不好的预感。
<br/>　　就在这时，他头顶上方空间扭曲了起来，一阵波动涟漪，激烈的爆响声不绝于耳。
<br/>　　灰妖王有动静了？
<br/>　　然而，涂小安看的到却是一条巨大的黑色大蛇腾空掠来，此蛇的气息极度强大，引的无数群蛇瑟瑟发抖，诚惶诚恐了起来。
<br/>　　这黑色大蛇看着并没有太出奇，但是散发出来的蛇类气息却携带着一股无与伦比的帝皇之气。
<br/>　　这种气息对蛇类可是最为致命的。
<br/>　　它的这股蛇类气息就连涂小安身下的泰坦巨蟒都不禁匍匐了下来，吓得大气都不敢喘，可见威力了。
<br/>　　旋即，这黑色大蛇横陈在了涂小安跟落子烟的面前，一阵光芒闪烁，变成了一位妖异的青年男子，脸上有着黑色花纹，很显霸气，一双眼眸闪烁幽然的绿芒。
<br/>　　妖异青年显化后，举手投足间尽显霸气，无数的蛇种因为他的出现匍匐在地，反而不小心死在了鼠兽的手上。
<br/>　　这蛇不简单啊，涂小安没想到如今还能出现一条非凡的蛇妖灵出来。
<br/>　　自带帝皇霸气，宛如一方诸侯。
<br/>　　目前为止，蛇潮中无数蛇种，十几位蛇妖灵，但眼前这位绝对是最可怕的蛇种，修为达到了伪妖王的层次。
<br/>　　让涂小安有点意外跟吃惊。
<br/>　　但他那种帝皇之气让涂小安有点别扭跟不舒服，感觉自己的蛇王威严受到了挑衅。
<br/>　　另外，涂小安明显察觉到对方现在才露面，显然不是来跟自己请安的，更像是故意搓一搓自己的锋芒锐气。
<br/>　　涂小安的想法其实很准确，只见妖异青年显化后，一双闪烁幽然的眼睛直接无视了涂小安，看向了落子烟，略带尊敬，微微躬身道：“离幽拜见姑姑”
<br/>　　光这一举动，眼中果然没有涂小安这个新晋蛇王。
<br/>　　落子烟看到妖异青年，就微微的蹙眉起来，淡淡道：“离幽，将你的帝皇气收一收，你难道不知道你的帝皇气息会对其他蛇种产生影响吗”
<br/>　　现在是蛇鼠大战，大量的蛇种身心收到干扰跟影响，转眼就会没了命。
<br/>　　“姑姑说的是，是离幽鲁莽大意了”
<br/>　　他稍微将自己的帝皇气息收了收，然后负手而立，最后将目光锁定在了涂小安的身上，完全没有准备拜见这位蛇王大人的意思。
<br/>　　到是有种针尖对麦芒。
<br/>　　其实妖异青年早就来到了猴儿山，比任何的蛇种来的都早，但他一直躲着没有出来，因为他想假借灰妖王的手，杀了这条抢它王位的蛇。
<br/>　　可奈何灰妖王愣是没出现，让离幽的好算盘落空，只能自己出手了。
<br/>　　不过无所谓，他有这个底气比这条证道的蛇强，虽然对方表现的很惊艳，但是离幽会让蛇族的群蛇知道，谁才是妖灵山脉最强大的蛇种。
<br/>　　“离幽，还不拜见我们蛇族的新晋蛇王”落子烟掀开眼，眉宇间似是凝着化不开的冰霜，又是提醒。
<br/>　　妖异青年闻言，并没有有所动作，而是看着落子烟一问：“姑姑，离幽不懂，什么时候我们蛇族的王立的那么随意，区区一条刚证道的蛇，就成了妖灵山脉蛇族的王，可以统领蛇族了”
