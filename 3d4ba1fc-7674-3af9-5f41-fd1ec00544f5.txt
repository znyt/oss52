　　“少阳，你怎么在这？”
<br/>　　身后响起小九的声音，叶少阳回过头，看着她穿着睡衣走过来，站直身子，伸了个懒腰，抿嘴笑了笑：“我好想你的。”
<br/>　　小九愣了一下，羞涩笑道：“不是刚才见过吗。”
<br/>　　“对你来说是刚见过，对我，却是两个月过去了。”
<br/>　　小九不解。叶少阳朝她走过去，说道：“我带你去一个地方，到那里了，再慢慢告诉你。”
<br/>　　他拉着小九回到了卧室里，用符印封住门窗，然后激活了山海印，带着小九穿越过去。
<br/>　　他们先是回到了宋朝，趁着夜色悄悄爬上附近的一座城楼，虽然不是长安城，但也是真正的古代，看着城市里的万家灯火，对现代人来说，别有一番风景。
<br/>　　因为只是看看，并不做任何改变，也不跟任何人接触，这种穿越行为是没什么危险的。
<br/>　　接着他们又去了元末明初，见识了蒙古和汉人的战争，因为太惨，看了一眼就走了，就这样不停穿越，去了很多地方，欣赏了很多不一样的风景，后来两人穿越到了民国，那时候的石城是民国首府，气象万千，两人手拉手在街上散步，看着夜晚的石城，来来往往的骑着自行车的学生，穿着旗袍的大家闺秀，留分头、穿笔挺中山装的男子。
<br/>　　小九曾经被吕洞宾封印几百年，不曾到过民国，因此看到什么都觉得新鲜，就是这些民国打扮的人都都她看好一阵子了。
<br/>　　对叶少阳而言，这本来是一场时空旅行，却意外地勾起了他的某一段心事。
<br/>　　“走，我带你去一个地方。”
<br/>　　叶少阳拿出山海印，先穿越回去，然后顺着自己的时间轴往前找，找到了1923年，想了一下，又往后数了两年，然后带着小九一起穿越过去……
<br/>　　1925年的杭州。
<br/>　　穿越时间，相对来说容易一些，最难的是穿越空间，山海印是时间的枢纽，没有瞬息移动的功能，但时空向来是一体的，其中有难以寻觅的规则。
<br/>　　好在徐福有几乎无限的时间，用了几百年来摸索其中的规则，总算让他弄明白了，利用时间的规则，来间接实现空间的穿越。
<br/>　　这其中的原理，极为复杂，叶少阳比关在时间虚空中的两个月，有几乎一个月都用来学习这门技能，好在他足够聪明，总算是学会了，回顾这门技能的复杂程度，不禁感慨，如果没有徐福，就算有山海印，自己也不知道要琢磨多少年，才有机会学到这个技能，很可能还没有学会，自己就在实验中把自己搞玩完了。
<br/>　　“我们去哪里？”小九一脸的新奇，她虽然也活了千年，但大部分时间都被封印，而且也从没有过这种奇异的体验。
<br/>　　“带你去吃好吃的。”叶少阳带着她，在杭州城里逛了一圈，街上买臭豆腐糖人冰糖葫芦什么的很多，因为没钱，两人也就看看罢了，最后来到一条最繁华的大街上，因为是中午，人来人往的。
<br/>　　叶少阳拉着小九，慢悠悠地往前逛，最后来到一座大酒楼的楼下，酒楼叫仙云阁，是三层的楼房，装修得很古风，里头宾客满堂。
<br/>　　两人刚站住，立刻有伙计从里头出来热情地揽客。
<br/>　　一楼是大堂，二楼是雅间，不过有最低消费，是一个大洋，叶少阳带了小九上到二楼，是屏风隔出来的若干小房间，挑了临窗的桌子，朝外面能看到街上热闹的景象。
<br/>　　“比我们那个时候，好像还要热闹些。”小九看着下面来往的人群，发表看法。
<br/>　　“过去的人娱乐项目少，也没什么宅男宅女，就喜欢逛街，住在城外的农人也要经常进城来买东西。还有，这时候全国就那么几个大城市，三教九流的人，闲不住的都会到这几个大城市来讨生活，不过出了城就不一样了，那些村庄什么的，比我们那时代的人少的多。”
<br/>　　小九笑着看他一眼，“说的好像你很了解似的。”
<br/>　　“那当然，我可是在这个时代亲自住过的。”
<br/>　　说完对招呼一直愣在边上的小二，让拿菜单过来。
<br/>　　这仙云楼倒是很讲究，菜单是写在一张扇子上的，字迹工整有型，小二见叶少阳盯着字迹出神，立刻眉飞色舞地说起这上面的字是找金陵第一笔某某写的，还是看他们老板娘的面子等等。
<br/>　　叶少阳要了东坡肉、西湖醋鱼、麻婆豆腐等几个有名的杭帮菜，小二问要喝什么，没等他们开口，说起道：“二位客观穿的这么讲究，一看就是新青年，我们店的师傅最近去上海，学会了一种茶，叫咖啡茶，上海的新青年都在喝，两位客官要不要尝尝？”
<br/>　　咖啡茶……
<br/>　　“行吧行吧，来两杯我尝尝。”民国的茶他喝过，但咖啡还没喝过，不知道跟自己那个时代的是不是一样。
<br/>　　小二麻溜地跑了，然后又麻溜地回来，捧来两瓷杯，在他们面前摆好，说是咖啡要现场制作，然后……拿出两个糖块放在杯子里，现场倒上开水……
<br/>　　叶少阳目瞪口呆，赶紧所谓的“上海咖啡”，就是咖啡糖泡出来的啊。
<br/>　　“二位品尝着，菜一会就来。”
<br/>　　小九望着叶少阳，捂着嘴笑。
<br/>　　不一会工夫，菜一盘盘上来，上西湖醋鱼时，小二表示这是老板娘亲自烧的。
<br/>　　小九是妖，但本质上是狐狸，也是吃荤的，只是吃的少，每样菜都尝了一点，“这鱼做的真不错！”小九很罕见地称赞了一声，夹了大块的鱼给叶少阳。
<br/>　　叶少阳尝了一口，虽然佐料很多，但他还是一口吃出了那熟悉的味道。
<br/>　　“你带我来这做什么，不光是为了游玩吧，我看你是特意来的。”小九道出心中的疑惑。
<br/>　　叶少阳笑笑，“先吃。吃完就知道了。”
<br/>　　咖啡实在喝不了，叶少阳又要了一壶黄酒，两人吃着喝着，好不惬意。吃完之后，叶少阳拍着肚皮，把小二叫来，一边拿牙签剔牙，问多少钱。
<br/>　　小二算了算，道：“零头就不说了，您给一两银子吧。”
<br/>(本章完)
