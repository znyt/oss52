
<br/>　　<br/>　　    车祸现场一片狼藉，几乎贴着路边的一排车都给怼的不成样了，那辆挂车最后是靠着惯性才停下来的，司机从车篓子里跳下来的时候脸都绿了。<br/>　　<br/>　　    翻过隔离带的陈重脸上的冷汗顺着面颊就往地上掉落，前胸都湿了一大片，不是天气热的，而是因为被吓的。<br/>　　<br/>　　    连续两次变故，前后都只差了几秒钟他们就从鬼门关上逛了一圈，只要晚那么一点点，此时肯定都死于非命了。<br/>　　<br/>　　    陈重拿着电话的时候手一直在哆嗦，里面王惊蛰喊了好几声他才听见。<br/>　　<br/>　　    “我，我没事，刚，刚才撞车了······”<br/>　　<br/>　　    王惊蛰看了眼陈臣的面相，这时候她那边脸的异样才淡去，这就表示陈重经过两次劫难之后，这个坎已经过去了。<br/>　　<br/>　　    王惊蛰现在都可以肯定，问题是出在陈重的那辆车，或者是司机的身上了，人家是下了个连环套，一环扣着一环，直接把陈重躲过去的机会全都给堵死了。<br/>　　<br/>　　    但可能对方也没想到的是，有个王惊蛰忽然伸出来天使的翅膀，把陈重从鬼门关里给拉了回来。<br/>　　<br/>　　    “好，你们留在原地吧，等我们这就过去了”王惊蛰疲惫的靠在座椅上，揉着发酸的脑袋说道。<br/>　　<br/>　　    “你不是说我爸是受什么祖辈的蒙阴，不会被人算计的么？”陈臣说话的时候都带着哭腔了，就这一段时间她相当于是在冰火两重天里泡着呢，两次她差点就跟陈重天人两隔了。<br/>　　<br/>　　    王惊蛰闭着眼睛说道：“我说的不会被算计，是像你那样，不会被孤魂野鬼给缠上，但并不代表他就不会死于非命了，往前推几十年有多少大人物都死于非命，并不只是因为命里到寿了，更何况你爸了，再说了他这两次要是没躲过去的话，都算是意外身亡，你肯定不会找到一点人为的算计，死了也就白死了”<br/>　　<br/>　　    陈臣抹了把眼泪，抿着嘴唇脚下狠狠的踩着油门，几乎十一路超速往川中和双流的方向赶了过去。<br/>　　<br/>　　    车祸现场，陈重和老马在一旁低语着，两人的脸色都很难看，他们已经各自联系人过来接应了，甚至还让警方也赶了过来。<br/>　　<br/>　　    这条车道的对向车道，停车带上停着一辆黑色的车子，紧闭的车窗上贴着反光车膜，看不清里面的状况。<br/>　　<br/>　　    “陈重没死，那堆石头没砸到他的车子，后来撞过来的那辆挂车也没有怼上，他现在就在路边站着呢”吴洋歪着身子一手搭在方向盘上，一走拿着电话，叹了口气道：“他的命怎么这么大啊，连续两次都给躲过去了？”<br/>　　<br/>　　    “他身边还有什么人？”电话中有人低沉着嗓音问道。<br/>　　<br/>　　    “袍哥的那个老马，还有他的司机，没了”<br/>　　<br/>　　    “你确定，没有其他的人了？”<br/>　　<br/>　　    “我正看着那边呢，陈重就开了一辆车去双流的·····”<br/>　　<br/>　　    电话里的人沉默片刻后，说道：“那就是他命大了，不是有人替他挡了两煞，也不算奇怪了，陈重这些年没少往出散财，做过不少慈善，再加上他也从来不干什么伤天害理，天怒人怨的事，阎王爷没在生死簿上为他写上一笔早逝也算正常了”<br/>　　<br/>　　    吴洋拉了下手扣，里面放着一把仿五四：“要不，我过去直接干掉他？”<br/>　　<br/>　　    “直接动手就太蠢了，陈重要是死于意外还没什么，你要是这么给他杀了，他家里的几个现存的老人和在位的大哥就该不依不饶的了，查下去的话我们就不太好过了，总有一些蛛丝马迹会露出来的，这次就这样吧，算他命大给捡回来了，以后再想办法就是了”<br/>　　<br/>　　    “行，那我回去了，这边暂时也没我的事了”吴洋挂上电话，发动车子后，从对向车道离开了。<br/>　　<br/>　　    临走的时候，吴洋眯着眼睛盯着陈重，挺无奈的摇了摇头：“富贵命啊！”<br/>　　<br/>　　    十几分钟后，川中的警方先到场了，封路，分散车流，然后先把肇事的挂车司机给控制了起来。<br/>　　<br/>　　    司机也没跑，这种情况下就是突发事件，顶多算他超载了，只要没毒驾，酒驾其他的也没什么大事。<br/>　　<br/>　　    警方初步了解了下状况后，一个支队长过来跟陈重说了下：“司机是正常驾车，没有毒驾和酒驾，排除这两点后他基本就属于疏于观察了，人也是本地的，司机驾龄七年多左右，没有什么故意肇事的状况，陈先生你看······”<br/>　　<br/>　　    陈重这也没心思跟警方掰扯这个，摆了摆手后说道：“你们看着办吧，到时候把结果送到我公司就行了”<br/>　　<br/>　　    警察走了后，老马跟陈重说道：“那个叫王·····惊蛰的人，你之前说是跟着齐泉来见你的？”<br/>　　<br/>　　    陈重说道：“马有失蹄，人有失眼，我这是看走眼了，面前有一尊菩萨没拜啊”<br/>　　<br/>　　    这时候，只要不是太傻，陈重就该明白那个几天前跟着齐泉去家中，他还以为是个马仔的年轻人，是肚子子里藏了绣花针，得是相当有货的了。<br/>　　<br/>　　    他现在叫啥？<br/>　　<br/>　　    这叫运筹帷幄于千里之外啊，隔着几百公里呢，就知道自己这边出啥状况了，这个年轻人要是错过了，陈重都得扇自己几巴掌。<br/>　　<br/>　　    到他这种身份和地位的人，自然之道世间有些人懂风水通阴阳，有些事是完全超出了常人所知范畴的。<br/>　　<br/>　　    陈重还知道，当年建国时京城广场那一片的地砖就是被人用心布置过的。<br/>　　<br/>　　    他还知道，京城有个国运大阵。<br/>　　<br/>　　    也知道在申城陆家嘴，有个风水阵，下面压了龙头。<br/>　　<br/>　　    这些年陈重也见过不少高人，但如今天这样，隔着几百公里能连救他两次的，还真没见到过。<br/>　　<br/>　　    陈重庆幸的感叹了一句：“时也命也，运道来的太让人措手不及了。”
