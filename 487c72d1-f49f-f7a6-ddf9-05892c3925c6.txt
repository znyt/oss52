　　佟千雪走出浴室，深吸一口气，平复了一下心情后，走出卧室去开门。
<br/>　　佟千雪知道的虽然不少，可她说到底也就是个理论派，用手给男人那个也是第一次。即便跟慕容飞之间该发生都发生了，该做的事情也全都做了，可是让佟千雪用手帮慕容飞解决他的生理需求，还是让她羞红了脸，俏脸都快要烧起来一样。
<br/>　　“果然陷入爱情中的女人智商都为零。”
<br/>　　佟千雪嘴里轻声嘀咕了一句，在自己脸上轻拍了几下，然后把门打开了。
<br/>　　黄虹看到佟千雪把门打开，顿时松了一口气，提着一袋子东西走进客厅，对佟千雪说道：“你怎么不开门也不接电话呢，我还以为你出什么事了？”
<br/>　　“刚才在浴室，没听到。”佟千雪时脸色微红的回答道。
<br/>　　黄虹听到从卧室浴室中传出来的水声，扭头看向玄关的方向，发现了慕容飞脱在那儿的鞋子，对佟千雪问道：“我家那臭小子回来了？”
<br/>　　佟千雪点了点头。
<br/>　　“那你们刚才在干什么！？”黄虹语气突然间就变得严肃起来，盯着佟千雪。
<br/>　　佟千雪被黄虹盯得脸上有些发烧起来，刚才在浴室里的事情，毕竟是不可描述的，自然是不能在自己婆婆面前漏了陷。佟千雪连连摆手，说道：“您想多了，我们什么都没做，他来的时候淋湿了，我帮他放了水，让他洗澡。估计是放水的声音太大了，所以没听到门铃和电话声。”
<br/>　　黄虹又是盯着佟千雪看了一会儿，才是将信将疑的点了点头，把菜拎进了厨房，然后回到客厅坐在沙发上，颇有一种等着慕容飞出来要审问他的架势。
<br/>　　大约过了一刻钟后，慕容飞擦着头发，踢踏着一双拖鞋从卧室里走了出来，看到坐在黄虹客厅里，故作惊讶的说道：“妈，你怎么来了！？”
<br/>　　佟千雪坐在一旁悄悄向黄虹递眼色，慕容飞上前一步，背对着佟千雪向她做了一个“OK”的手势。
<br/>　　“说，你们两个刚才在浴室里干嘛了？”黄虹跟审讯疑犯似得，盯着慕容飞问道。
<br/>　　“浴室里当然是洗澡，还能干嘛？”慕容飞眨巴着眼睛，一脸无辜的看着自己的老妈，回答道：“我来这里的时候被雨淋湿了，千雪把我着凉感冒，就放水让我洗澡，刚放完水，千雪就听到你声音了赶紧出来开门了。”
<br/>　　慕容飞受过这个世界上最严厉的审讯和反审讯训练，面对自己老妈这种低级别的审问，慕容飞要是不能做到面不改色心不跳，那就可以从龙隐部队滚蛋了。
<br/>　　黄虹把慕容飞和佟千雪的回答一对照，并没有发现问题，也就打消了心中的疑虑，对慕容飞问道：“你小子回来了，怎么不回家来这里了？”
<br/>　　“这好不容易有了老婆，当然是先来看老婆了。”慕容飞拧开茶几上那瓶佟千雪喝了一小半的矿泉水，喝了一口说道：“要是一个不当心老婆跑了就没了，老妈又不会跑了。”
<br/>　　听到慕容飞的回答，黄虹顿时满脑袋的黑线。这个回答……，嗯，没毛病！
<br/>　　黄虹本来想要给佟千雪做完中午饭再离开，可是这几天天气闷热，加上佟千雪刚怀孕，没什么胃口，很多时候做好的菜，只吃掉一小部分，绝大多数都是倒进了垃圾桶。见佟千雪的确不要自己留下做午饭，黄虹便打算要起身离开。
<br/>　　“你！”
<br/>　　黄虹刚站起身，就对着坐在一旁椅子中的慕容飞勾了勾手指，用命令的语气说道：“跟我回家！”
<br/>　　“为什么？”
<br/>　　慕容飞一脸不情愿的看着自己老妈，显然是不愿意跟着黄虹回家。
<br/>　　“什么为什么？”黄虹眼睛一瞪，伸手就要去揪慕容飞的耳朵，但是却被慕容飞给躲开了，怒声道：“你留在这里想干什么？我可告诉你，千雪现在才刚一个多月，你别动歪脑子，要是害得我孙子没了，老娘跟你没完！”
<br/>　　慕容飞嘴角一阵抽搐，这才一个多月，孩子都没成型呢，哪里能知道是男是女？！
<br/>　　“我知道千雪怀孕了，当然明白现在不能那啥，我又不会乱来。”慕容飞坐在沙发中，丝毫没有起身的意思，说道：“而且她一个人在这里住着你也不放心，我在这里至少能照顾一下。”
<br/>　　“你一个男人懂什么？能照顾什么？”黄虹皱眉道：“要留也是我留下。”
<br/>　　“你留下有什么用？”慕容飞反驳道：“关键时刻需要去医院，我会开车，你会吗？”
<br/>　　“我就不会打电话啊？”黄虹瞪着慕容飞说道。
<br/>　　“拉倒吧。”慕容飞不屑地撇嘴，说道：“等到救护车或者出租车来，黄花菜都凉了。”
<br/>　　“没事的妈，就让他留在这儿吧。”佟千雪这个时候开口，结束了慕容飞和黄虹母子间的唇枪舌战，对黄虹说道：“家里有个男人总归是安心一点，而且我自己就是医生，知道怀孕期间要注意些什么，不会乱来的。”
<br/>　　有了佟千雪的话，黄虹才算是放弃了让慕容飞跟着她回家的打算，离开前再次对着慕容飞警告了一遍。
<br/>　　等到黄虹脚步声走远，佟千雪才算是松了一口气，对慕容飞说道：“刚才你出来的时候，我就怕你露馅，没想到你还挺聪明的，跟我说得一样。”
<br/>　　“你真当我能算到你会怎么跟我妈说？”慕容飞靠在沙发中，说道：“你们刚才说得话，我在房间里都听到了，我是听完你们对话，再去洗澡的。”
<br/>　　“那你……”佟千雪欲言又止的看着慕容飞。
<br/>　　慕容飞自然是知道佟千雪想要问什么，抬起自己的右手，说道：“怎么说，我也是个有了老婆的人，这有了老婆还需要靠‘五姑娘’来解决生理问题，也未免太悲催了，所以我以强大的定力忍了下来。”
<br/>　　“就这么憋着？”佟千雪美眸扫了扫慕容飞，问道。
<br/>　　慕容飞正视着佟千雪，说道：“你可别在挑火了，真要把我挑起火来，我自己都害怕。”
