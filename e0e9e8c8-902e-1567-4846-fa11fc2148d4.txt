<br/>
<br/>        江俞轩要赶到北京参加耗材上市的推广会，从顾恩赐办公室出来将凌辰送到了陈子霖的公司，对凌辰说：“希望或者明天早上估计东西就会退回来。你安心的等待吧。我要去北京，后续的事情你们自己解决。”
<br/>
<br/>        “谢谢江总。”
<br/>
<br/>        “顾恩赐他们目前还不知道你们的身份，等会你重新过去一趟，就说你们想在下面小县城做批发，你们已经认识到错误了，交点罚款吧。”江俞轩看着凌辰接着说：“目前你们只能抽身而退，他们已经有了防范，你们什么也查不到。”
<br/>
<br/>        “行，我等会就过去，另外我会和董亚子离开西城去贵州。给你们添麻烦让我很愧疚。”凌辰叹了口气，这一次这么好的机会泡汤了，后期怕是会越来越艰难了。
<br/>
<br/>        凌辰其实是从来都不喜欢欠别人人情，有事情自己解决，哪怕再艰难他都很少去麻烦人的，她每次出差都是独来独往，从来不让代理商接送，这一次要不是宁雅叮咛要她替宁雅看看陈子昂，他是绝对不会麻烦陈子昂的。
<br/>
<br/>        “我们是朋友，你不麻烦朋友你麻烦谁？不要害怕麻烦人，好朋友都是麻烦出来的，这叫什么你知道吗？叫同甘共苦。”江俞轩哈哈哈大笑。
<br/>
<br/>        凌辰人真是不错，仗义也大气，除了整天爱装冷冰冰以外都很好。
<br/>
<br/>        当然，还很痴情，就是不知道有没有结果。
<br/>
<br/>        “哈哈哈，好，那我以后就多找找你的麻烦，给你添添乱，在陈子昂的面前多说说你的坏话。”凌辰大笑。
<br/>
<br/>        车到山前必有路，虽然这一次被那个什么顾恩赐摆了一道，后期要让他们一锅端，就先让他们嚣张一段时间。
<br/>
<br/>        “我得给陈子昂打个电话。”凌辰拿出电话，准备给陈子昂打电话。
<br/>
<br/>        “又不是什么大事情，别打了，明天该干啥干啥，顾恩赐其实并不傻，他知道该怎么做。”江俞轩阻止了凌辰给陈子昂打电话，陈子昂今天的事情肯定多，不想让她分心。
<br/>
<br/>        西城有陈子霖和易倾秋他们就行了。
<br/>
<br/>        江俞轩将凌辰放到公司门口就赶往机场，北京，我回来了！
<br/>
<br/>        凌辰没有给陈子昂打电话，陈子昂却将电话打给了陈子霖，她想知道结果。
<br/>
<br/>        “哥，怎么样？事情解决了没有？是什么结果？”陈子昂站在自己的办公室前看着远去北京街道的车水马龙，看着自己的厂房，这是她努力的象征，是她智慧的结晶。
<br/>
<br/>        “俞轩带着凌总去找顾恩赐了，还不知道是什么结果。”陈子霖准备找找他四姐，看看能不能想办法，或者是找三姐夫。
<br/>
<br/>        找四姐，四姐会委婉一些，找三姐夫，三姐夫会捅娄子，只是要看凌辰想不想捅娄子，还是等凌辰回来在商量。
<br/>
<br/>        “直接找顾恩赐去了？”陈子昂对顾恩赐不是非常理解。
<br/>
<br/>        “是啊。”
<br/>
<br/>        “那你们就等消息吧，江俞轩不打无把握的仗。”不对公而对私，江俞轩打的什么主意？难道是要凌辰暗中调查，不予他们正面去冲突？
<br/>
<br/>        “嗯，江俞轩今天去北京，我们这里你不用担心，明天该干啥就干啥，自古邪不压正！”陈子霖才不怕举报，趁着这个机会好好闹腾一下，让一些人收敛收敛。
<br/>
<br/>        “好！”陈子昂知道江俞轩会将事情解决了也就不用担心。
<br/>
<br/>        凌辰转身又到了某部门，这一次凌辰的态度特别好：“领导，您看，我是第一次想做这个生意，原本想着拉回我们小县城去卖，你不知道我们小县城一过事情，几十箱的喝，我寻摸着能赚钱，所以就囤了些东西，再说，我真是买的，我自己又不会造，您们大人大量，就将人家的耗材放行了，酒什么的我就不要了，我给您交一点罚款，这个事情就了了好不好？”
<br/>
<br/>        “你不是厂家的了？”小头目说。
<br/>
<br/>        “嗨，你看我这样子像厂家的吗？再说了，我要是厂家的我不直接找代理商去了？我还做这些事情干嘛啊？”凌辰苦巴巴的样子让小头目有点想笑。
<br/>
<br/>        “认错太度不错，不过你这事情我拿不了注意，我还得请示领导，你先回去等着吧。”
<br/>
<br/>        “领导，您看啊，这个耗材是我朋友的，人家明天要给客户送货呢，他们公司其实是很大的，全国联动，能做到这一步也不算是小公司了，如果真有问题人家不会大张旗鼓的推向市场的，况且我的这个东西你知我知，什么猫腻你心里大概也清楚，别到时候你就是个背锅的。”
<br/>
<br/>        小头目愣了一下，看了看凌辰，凌辰拿出一根烟，递给小头目，小头目接过去后，凌晨拿出打火机“啪”给小头目点着烟。
<br/>
<br/>        小头目吸了口烟：“烟不错啊，什么牌子的？”
<br/>
<br/>        “1916，您觉得可以了，我给您送两条。”
<br/>
<br/>        “别，等以后再说。”小头目狠狠的吸了一口烟，然后将烟掐灭看了凌辰好会，盯得凌辰都不好意思了。
<br/>
<br/>        “你先出去等着吧，我打给电话。”
<br/>
<br/>        ........
<br/>
<br/>        “周总，那个某公司的人又来了，这一次态度很好，说是原本想做生意，他对这个行业也不是很懂，愿意交点罚款，货也不要了，让把耗材退还给人家。您看，是不是退给人家，这个事情就了了？在深究下去就没有意思了。”小头目心思活络，凌辰的几句话将他说动了。
<br/>
<br/>        “再拖几天！”对方“砰”的一声挂了电话。
<br/>
<br/>        “泥玛，拖下去出事情了谁负责啊？”小头目无奈的放下电话，靠在椅背上，他就是一个小人物啊。
<br/>
<br/>        过了一会电话响起来了。
<br/>
<br/>        “你好！哦，周总，您好！您说，好好好！什么我亲自开车送过去？好吧！”小头目挂了电话，无语的看了看天，不，是看了一眼头顶的墙壁，什么才是出头之日啊？
<br/>
<br/>        “那个凌辰，你过来。”小头目喊凌辰。
<br/>
<br/>        “您好，是有结果了吗？”
<br/>
<br/>        “你的本事挺大嘛，还让我将东西给你送回去，走吧，我送你回去，不过你的这些东西会被销毁的，你就别想了。”
<br/>
<br/>        “不想了，不想了，吃一堑长一智。”
<br/>
<br/>        小头目开着车将耗材和凌辰送到了陈子霖的公司。
<br/>
<br/>        “让你们的人来卸货。”小头目烦躁的说。
<br/>
<br/>        “好嘞，谢谢您，留个电话？改天请您吃顿饭。”
<br/>
<br/>        “把你货都没收了，你还请我吃饭？”
<br/>
<br/>        “我知道这不是你的本意，人生在世，不是被这个踩就是被那个压，我们是同一种人。”
<br/>
<br/>        “哼，我才不会和你是同一种人呢，奸商！”小头目鄙夷的看了一眼凌辰，但还是将电话留给了凌辰。
<br/>
<br/>        凌辰一笑：“我会安排人给你送两条烟，后会有期！”
