　　这一声师兄，让叶少阳感到了他的无礼：茅山与龙虎山关系一直很近，辈分是一起算的，龙虎山人比茅山晚两辈，张无生实际上是叶少阳的师侄，但因为年纪差距大，两派一向有个潜规则，就是尽量不按真实辈分，只按年纪来叫。
<br/>　　所以叶少阳称呼张无生为师叔，在正式场合上，双方一般都是称呼对方的职位，如果必须以辈分相称，那是必须要按实际辈分的。
<br/>　　那么，慕寒就要叫自己师祖，如今他却公然叫自己师兄，当然叶少阳不是在乎这个的人，只是通过这件小事觉察到了他内心的情绪。
<br/>　　是不甘。不甘落后给自己。
<br/>　　“我不是天选之子。”叶少阳诚恳说道。
<br/>　　慕寒听了，点头说道：“叶师兄是个老实人。但是……你不是谁是？”
<br/>　　叶少阳这才坐直了身子，凝视着他，问道：“你到底想说什么？”
<br/>　　“世人都说你是天选之子，你的未婚妻是转世鬼童，叶师兄方才所言，倒是令我们吃惊啊，他们也许会觉得你在谦虚，但我知道，你说的是实话。”
<br/>　　“慕寒，不要绕弯子了，你到底想说什么，你一个年轻人，怎么说话这么不利索！”慈心师太尖着嗓子嚷道。
<br/>　　慕寒看了她一眼，又看看叶少阳，道：“我。”
<br/>　　叶少阳怔了一下，问他：“你？”
<br/>　　“我，我是天选之子。”
<br/>　　此言一出，在场所有人都惊呆了，没一个人说话，用一句小学课本上的话来形容的话，就是连针落在地上的声音都能听得一清二楚。
<br/>　　“哈哈……”
<br/>　　慈心师太狂笑起来。
<br/>　　叶少阳朝慕寒身后看去，连龙阳真人在内的几个龙虎山弟子，听见这话，脸上也是难以掩饰的震惊。他们也不知道……
<br/>　　“大言不惭，张掌教，你可真是大言不惭！”王道乾似乎忘了之前自己是怎么丢人的，拍着腿大笑起来。“从前，我只当叶少阳是法术界最狂妄的人，如今看来，他比你还是谦虚多了啊。至少他不会无中生有……”
<br/>　　慕寒剑眉一竖，原本内敛的眼神一下子变得锋利起来：“你说我无中生有？”
<br/>　　王道乾被他眼神看的心头颤了一下，冷哼一声，刚要开口，慕寒道：“你方才已经吃过亏了，我劝你还是不要说话了，免得更加自取其辱！”
<br/>　　“嗬，你好大口气，你一个刚即位的后生晚辈，如此无礼，贫尼今天要好好教你怎么做人！”慈心师太再忍不住了，一巴掌拍在桌子上，冲进了道场。
<br/>　　慕寒一笑，手一抖，亮出了龙虎山镇山法器九天元辰尺，指着慈心师太。
<br/>　　慈心师太再不多言，手捧玉净瓶迎上去……
<br/>　　叶少阳跟随大家一起，两眼一眨不眨盯着战局，这是慕寒第一次出手，虽然是慈心师太主动出战，但叶少阳看得分明，这一战实际是慕寒挑起的。
<br/>　　面对实力比王道乾略高一头的慈心师太，他看上去丝毫不乱，这让叶少阳也产生了强烈的好奇，他到底有多强的实力？
<br/>　　“轰……”
<br/>　　两道力量碰撞在一起，爆发出强烈的反噬之力，如一道疾风吹过。大家定睛看去，战场之中，慈心师太与慕寒面对面站立，慕寒手中的九天元辰尺，顶在了慈心师太的额头上。
<br/>　　三秒钟后，慈心师太双腿一软，眼看就要倒下，慕寒上前一步，在她肩膀上托了一下，脸上是自信满满的微笑：“师太，承让了。”
<br/>　　结束了……在场众人刚做好看一场精彩打斗的准备，这战斗就结束了。
<br/>　　十几秒钟，三个回合左右，慈心师太就败了，而且败得如此彻底。
<br/>　　她的实力，在法术界属于一线，就算是昔日的张无生，也不能太轻松的战胜她，但这个年轻人用了十几秒钟就做到了。
<br/>　　一代强者，就这么横空出世了，以一种谁也没想到的方式。就在昨天，除了龙虎山之外，外界还没几个人听过他的名字，今天，当着大伙的面，他用最直白的方式让自己快速成名了。
<br/>　　“好厉害啊……”瓜瓜贴在叶少阳耳边，发表着感慨。
<br/>　　又是一个宿命对手？
<br/>　　慈心师太脸色煞白地从慕寒手下挣扎出来，在自己几个弟子的搀扶下，回到了座位上。
<br/>　　慕寒神色不再像之前那么雍容优雅，双眼中充满了凌厉和不可一世，手持九天元辰尺，面对人群，一点点指过去，冷冷道：“还有谁不服！”
<br/>　　指到叶少阳的时候，九天元辰尺停留了一下，一时间，所有目光都聚集在了他的脸上，大家都明白，在场这些人里，除了他，再没有谁能跟慕寒一战了——四宝或许可以，但他是叶少阳手足，自然不会抢这个风头。
<br/>　　叶少阳很反感慕寒的作法，刚要开口，坐在自己身边的静慧师太不满地说道：“张掌教，你的实力，我们是见识了，但我法术界从来都是以德服人……”
<br/>　　“好一个以德服人，我却以为，这一向是实力不济的借口。”慕寒淡淡一笑。他这话说得很不客气，对方又是自己尊长，静慧师太一听也很生气，想要理论，叶少阳在她胳膊上按了一下，望着慕寒，淡淡说道：“你觉得你很厉害是吧？”
<br/>　　“不敢，我从小隐修，连龙虎山弟子都不知道我的存在，比起从小就名震天下的道风和你，自然是差得远，我只想告诉你们，这天下强者，不止你们茅山一家！”
<br/>　　叶少阳耸耸肩，“我从来没这么想过。”
<br/>　　边上静慧师太怕他们再说下去要打起来，接过话头，冲慕寒说道：“纵然我们承认你手里强大，可这跟天选之子有什么关系？”
<br/>　　后面也有人不爽慕寒这样盛气凌人，便拿叶少阳来压他，附和道：“没错，我们一向都知道天选之子是叶少阳，什么时候成了你了！”
<br/>　　“诸位不信，那便随我移步后山。”慕寒叫来一众弟子，跟随众人，免得有人半路逃掉，自己在前面带路，绕过正殿，往后面山谷中走去。
