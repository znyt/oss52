　　高少辉一下子面色难看至极，恶狠狠的看着旺财：“你觉得他能放过你？我们两个楚涵谁也不会放过。”
<br/>　　“现在怎么办？”旺财急了：“咱们这药和食物、水什么的，还给不给他？”
<br/>　　“给，扔下去。”高少辉说罢，直接打开机舱的一条缝，将一个包裹从高空投放。
<br/>　　楚涵还在思考眼前这一切是怎么回事，便看到天照3000上一个包裹被扔了下来，几乎是下意识的反应，楚涵一跃而起接住，而后在地上利落的翻滚一圈，将包裹牢牢护在怀里。
<br/>　　只是当他做完这一切后，又忽然觉得少了什么，下一秒他的脸一黑。
<br/>　　“旺财，我的修罗战斧呢？”楚涵的声音传递到旺财大脑里，带着怒意。
<br/>　　旺财吓的一下子捂住嘴巴，同时拼命给高少辉打手势让其快走。
<br/>　　“你说话，他听不见。”高少辉嫌弃的开口。
<br/>　　旺财小心翼翼的出声道：“问我斧头在哪呢！”
<br/>　　高少辉一愣，道：“对，你把他斧头藏哪了？”
<br/>　　旺财黑脸道：“你少问这么多废话，快走。”
<br/>　　楚涵没有得到旺财的回应，只看到天照3000一下子飞远消失不见，顿时周围一片安静，剩下的只有一整座孤岛。
<br/>　　打开包裹，楚涵看到其内的一堆药品和食物后，很快便推理出了大致情况。
<br/>　　他被高少辉和旺财合力限制在了这座岛上！
<br/>　　啪！
<br/>　　楚涵重重的将包裹扔在地上，有种极大的憋屈感，他的高烧已退，思维清醒，但他根本没办法离开这里，四面都是海，重要的是他根本不知道这里是哪片海域，距离最近的临海城市有多远？
<br/>　　游过去简直是空谈，高少辉可是用一天便能绕地球好几圈的天照3000将他扔在这里，所以这个孤岛说不定就是哪个大洋的中心点，根本无法逃离的那种。
<br/>　　在最初的愤怒烦躁即将爆发的临界点时，楚涵又忽然转念一想：“没这么蠢，高少辉没这么蠢。”
<br/>　　紧接着楚涵干脆席地而坐，托腮沉思：“阴阳谷到底发生了什么？我已经痊愈了高少辉依然不然我去……”
<br/>　　此时已经回到旁边最近岛屿的旺财和高少辉，也进行了类似的对话。
<br/>　　“我看楚涵已经差不多好了，要不放他走吧？”旺财试探性的说道：“你也知道他那脾气，你敢这么限制他，他回头真能报复你。”
<br/>　　高少辉神色严肃，头也不抬的自顾自整理烧烤架：“不能去。”
<br/>　　“怎么不能去？”旺财不懂，道：“白允儿要是出了什么事，楚涵可饶不了你。”
<br/>　　“那也不能去！”高少辉忽然情绪激动起来：“现在的楚涵太弱了。”
<br/>　　“我知道，我知道。”旺财并不能理解高少辉真正的用意，只是尝试说服道：“或许靠武力不能救出白允儿，但你知道的楚涵很聪明，咱们可以用计啊！”
<br/>　　“根本不是这个问题。”高少辉有些情绪不稳定，盯着旺财道：“你自己是什么你不知道吗？你待在楚涵身边这么久，还不知道他是谁吗？”
<br/>　　“啊？”旺财一脸错愕的挠了挠脑袋：“我是啥？他是啥？”
<br/>　　这样的回答让高少辉彻底愣住，甚至神情都看上去有些呆傻：“守护者，你在逗我？”
<br/>　　旺财继续挠了挠脑袋：“实不相瞒，我有三块碎片永远拿不到了，虽然我们找到了其他东西的能量代替，但在那三块碎片内的记忆，永远回不来，所以哪怕到了巅峰，我的记忆也只能恢复到70%。”
<br/>　　说罢旺财耸了耸肩：“所以有些东西，我还真不知道。”
<br/>　　听着这样的回答，高少辉震惊无比，当场爆了粗口：“握草！十块碎片失了三块？你俩过家家呢这么随便！”
<br/>　　旺财翻了个白眼，情景好似回到了当年它与楚涵的对话，只是这时候换成了高少辉激动，已经完全接受的旺财一脸淡定。
<br/>　　“没了就没了嘛，人生的精彩就在于未知。”旺财说着，还给了高少辉一个微笑。
<br/>　　高少辉只觉得眼前一黑，有种想要吐血的冲动，但他还是很快镇定下来，深呼吸了好几下，这才语重心长的说道：“首先……”
<br/>　　“高家的信息是白家和罗家的人，已经进入了阴阳谷，并且还是从阴谷入口开始的，也就是说在阴谷内他们极有可能留着人。”
<br/>　　“然后……”
<br/>　　“我们高家也派人去了，你知道的，水火两族眼下就在阴谷内！”
<br/>　　“这就已经是五门聚齐了，好，接下来的肖家，你认为肖七会放弃这次热闹盛宴？”
<br/>　　旺财无所谓的点点头：“那也只是六门，无论是阴谷内的禁制，还是阳谷内的机关，都需要八门聚齐才能打开，我不在楚涵不在，制裁者不在，怎么了呢？哪怕我和楚涵都在阴阳谷，制裁者远在……咳咳，也还是七门而已啊！”
<br/>　　“所以我才说你俩不能去啊！你俩任何一个都不能去！”高少辉激动道：“你知道的，有一种特殊情况，最恶劣的犯罪行为发生时，制裁者会被强行召唤到犯罪现场！”
<br/>　　听到这话，再看着高少辉紧张无比的神色，旺财终于认真了起来：“你是说，白家或者是罗家，想在阴阳谷搞事情？！”
<br/>　　“这只是我的猜测，或许想搞事情的人还有肖七。”高少辉额头已经冒出了冷汗，道：“原本白忧和罗杰都派出了死士去杀楚涵，但这两家族内的长者双双亲自出动，虽然我不知道去的人是谁，但我知道他们是过去阻止楚涵被杀。”
<br/>　　旺财经历过那场生死一线的战斗，明白当时的楚涵有多惊险，更清楚之后出现的白晟有多可怕，这时候再听高少辉的一分析，也差点吓的冷汗直冒。
<br/>　　“你说的让我想起来了，白晟来了，差点将楚涵迷惑住，走之前还让楚涵去阴阳谷，不然白允儿就怎么怎么的……”旺财呐呐开口。
<br/>　　高少辉‘啪’的一拍自己手掌，激动道：“妈的这就是他们的目的！你仔细想想如果这时候楚涵在阴阳谷，哪怕他不是我们所想的那个存在，但你肯定会跟着楚涵去阴阳谷，加上水火两族，白、罗、高、肖，七门聚齐后……最恶劣的犯罪行为，制裁者被强行召唤，他们想做什么，你敢想？”
