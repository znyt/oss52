　　站在巨大落地窗前，一身黑色宽松衣袍的苏重，静静的打量着整个枯枯戮山。
<br/>　　这已经是他回到家中的第三天。
<br/>　　这些天，他什么也没干。除了做了一些简单的运动，保持身体敏锐，没再做其他严苛训练。反而是在尽情的放松心神，缓解长久以来，一直紧绷着的神经。
<br/>　　过去的一年里，苏重几乎在不眠不休的修炼。即便他身体强悍，根本感觉不到疲累。但精神上却需要休息。
<br/>　　“不过，休息了这几天，也是时候开始下一阶段的修炼了。”苏重转头，通过身侧窗户，看向光秃秃的枯枯戮山。
<br/>　　“即便已经冷却凝固，这到底是一处火山。在很久以前，这里甚至就是一片火山群。地底之下，肯定有更广阔的岩浆河流。”苏重查过家族记载，知道枯枯戮山只不过是整个山区的一部分。
<br/>　　根据记载，在很久以前，这里根本没有山体。后来地下岩浆运动，地壳上升，地震的同时伴随着火山喷发。这才形成了整个山脉地区。
<br/>　　只不过那已经是很久之前的事情，到了近代，就只剩下寥寥几个火山口还能冒烟，其中就包括枯枯戮山。
<br/>　　站在山顶，苏重神色淡漠，一跃而下。
<br/>　　轰！
<br/>　　弥漫着丝丝缕缕黑烟，趋于稳定的火山底部，立刻被打破宁静。一个十米大坑凭空出现。
<br/>　　苏重直挺挺的站在大坑底部，他根本就没做任何缓冲。直接跳下来，巨大撞击力之下，他自己安然无恙。坚硬的凝固岩浆，反而被砸出了个大洞。
<br/>　　“不只是身体强度增加了，重量也增加了。”苏重捏了捏胳膊，一种坚韧感觉传来。而且密度非常高。
<br/>　　即便不用测试，苏重也感觉到，自己的体重在快速暴增。他外表看着就是个十八九岁的清秀少年，而且还有些偏瘦。可身体重量，足有数百斤。根本就是一个浓缩了的怪物！
<br/>　　仔细感受着身体变化，苏重心中有了明悟。魔龙七段的修炼，就是在进行肉体改造。自己虽然不会真的变成魔龙，但细胞却在不断蜕变。早就不再是单纯的人类细胞。
<br/>　　这也是他外形不变，体重增加的原因。
<br/>　　苏重不由自主想到了嵌合蚁的王。面对尼特罗层出不穷的攻击，对方根本就不防御，只是用身体硬扛。除了疼痛，竟然丝毫没有损伤。
<br/>　　虽然有着大概的人体外形，身体细胞却完全就是另一种结构。苏重也在发生这种变化。
<br/>　　自己正在变得不是人？
<br/>　　好在除了体重增加，外形并未有什么变化。苏重也不怎么在意。摇摇头不理会这些杂念，气魄陡然爆发。一个黑红色好似岩浆表面的圆球陡然出现。
<br/>　　嗤嗤嗤！
<br/>　　一阵白烟冒起。坚硬的带面，好似凝固油脂遇到烧红的刀子。光球几乎立刻开始向下落。
<br/>　　地下岩石，一旦碰到气魄，立刻就变成了黑色粉末，不堪一击。
<br/>　　他根本就不用发力，仅仅靠着气魄释放，就一路下降。
<br/>　　周围温度不断上升，身旁黑色岩石也开始慢慢变成红色。
<br/>　　噗！
<br/>　　好似一个泡泡，苏重一下钻进一片红色液体中。
<br/>　　“到岩浆河了，可惜，还不够。”苏重摇头，这里温度足够，岩浆也有不少。可到了苏重如今阶段，根本就不够用。
<br/>　　他只要放开吞噬能力，只需一下，就能把身周的岩浆完全吞噬干净！
<br/>　　不理会周围咕嘟嘟好似开锅似的岩浆，苏重继续下降。身后岩浆猛然合拢，堵住气魄圆圈留下的孔道。顺着苏重的来路，就要往上喷发。
<br/>　　可下一刻，苏重身上陡然散发出一股绝强吸力。那些不断向上涌的岩浆，顿时就像是被套上了缰绳的火龙。使尽全力不断往上挣脱，速度却越来越慢，红色身体也在不断变暗。几个呼吸之间，就彻底凝固。把苏重身后的通道给堵死。
<br/>　　苏重可不想因为自己，搞得枯枯戮山火山爆发。他现在可是个揍敌客。要是真把枯枯戮山给搞的火山爆发，揍敌客家族驻地被毁，马哈揍敌客那个老怪物，肯定会来找他麻烦。
<br/>　　不知下降了多久，知道苏重把圆尽全力放出去探查，四周仍然全都是岩浆后，苏重终于停了下来。
<br/>　　“以这里的岩浆含量，足够我修炼一段时间了。”
<br/>　　心中一动，魔龙法相猛然出现在身后。
<br/>　　吼！
<br/>　　仰天一声嘶吼。一股波纹无声无息扩散。四周岩浆立刻被排挤出去，周围顿时出现一个直径二十多米的空洞区域。
<br/>　　“二十米？这还不是极限。”苏重回头打量魔龙法相。火车头一样的狰狞魔龙头颅，同样低头看向苏重。
<br/>　　透过猩红竖瞳，一种冷漠残暴的感觉直冲苏重心灵。“竟然还有攻击心神的能力？”苏重若有所思。
<br/>　　他的魔龙法相，到底不是正统的百式观音。是通过吸收学习比杨德的魔像·神罚而来。
<br/>　　“看来是吸收了魔像蛊惑人心的能力。”苏重想到了比杨德魔像上的扭曲鬼面。
<br/>　　他魔龙法相一成，交手的第一个人就是尼特罗。对方为了提高修为，曾经十多年如一日不停的练拳。更是在不知不觉之中练成了百式观音。精神意志之纯粹强悍，已经堪称非人。
<br/>　　苏重身体被魔龙七段改造，越来越不像人。尼特罗的外形虽然仍然是人，可精神意志已经恐怖的堪称怪物。
<br/>　　魔龙法相震慑人心的效果，对他根本就没有效果。
<br/>　　“倒是一个不错的清场能力，适合群战。可惜，对意志坚定的人，效果却微乎其微。对上尼特罗那种，根本就像是清风拂面，毫无影响！”
<br/>　　“接下来，就让我看看，这魔龙法相的极限在哪里吧！”
<br/>　　身体微微下蹲，丹田道种猛然震动。苏重双手猛然探出，两个黑色洞窟骤然粗现在苏重身前。
<br/>　　吞噬！
<br/>　　海量岩浆立刻挤入黑色旋涡中。
<br/>　　苏重感觉胃部猛然间一热，一股热流顿时从胃部冲出，流入四肢百骸。随着魔龙七段不断强化身躯。
<br/>　　还有一大半则突兀消失，顺着诡秘路径，流入身后魔龙法相。
<br/>　　经过这么一分润，吞进体内的岩浆，几乎瞬间就被瓜分干净！
<br/>　　“不够，不够，远远不够！”苏重脸色狰狞。
<br/>　　轰！
<br/>　　他身体表面，骤然浮现出大量鳞甲状纹路。眉心火焰团更使不断摇曳变长，好似要冲出额头！
<br/>　　呼！
<br/>　　气魄光膜骤然缩回体内，岩浆顿时倒灌而回。几乎在接触苏重体表的瞬间，一股恐怖吸力顿时爆发。
<br/>　　周身每一个毛孔之中，都发出吸扯力量。这一刻，苏重竟然用全身毛孔吞噬岩浆！
<br/>
