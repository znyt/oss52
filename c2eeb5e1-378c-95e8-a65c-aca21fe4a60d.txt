很快陆寒霆就回来了，他将一株紫色的药草交给了夏夕绾，“给你。”
<br/>
<br/>        夏夕绾伸手接住了，“谢谢。”
<br/>
<br/>        陆寒霆没有走。
<br/>
<br/>        夏夕绾仰起黑漉漉的翦瞳望着他，“陆总，非礼勿视，请你背过身去。”
<br/>
<br/>        “我帮你？”陆寒霆看着她受伤的位置。
<br/>
<br/>        夏夕绾一张绝色的小脸红白交加，他怎么帮啊，她看他就是想占便宜！
<br/>
<br/>        “不用。”夏夕绾拒绝道。
<br/>
<br/>        陆寒霆见她态度坚决，也就转过了身，这时身后还传来女孩软糯的嗓音，“陆总，你应该不会偷看的吧？”
<br/>
<br/>        陆寒霆单手抄裤兜里,回答的流氓又坦然，“不确定。”
<br/>
<br/>        “…”
<br/>
<br/>        夏夕绾也背过了身，她将身上的衣服给脱掉，然后将药草放在嘴里咀嚼了一下，最后敷在了自己的伤口上。
<br/>
<br/>        这里很安静，陆寒霆就听到了身后传来的悉悉索索脱衣声，他凸起的喉头上下滚动了两下。
<br/>
<br/>        很快他就听到她叫了一声，大概是敷药草疼了，她贝齿咬着红唇，两眼水汪汪发出的声音，少女软糯的叫声刺激的他眼角一红。
<br/>
<br/>        陆寒霆觉得自己不能再待下去了，他走了出去。夏夕绾处理好了伤口披上衣服，但是很快她就察觉到了身体的不对劲，身体好热，小脸都烧了起来，体温节节攀升，她好想把身上的衣服给脱掉。
<br/>
<br/>        她伸出纤白的小手又拉开了衣领，露出了自己雪白的肩头。
<br/>
<br/>        这时耳畔传来一道低沉磁性的嗓音，“你在干什么？”
<br/>
<br/>        夏夕绾抬头一看，陆寒霆回来了。
<br/>
<br/>        陆寒霆出去捡了一些树枝，他身上是一件黑色衬衫，衬衫的衣袖往上卷了两道，露出他结实的小臂，现在他高大英挺的身躯伫立在那里，眼眸带着几分侵略和打量看着她。
<br/>
<br/>        莫名的，夏夕绾觉得身上更热了一点。
<br/>
<br/>        这时陆寒霆拔开长腿走了过来，他单膝蹲地，大手抓住了她纤白的小手，不让她乱扯衣服，然后他用修长的指腹摸了摸她发红发烫的小脸。
<br/>
<br/>        男人的大掌摸了上来，带着那股强烈的男人阳刚，好像能解她的热，夏夕绾清丽的眉眼里挑出了几分青涩的风情，水汪汪的盯着他看，“陆总，我身上是不是很热？”
<br/>
<br/>        陆寒霆看着她，“不是热，是…发烧了。”
<br/>
<br/>        他说她发烧了。
<br/>
<br/>        夏夕绾那双翦水澄眸变得湿漉漉的，小鹿乱撞一样，她不知道他那句“发烧”是不是另有所指。
<br/>
<br/>        “刚才你给我摘了什么草药？”
<br/>
<br/>        “就是你说的那株紫色的。”陆寒霆用目光指了指。
<br/>
<br/>        夏夕绾这么一看才知道他摘错草药了。
<br/>
<br/>        那里有两种紫色的草药，他摘的这一种除了止血消炎外，还带着催.情的成效。
<br/>
<br/>        怪不得她身上这么热！
<br/>
<br/>        “陆寒霆，你摘错草药了，你是猪吗？”夏夕绾生气的骂道。
<br/>
<br/>        被骂成猪的陆寒霆也不气，他大致已经猜出发生了什么。
<br/>
<br/>        陆寒霆伸手，用粗粝的拇指指腹覆上了她的红唇，来回按压，看着她娇美的菱唇在他的指腹下失去了颜色，他嗓音暗哑道，“现在骂我也无济于事，你只需要告诉我，你想要不想要我，恩？”
