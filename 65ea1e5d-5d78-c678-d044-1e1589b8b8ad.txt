　　嘭…
<br/>　　一股火焰，从萧逸手中凝聚。
<br/>　　原本略显昏暗的山洞，霎时火光明亮。
<br/>　　而那一份份珍贵至极的修炼资源，更是足以让人目瞪口呆。
<br/>　　萧逸指尖火焰已凝聚。
<br/>　　嘭…
<br/>　　火焰打出，但面前，却并非是八龙焚火炉。
<br/>　　而是八荒炉。
<br/>　　八龙焚火炉，此时里头还封印着大量金光力量，根本无法用于炼制丹药。
<br/>　　虽说八荒炉的层次，远远比不上八龙焚火炉。
<br/>　　但对萧逸而言，这并不算得了什么。
<br/>　　以他的炼药本事，用八荒炉炼药也一样，只是速度要比八龙焚火炉慢上不少罢了。
<br/>　　嗖…嗖…嗖…
<br/>　　一份份天材地宝，一粒粒妖兽内丹，随手抛入火炉之内。
<br/>　　看似随意，看似杂乱，实则每一次抛入皆是有条不紊，皆是自成玄妙，完美至极。
<br/>　　单单这一手炼药本事，怕是已足以让不少炼药大能为之汗颜。
<br/>　　……
<br/>　　两个时辰后。
<br/>　　周遭，已一堆堆完美级别的高品丹药炼制而出。
<br/>　　而这时，萧逸已停下炼药。
<br/>　　“嗯？”萧逸眉头一皱，“这么快炼完了？”
<br/>　　萧逸看了眼身旁周遭，已无材料。
<br/>　　说起来，他身上有不少珍稀而高品的天材地宝和材料等等。
<br/>　　但说到数量，似乎着实不多。
<br/>　　上一次炼制丹药，提升修为，是数月前在金光险地之外的大城。
<br/>　　而这数月的十八府历练之行，他倒确实斩杀了大量妖兽，得了不少妖兽内丹和精血。
<br/>　　可天材地宝方面，收集的倒是不多。
<br/>　　总不可能他去任何妖兽森林或者险地都能碰上大量天材地宝。
<br/>　　顶多就是运气好时，遇到一株在外万年难遇的珍稀天材地宝。
<br/>　　故虽然历练数月，而且是高强度地极限历练，收集的天材地宝并不算太多。
<br/>　　当然，这种不多，只是对于他那庞大气泉而言罢了。
<br/>　　对于别的武者来说，这已经是一笔足以让人垂涎的修炼资源。
<br/>　　萧逸皱了皱眉，将从数千剑修那得来的乾坤戒一一打开。
<br/>　　里头修炼之物，一一倾倒而出。
<br/>　　又是数个时辰后。
<br/>　　一堆堆高品丹药被炼制而出。
<br/>　　而各种材料，则已被炼制完毕。
<br/>　　想来也是，那数千剑修，大多在万剑碑林中参悟后便修炼了，也大幅度提升了实力。
<br/>　　自然的，他们已消耗了大量修炼之物。
<br/>　　剩下来的修炼之物，自然不多。
<br/>　　倒是气宗宗主的乾坤戒内，身家比较丰厚。
<br/>　　不过，这也禁不止萧逸的快速炼制。
<br/>　　再者，气宗宗主的身家，应该大部分在他气宗之内。
<br/>　　“算了。”萧逸摇了摇头。
<br/>　　现今，还是先修炼为主。
<br/>　　萧逸双手齐出，吸摄着这一堆堆高品丹药。
<br/>　　体内两大武魂亦是齐出，吸收速度快得惊人。
<br/>　　一分钟后。
<br/>　　嘭…
<br/>　　萧逸身上一阵气势爆发，这是突破的气势。
<br/>　　“圣皇境，到了。”萧逸脸色一喜。
<br/>　　他的修为，本就在圣王境巅峰。
<br/>　　而且体内气泉早已满，小世界也已被‘冰山火海’填满，要突破至圣皇境，本就只是一个念头的事。
<br/>　　不过，虽然突破到圣皇境，可‘冰山火海’的消失，无疑让萧逸实力下降。
<br/>　　这也是他在万剑碑林时没有选择突破修为的原因之一。
<br/>　　说起来，萧逸的修为提升，倒也不慢，反倒惊人般的快。
<br/>　　距离上次提升修为，不过是数月之前。
<br/>　　而且在剑帝碑空间第二层内，他更直接把最为麻烦的‘冰山火海’也一并填满了。
<br/>　　要知道，他数月前，也不过是圣王境六重的修为。
<br/>　　现今，已然圣皇境了。
<br/>　　也就是说，他已然是武道皇者。
<br/>　　武道皇者，便是圣皇境的称谓。
<br/>　　这个境界，武者掌控的完整武道，已达千条之多。
<br/>　　对比以前掌控10条完整武道，踏入圣境；以及掌控100条完整武道，踏入圣王境。
<br/>　　现今，掌控一千条，自然是等同武道中的皇者。
<br/>　　虽说萧逸斩杀过的武道皇者，已然不少了，甚至连武道大能乃至绝世强者也斩杀过。
<br/>　　但这不过是他接触的层次太过。
<br/>　　他所招惹的，无不是中域之内传承数十万年以上的霸主势力。
<br/>　　自然的，碰上的敌人，也极强。
<br/>　　但放眼整个中域，武道皇者，已经是武者中等同君皇一般的存在。
<br/>　　圣皇境，已经有开宗立派，雄踞一方的本事。
<br/>　　如当年在黑云地域的天王山。
<br/>　　“圣皇境一重了。”萧逸自语着，也思索着。
<br/>　　接下来的修炼，将比以往困难百倍。
<br/>　　任何一个武道皇者，其后的修炼，都难如登天。
<br/>　　其原因，便在于突破所需的武道。
<br/>　　圣皇境之后，每突破一重修为，需得多掌控一千条武道。
<br/>　　也就是说，到达圣皇境二重，需要掌控两千条完整武道。
<br/>　　而到达圣皇境九重，则需要掌控九千条完整武道。
<br/>　　这个阶段，仍旧是属于武道皇者范畴。
<br/>　　待得武者掌控九千零一条完整武道时，便是踏入圣皇境巅峰。
<br/>　　掌控九千五百条完整武道，便是踏入武道大能范畴。
<br/>　　掌控9990道，便是绝世强者。
<br/>　　圣皇境中的修炼，不仅难，且泾渭分明；武道皇者、巅峰圣皇、武道大能。
<br/>　　称谓不同，虽仍旧在圣皇境范畴，却是实力天差地别。
<br/>　　当然了，对萧逸而言，武道参悟等等，不算什么难事。
<br/>　　唯一卡住他的，只有他那庞大气泉，以及所需的庞大修炼资源。
<br/>　　“呼。”萧逸深呼吸一口气，再次开始了修炼。
<br/>　　一粒粒完美的高品丹药，被他快速吸收。
<br/>　　半个时辰后。
<br/>　　“圣皇境二重了。”
<br/>　　三个时辰后。
<br/>　　“圣皇境五重了。”
<br/>　　这时，萧逸微微睁开了眼睛。
<br/>　　短短数个时辰，极限参悟四千道完整武道，这已经是极其夸张的事。
<br/>　　属于天地中的这四千条武道的力量，早已悉数降临，被他吸入小世界中。
<br/>　　不过，这四千条武道，尽皆是一流、二流层次的武道。
<br/>　　其中二流层次居多。
<br/>　　实在是他压根没有那么多顶尖武道，再者顶尖武道也难以融合，更绝非短时间内能参悟的。
<br/>　　他向来参悟的武道，都是顶尖中的顶尖。
<br/>　　现今参悟这些二流武道，倒是速度快得惊人。
<br/>　　至于他暂时停下修炼的原因，便是因为他身旁，一条火焰长河，忽然而现。
<br/>　　火焰长河，如若虚幻，却蕴含惊人的武道力量。
<br/>　　“武道真意。”萧逸皱了皱眉。
