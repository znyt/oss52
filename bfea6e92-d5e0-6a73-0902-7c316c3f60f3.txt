　　这毒气根源在主殿，现在楚风眠也不急于去探察这毒气根源的事。
<br/>　　纵然是楚风眠是半步不朽真身，他也必须要小心，这毒气很有可能是天元商会，这庞然大物崩塌的根本。
<br/>　　半步不朽真身，终究不是真正的不朽真身，做不到真正的完美，楚风眠也不敢轻易的冒险。
<br/>　　所以楚风眠的打算，还是先探察这周围的宫殿，看看有没有留下的丹药宝藏等，在向着主殿的方向进发。
<br/>　　这宫殿巨大无比，楚风眠打开灵识，开始扫视着其中的没一处区域，突然在这宫殿之中，见到了一个木箱。
<br/>　　这木箱，也已经是被毒气侵蚀的不像话，呈现出了一种黑色。
<br/>　　而且这整个木箱，不知道是被毒气沾染了多少年，已经是化为了一种剧毒之物。
<br/>　　不过楚风眠却是丝毫不惧，他直接用手猛然一抓这木箱，剧毒便是从木箱之内，想要钻入到楚风眠的身躯之中。
<br/>　　但是这十二万九千六百枚血肉晶体，死死组成的屏障，却是让没有任何一丝的剧毒，可以钻入到楚风眠的身躯之中。
<br/>　　随即楚风眠手一用力，这木箱被楚风眠掀开，在这木箱之中，大大小小摆放着数十个玉瓶。
<br/>　　每个玉瓶之中，都是盛放着几枚各色不同的丹药。
<br/>　　“果然有丹药遗留了下来！”
<br/>　　楚风眠看到这些丹药，兴奋无比。
<br/>　　在他发现这天元商会，不是被外敌入侵，而是因为这毒气毁灭的一刻，楚风眠就有猜想，只怕是天元商会内还留有不少的宝藏。
<br/>　　果然如楚风眠所料的，这些丹药，根本没有人带走。
<br/>　　也因为毒气的关系，十万年来，也没有人可以从这天元商会之中带走任何的东西。
<br/>　　半步天人境级别的强者，只怕碰到这样一座毒城，也不敢下手，只有楚风眠这种凝练了半步不朽真身的人，万毒不侵，才敢踏入其中。
<br/>　　“这些可都古圣丹，至少都是中古时代的，甚至不少都是上古时代的古圣丹！”
<br/>　　楚风眠熟读古籍，其中不少的古圣丹的品级，他一眼就看的出来。
<br/>　　“这是清水圣丹！”
<br/>　　楚风眠随手抓起了一个玉瓶，在这玉瓶之内，摆放着三枚透明的丹药。
<br/>　　这清水圣丹，看起来平平无奇，但是在十万年前的时代，却是特别的抢手货，一枚清水圣丹，就足以洗髓身躯，将身躯之中的杂质全部驱除。
<br/>　　一般就算是圣丹，甚至是古圣丹，也都没有完美无缺的丹药，在其中必然会有着杂质。
<br/>　　这些杂质，常年累计下来，会堆积下来，境界低不影响，但是一旦是踏入古帝，甚至是冲击天人境的时候，则必须要肉身完美。
<br/>　　一点点杂质，都可能影响冲击天人境。
<br/>　　这清水圣丹，正是可以将身躯中的杂质全部驱除，所以许多九劫古帝，想要冲击半步天人境，甚至是想要冲击天人境的时候，都会购买清水圣丹。
<br/>　　因此这清水圣丹，虽然只是中古级别的古圣丹，但是论起价值比起上古时代的古圣丹，还要高。
<br/>　　只是清水圣丹的丹方，也是被天元商会所掌控，以至于在天元商会消失之后，这清水圣丹也随之消失了。
<br/>　　后市虽然也炼制出了类似的圣丹，但是效果依然不如清水圣丹，这三枚清水圣丹拿出去，只怕是会让无数九劫古帝抢破头。
<br/>　　不过楚风眠可没有想要将这些圣丹卖了的想法，他不缺财富，楚风眠是半步不朽真身，身躯接近完美，清水圣丹对他无用。
<br/>　　但是对寒月漓，洛汐，甚至青梦等人，这清水圣丹都有着极大的作用，尤其是他们想要冲击半步天人境的时候，必须要洗髓身躯。
<br/>　　这清水圣丹，正是最完美的选择，三枚的数量还是太少了。
<br/>　　楚风眠一边想着，一边打开了玉瓶，想要看看这清水圣丹，可当他打开的一刻，出现的不是药香，反而是一种恶臭。
<br/>　　只见这清水圣丹之上，居然隐隐约约包裹着一层绿色的纹路。
<br/>　　看样子，是被这宫殿内的毒气所侵蚀了。
<br/>　　看到这一幕，楚风眠眉头一皱。
<br/>　　随即他又打开了一个玉瓶，接过一样，这玉屏之中的圣丹，也都是被毒气侵蚀。
<br/>　　虽然其中的药力还在，但是更多的却是剧毒。
<br/>　　这样的圣丹吃下去，绝对是天下间最为猛烈的剧毒，没有好处，反而是坏处无数。
<br/>　　看来这些盛放丹药的玉瓶，也无法阻拦毒气的侵蚀，十万年的时间，这些圣丹，全部都是被毒气侵蚀，成为了一枚枚的毒丹。
<br/>　　这让楚风眠眉头紧锁，这些毒丹，给楚风眠也没用，这让楚风眠有些失望。
<br/>　　得到了一座宝藏，但是这宝藏之中的一切，都没有任何的用处，简直是就连垃圾都不如。
<br/>　　“毒气侵蚀……既然是药力还在，那只要是化解了上面的剧毒，这些毒丹也可以恢复本来的药力才对！”
<br/>　　“这剧毒虽然霸道，但未必不能化解，以吞天秘术来吞噬看看！”
<br/>　　吞天秘术，号称有着吞噬诸天之能，楚风眠的吞天秘术虽然还没有练到这个地步。
<br/>　　但是吞噬一些剧毒，应该可以尝试。
<br/>　　楚风眠死死的盯着手心之中的清水圣丹，突然之间手心之中，一股吞天秘术的力量凝聚而出，化为了一道小小的漩涡。
<br/>　　随即楚风眠手指一弹，这清水圣丹，就落入到了吞天秘术之中。
<br/>　　吞天秘术运转起来，开始尝试吞噬这清水圣丹上的剧毒。
<br/>　　这些圣丹，都是古圣丹，品级极高，所以剧毒倒是无法侵入其中，只能够是在外围聚集。
<br/>　　楚风眠利用吞天秘术，当即开始吞噬，一缕缕的剧毒，都被逐渐的吞噬，分解，纷纷化为了精粹的灵力。
<br/>　　“有希望！”
<br/>　　楚风眠眼神一亮，楚风眠尝试用着吞天秘术去炼化这些古圣丹上面的剧毒。
<br/>　　果然这清水圣丹，落入到吞天秘术之中，几分钟的时间，楚风眠就察觉到上面的剧毒，被吞噬了一部分。
<br/>　　这些剧毒，果然是可以用吞天秘术来吞噬！
