　　一切真相，都跟妙心之前猜测的差不多。
<br/>　　对这座古墓什么的，叶少阳其实并没有什么兴趣，也不愿意在这上面多费精力，唯一让他感到困惑的地方，就是碧清……她跟苏沫之间，到底是什么样一种关系？为什么她对李浩然那么感兴趣？
<br/>　　叶少阳觉得，事情好像越来越复杂了，而且冥冥之中似乎有什么人在安排，故意让自己去经历这些事情……
<br/>　　河南，中原腹地，一个平原地区的小镇。
<br/>　　虽然大清亡了有了念头了，连袁大头都没了，现在是民国，北京上海这样的大城市，以前在悄然接受西方文化的入侵，涌现出了一大批时髦的青年。
<br/>　　但是在这个远离都市的小城里，像别的很多地方一样，还保留着过去的传统。
<br/>　　镇上有一个前清的进士，姓方，叫什么大家早就忘了，因为人人都叫他方员外。
<br/>　　方员外曾经当过官，后来告老还乡，散尽家财，为当地做了无数的好事，是士绅之首，当地最德高望重的人。
<br/>　　方员外已经七十多了，是个瘫子，没有儿孙，也没有成家，但是家里有很多仆人，每天就是坐在自己家的后花园里晒太阳发呆，思考人生。日复一日。
<br/>　　这一天，他的仆人又把他搬进了院子里，结果坐下没多久，就有仆人来通报，外面有人求见。
<br/>　　不知道为什么，方员外突然有了一种不详的预感，直觉不应该见这个人，但又觉得避免不了，于是让仆人放他进来。
<br/>　　在惴惴不安的等待中，方员外听见了脚步声，远远看到一个穿蓝衫的人走了进来，打扮像是一个道士。
<br/>　　道士一直走到他对面，十步左右，站住了。
<br/>　　方员外皱着眉头，凝视他的脸，突然间，满脸的皱纹居然舒展开了，长吁了一口气，“原来是这样……”
<br/>　　道士点点头。
<br/>　　“你来了。”
<br/>　　“我来了。”
<br/>　　“你本不该来的。”
<br/>　　“但我还是来了。”
<br/>　　方员外苍然一笑，“你来了，我就要死了。”
<br/>　　道士抬起头看他，道：“你就是我。”
<br/>　　方员外道：“我知道，见到你的时候，我就知道了。”
<br/>　　随即苦笑着叹了口气，“我说最近怎么心慌慌的，不知道要发生什么事情，原来……是这样。我只想问，你是哪一世的我，你叫什么？”
<br/>　　“来自百年之后，李浩然。”
<br/>　　方员外点点头，有了一种大彻大悟的感觉。说道：“我不知道你是怎么来的，但你本不该来找我，你坏了天地造化的规矩。”
<br/>　　“我既然能来到这里，便也是天地造化。”
<br/>　　方员外仔细想，点了点头，随即目光从他身上移开，看着远处，说道：“我这一生，不知天命，见到你的一刻，我才知道这一切，三千繁华，一生造作，原来只是黄粱一梦……
<br/>　　我这一生，行善积德，体会了人间至善……却是个瘫子，顺便也体会了病痛，大善大痛。也是百世轮回中的一环了。我却不知，你这一生，又历经了什么？”
<br/>　　李浩然道：“当年我主西行，命我轮回百世证道，只有如此，才能斩断人间一切欲念，到我这一世，却是小圆满了，我这一世之前，如你这般，每一世都是痴迷人世，但不知天命，却与道佛有缘……你虽位修行道法，你这一生，却是修行了道心……但我不同，我一出生便知天命，拜无极天师为师，将生生世世的积累尽数施展，打磨道法……却是小圆满了。”
<br/>　　方员外看着他，喃喃道：“这么说，我说起来，其实只是你一抹神念了。”
<br/>　　李浩然颔首。
<br/>　　“那你今日找我，却又是为何？”
<br/>　　“天无二日，这个世上，只能有一个我。我既然不得不留下，你却只有先走，继续轮回。”
<br/>　　方员外点了点头，双手扶着椅子，这个一生从来没有站起来过的瘫子，居然稳稳地站了起来，朝李浩然走了过去。
<br/>　　一边走，他的身体一边融化，迅速腐烂，直到消失，魂魄越走越淡，却是破开虚空，直接前往阴司的轮回司去了……
<br/>　　“但愿，我没有做错……”李浩然喃喃说道，徐福，这一切都是你逼我的。他日若见到你，我必要将你碎尸万段！
<br/>　　从天际收回目光，李浩然低声说道：“你们从桃花山一路跟踪到这里，也不容易，都出来吧。”
<br/>　　说完这句话，过了有十秒钟左右，几道人影，从花丛中走了出来。
<br/>　　四个人，有道士，也有和尚。看相貌和打扮，都很普通。年纪也不大，都是三四十岁。
<br/>　　李浩然一眼看去，这些人身上都有光华闪现，异常明亮，知道都是天师和准地仙牌位（和尚是禅师），沉声说道：“法术公会什么时候这么强盛了，连拍出来盯梢的，也都是天师牌位。”
<br/>　　四人面面相觑，一人道：“原来你竟然是青牛祖师……怪不得鲲鹏师兄败在你手上。”
<br/>　　“你们不是人类？”
<br/>　　“我等是轩辕氏……星月奴门下，实不相瞒，我们负责监视祖师，近日一向在桃花山蛰伏，见祖师下山，这才一路跟随……祖师当真来自百年之后，我等不知其中缘故，但十分好奇，祖师来到当世，所为何事？”其中一个道士说道。
<br/>　　“星月奴门下的一条狗，有什么资格问我？”李浩然淡淡说道。
<br/>　　道士变了脸色，但不敢多言。
<br/>　　一个和尚见状，说道：“祖师，这且不关我等之事，不过当下赶逢乱世，人间昏暗，法术公会有形助人间重整旗鼓，度过乱世，可谓是一番功德……不知祖师可否助我等一臂之力？”
<br/>　　李浩然微微一笑，“你们怕我？”
<br/>　　一个道士上前，拱了拱手，语气略有些冰冷地说道：“祖师道法通天，我辈自然不是对手。不过，我轩辕氏为天人正朔，绵延千古，却从来没怕过人间的什么人……无非是图合作而已。”
<br/>　　李浩然道：“人间法术界，终归是人间来管，所你们轩辕氏想要插手，那只有赶你们回去。”
<br/>(本章完)
