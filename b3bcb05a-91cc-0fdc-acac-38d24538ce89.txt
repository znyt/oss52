　　第372章
<br/>　　派了专人护送，一路行舟乘马，大概在十天后抵达长安，将肉球一样的崔弄海送回相府。
<br/>　　宋伯康在这里也整顿兵马，磨刀霍霍，因为他知道，崔弄海送回去，丞相必要求自己与程大雷一战。
<br/>　　从宋伯康本心出发，是不希望和程大雷发生战斗的。第一、如今的程大雷已经不是一伙小势力，琴川关披甲过万，八百里琴川之主，岂容小觑。第二、程大雷为人并不是传说中那样霸道，只要你不去招惹他，他也不来招惹你，双方在某些事情的合作，甚至称得上愉快。第三、自己儿子可还扣在琴川关呢。
<br/>　　可不打这一仗，崔相哪里也不好交差，如果崔相的命令下来，自己也只能硬着头皮进攻。现在凉州城的军策府日夜忙碌，正在制定作战策略。
<br/>　　可宋伯康没有想到的是，人送回去后，崔相哪里并没有什么动静，他一等二等，没有等到相府的命令。
<br/>　　宋伯康愈发困惑，不知崔相大人，何时变得如此好脾气。
<br/>　　也就是等去的人回来，长安的消息传入宋伯康耳中，他才算明白发生了什么事。
<br/>　　绝对不是崔相大人性情大变，突然好脾气了；也不是崔相不想灭掉程大雷，关键是崔相现在腾不出手。
<br/>　　幽州，正义教起事。
<br/>　　江南一战，李星率领的星字军攻占正义教老巢。正义教高层死走逃亡，教徒死伤无数。为此，明帝还特意封赏了李星，朝廷欢庆。
<br/>　　于此同时，帝国全力通缉正义教教徒，凡家中有正义教的祭祀之物，一旦被搜出就是斩立决。各州各府，长街之上经常有成排的人被砍头。在这种情况下，为了领功之类的事情，经常有人被冤杀。
<br/>　　如此大的力度，通缉一波匪类，所有人都相信正义教已经是一堆死灰，且再没有复燃的机会。
<br/>　　但在所有人都没有提防的情况下，正义教匪首潜入幽州，在那里聚集教徒。
<br/>　　青牛山一战后，幽州已经成为废土，帝国既无心也没有能力去经营这片土地。这给了正义教发展的空间和时间，招兵买马，只用短短一年的时间，正义教大旗之下已聚集十万教众。
<br/>　　这其中未必没有帝国朝廷的推波助澜，朝廷全力通缉正义教，夹着尾巴也活不下去的人，索性逃到幽州起事。
<br/>　　当朝廷上层注意到这件事时，正义教已经竖起了他们的大旗，这次正义教打出的旗号是：
<br/>　　斩妖除魔。
<br/>　　他们是真的把李氏皇族，当做妖魔了。
<br/>　　明帝亲自下令，调青州和并州两地之兵，合计兵力十五万，攻打正义教。
<br/>　　当宋伯康知道这件事时，几乎同时，程大雷也得到了消息。
<br/>　　当然，程大雷是通过江湖传言得到的情报，所得消息就没有那么准确。传言中，明帝发动了百万大军……
<br/>　　看着手中的情报，程大雷不知该是喜是忧，喜的是林少羽已经做得不错，短时间内就可启动这样一支兵马，忧的是他能否在大军围剿下活下来。
<br/>　　幽州和凉州相距万里，程大雷想做什么也做不到，只好不停派出人打听那边的消息。得到情报后，和山寨里的人一起分析。
<br/>　　按照程大雷的估计，如果林少羽能撑过这一战，稳住现在的局势，未来的他未必不能成为一股与帝国朝廷抗衡的力量。只希望他不要太过着急……
<br/>　　同时，程大雷也在忙碌自己的事情。
<br/>　　落玉寨升到三级后，从商店里开出了火药，他购买一箱后，交给工匠坊的人研究。
<br/>　　火药的配方倒也不复杂，无非是硫磺、硝石、木炭。帝国也并不是没有火药，烟花爆竹之类的很早就出现了。很简单工匠坊就拿出简易的的火药，只不过威力完全无法和程大雷从系统商店购买的火药相比。
<br/>　　主要工作是如何提升火药的威力，改良配方，提纯原材料。
<br/>　　程大雷亲自抓这件事，主要是火药毕竟属于爆炸品，手下人又没有什么安全意识，一不小心就会造成伤亡。
<br/>　　程大雷向来往的商队购买硝石、硫磺，琴川关现在有钱有粮，为此支付大价钱也是舍得的。
<br/>　　这倒不是太困难的事情，在这个冬天还没过去，工匠坊已经可以拿出成品火药。做了爆破实验，虽然还不能和系统出产的比，但威力已经算不错。
<br/>　　只到这一步，差不多也算是足够。整个帝国，还尚无多少用火药战斗的意识，最起码琴川关在武器上已经走到整个世界的前列。
<br/>　　但程大雷自然不会为此而感到满足，火药是拿出来了，如何将火药变成武器还有一个过程。
<br/>　　这对工匠坊来说，并不算太大的困难，短短时间后，火箭、火枪、炸天雷等武器都拿了出来。
<br/>　　如此，这项研究工作才算是结束，程大雷命令工匠坊开始生产火药武器。看着仓库储存着满满的武器，程大雷面对这世界也有了些底气。
<br/>　　自己现在就算面对宋伯康，也有了叫板的资格。双方兵力到一定程度，也就不是数字上的比拼。士气、武器、战略都将成为决定胜败的关键。
<br/>　　当然，程大雷目前没有向外扩张的意图，他本身性格也并不向往这些。所做的一切，无非是为自保而已，目前能端好手中的饭，程大雷已经心满意足。
<br/>　　这个冬天便如此过去，琴川关也迎来了三月初春。于此同时，程大雷也收到了关于幽州的情报。
<br/>　　并州、青州两地之兵，合计十五万攻入幽州。
<br/>　　攻一地，破一地，正义教节节败退，朝廷上下闻到战报之后欣喜非常，明帝亲自传令，要在这个冬天结束战争。
<br/>　　十五万帝国军长驱直入，直扑正义教老巢。正义教老巢就在程大雷当年驻扎过的山谷，帝国军强攻一月不下。
<br/>　　就在人困马乏之际，一支伏兵杀出，与城内两方夹击，帝国军一败涂地，死伤被俘无算。
<br/>　　当朝廷复盘这场战役时，发现正义教的节节败退，更像是诱敌深入，最后里外夹击，像是很早就设计好的。
<br/>　　不得不说，林少羽在蛤蟆寨上课那些日子，当真是个好学生。
<br/>(本章完)
