　　荒塔封锁了北斗星空，瞬间便惊动了北斗上的众人，他们也认出了那是东荒神器荒塔，但不知道这是要干什么。
<br/>　　“啊……”
<br/>　　不过片刻的时间，成仙路中便飞出一位至尊，他浑身血肉模糊，晶莹的血珠就好像是一颗颗红宝石般剔透，美丽炫目，洒落一地，莹白的骨茬都露了出来，模样极为凄惨。
<br/>　　“这一世，也对，也错，竟是这般，有希望，但也让人绝望！”
<br/>　　这名古代至尊声音低沉，语气之中充满悲伤，万古的等待，竟是这样一个结果，到头来终究没有成仙。
<br/>　　众人闻言皆心惊不已，不知道发生了何事？
<br/>　　轰！
<br/>　　突然，那混沌洞中又是一阵巨响，鲜血飞溅，鲜艳的刺眼，让人心尖都在抖动，浑身寒毛倒立。
<br/>　　鲜红的血，像红色钻石般飞溅，晶莹的白骨都坠落了下来，触目惊心。
<br/>　　一位古代至尊陨落于其中。
<br/>　　众人都沉默了，之前还以为等待了万古，要成仙了，可如今却是如此一个结果，成仙只是一个虚妄的假象？
<br/>　　“啊……”
<br/>　　又是一声大吼，第三位至尊从其中冲了出来，浑身是血，在那里摇动，似乎即将倒下。
<br/>　　“为什么会这样？”
<br/>　　至尊悲痛天地，是那么的绝望。
<br/>　　“天上地下，唯我独尊……”又一声苍老的声音传出，第四位至尊出现，终究是败了。
<br/>　　接下来，第五位，第六位，第七位至尊出现了。进去的至尊，死了两一位，只有五六人还活着。甚至连阿弥陀佛的信仰之身都不见了。
<br/>　　“等待的人，你们也错了，这一世不算对，也不算错，正确的时间，正确的地点，只是缺少一个正确的人而已！”
<br/>　　至尊在悲呼，天地在颤抖。
<br/>　　域外，混沌道君看着剩下的几位至尊，面色冷淡，看不出丝毫情绪，这是一场注定失败的道路，成仙路是给有准备的人留下的，至尊自从斩断那一刀，便与仙绝缘了。
<br/>　　“凌天，你看，你那位仇人好像还没死啊。”道君轻飘飘说了一句，看向荒古禁地前的长生天尊。
<br/>　　“没死更好，那我便去送他一程。”凌天冷笑一声，接着踏步走出，消失于星空之中，进入北斗荒古禁地。
<br/>　　“你不去看看吗？”在凌天走后，道君淡淡对着身后道。
<br/>　　身后，一道白色身影出现，面上迷雾笼罩，让人看不清虚实，正是女帝。
<br/>　　女帝不说话，走上前来，目光看着北斗，没有一丝变化。
<br/>　　突然，女帝出手了，她一巴掌穿过虚空，出现在荒古禁地之前，一巴掌就拍进混沌古洞，神威盖世，那里发生大崩溃，整座古洞化为了虚无。
<br/>　　成仙路被女帝生生击断，化为虚无。
<br/>　　成仙，已经彻底无望了。
<br/>　　“你这是何意？”
<br/>　　石皇震怒，他手中大戟横击向女帝的大手，然而，女帝并不在意，只是轻飘飘的向着这边一扇，简单而直接，便将石皇连同大戟打得倒飞出去，石皇吐血不已，遭受重创。同时，女帝的巨手也消失不见了。
<br/>　　举世震惊！
<br/>　　女帝一次出手，便震惊了世人，一击重创石皇！
<br/>　　石皇震怒，手中大戟横舞，想要追上女帝，前去击杀。
<br/>　　“成仙路上无死敌，两位还是息战吧！”长生天尊劝道。
<br/>　　轰！
<br/>　　突然，一道巨大的金色拳头洞穿虚空，轰杀过来，六个巨洞演化为一项无上霸拳，杀向长生天尊。
<br/>　　“谁？”
<br/>　　长生天尊双眸放出凌厉之光，手中长生剑迅速做出反应，无上大道法则在飞扬，斩向金色拳头。
<br/>　　大成圣体凌天的身影显露出来，他毫不犹豫挥起拳头，轰向长生天尊。
<br/>　　“长生天尊，你没有陨落在飞仙洞中，很好，那吾便来杀你！”
<br/>　　“大言不惭，大成圣体，别以为我会怕你！”长生天尊此时也认出了凌天，他冷声道，手中帝剑挥舞，无上法则缭绕，将天动地。
<br/>　　长生天尊曾经是地府之主，后来才成为仙陵之主，大成圣体之前身体上的红毛，晚年的不祥，便与地府有关。
<br/>　　两人早已是不死不休的局面了，一出手便是惊天动地，完全没有保留。
<br/>　　“动手！”
<br/>　　一声大喝，与此同时，其他众多轮回者也开始出手了，青帝一头长发乱舞，一道巨大的青色身影迈步而来，宇宙大道法则在颤栗，一柄青莲帝兵舞出，斩灭天地，杀向不死山石皇。
<br/>　　“青帝，你竟然没死！”
<br/>　　石皇神色震惊，但心中却是冰冷一片，他手中却毫不犹豫，挥起黑戟，无上皇道法则轰击而出，与青帝大战起来。
<br/>　　“既然没死，那我便送你去死！”
<br/>　　青帝的出现，瞬间惊动了整个宇宙，但让人更加震惊的却是接下来的一幕：
<br/>　　“杀！”
<br/>　　随着一声大喝，其他轮回者也没有停手，杀向其他至尊。
<br/>　　火麟儿哭喊这跑向麒麟古皇，这位古皇并没有发动黑暗动乱的想法，火麟儿的想法则是劝住古皇，帮忙对付禁区之人……
<br/>　　大成圣体叶梵头顶万物母气鼎，挥动手中灵宝天尊留下的的绝世杀剑，随着段德、圣皇子、姬子、轩辕黄帝杀向生命禁区不死山。
<br/>　　老疯子、老子、人魔东方太一杀向轮回海，轮回海如今只剩下昀亘至尊一人，是众生命禁区中最弱的。
<br/>　　昀亘至尊虽然只有一人，但却比其他至尊要厉害多，因为他是神话时代九天尊之一的逍遥天尊！
<br/>　　逍遥天尊如长生天尊一样，是神话时代的人物，更是开创了九字秘之行字诀，演化出时间法则，厉害的无边。
<br/>　　需要多去几人才能对付，逍遥天尊速度很快，甚至影响到了时间，需要有人防备他逃走。
<br/>　　另一边，神王姜太虚与神农大帝带着姜家恒宇炉，与琴帝华云飞，准帝帝皇、中皇项宇飞等带着一群轮回者来到太初古矿外守着。
<br/>　　太初古矿是诸多禁区之中最为神秘的存在，存在时间也最为久远，甚至实力在众多禁区中也是最为顶级的，谁也不知道这其中到底还有多少至尊。
<br/>　　九幽大帝盖九幽带着一群轮回者来到仙陵之外守着，不让里面的至尊出去增援，亦不让外面之人回来，他们的任务便是等着众人灭了五位受到重创的至尊之后，再合力毁灭一个个禁区。
<br/>　　其他轮回者则是去神墟与葬天岛外守着，不求禁区扫灭禁区，但求拖住时间。毕竟禁区内的至尊也不想死，不到最后一刻，谁也不想极尽升华，因为升华之后，便只有死路一条。
<br/>　　众多轮回者的出现，瞬间便惊动了整个宇宙，数位大帝级别的高手，无数准帝强者，道则惊天动地，整个宇宙都在颤抖摇晃，似乎是经受不住这些强大的道则轰击，想要毁灭。
<br/>　　这一刻，举世皆惊！
<br/>　　这一刻，万灵皆颤！
<br/>　　这一刻，天地失声！
