　　一杆黑色的大枪，突然从虚无中飚射而出，洞穿层层虚空，急速向苏莫袭杀而去。
<br/>　　这杆黑色的大枪，正是上次苏莫潜入黄泉魔宗，从阴罗城逃跑之时，所攻击过他的那杆黑色大枪。
<br/>　　此枪长约一丈有余，闪烁着冷厉的寒光，洞穿了虚空，裹挟惊天杀机，以无与伦比的速度袭向苏莫。
<br/>　　嗤！
<br/>　　黑色大枪穿过虚空，因速度太快，与空间摩擦迸射出刺目的火光。
<br/>　　这一枪是绝杀的一枪，不仅威力强大到极致，无可匹敌，更是胜在出其不意。
<br/>　　“什么？”
<br/>　　苏莫看到大枪的那一刻，瞬间面色大变，他自然知道此枪的强大，这一击蕴含的威势，让他头皮一麻。
<br/>　　难怪阎泰等人不着急追他了，原来还有这种后手。
<br/>　　苏莫虽然不敢肯定施展这黑色大枪的人，是不是黄泉魔宗的宗主，但他能绝对的肯定，此人必定是一位准帝强者，这个整个苍穹世界最巅峰的存在。
<br/>　　黑色大枪速度极快，又是出其不意，乍一出现便临近了苏莫的身前，让他避无可避。
<br/>　　“喝！”
<br/>　　来不及多想，电光火石之间，苏莫厉喝一声，身上玄力涌动，一剑狠狠的劈向黑色大枪。
<br/>　　铛！
<br/>　　帝皇剑斩在黑色大枪上，顿时一声巨响，刺爆发出耀眼的火光。
<br/>　　苏莫顿时全身剧震，握剑的右臂瞬间被震开，险些握不住手中之剑。
<br/>　　轰！
<br/>　　而黑色大枪，遭到苏莫的斩击，只是微微一顿，稍稍有些偏离，但依然猛烈的刺在了苏莫的胸膛上。
<br/>　　远处，阎泰五人的身形都停了下来，静静的看着这一幕，心中俱是松了口气。
<br/>　　结束了！
<br/>　　二魔老出手，苏莫必死无疑了，根本没有任何悬念！
<br/>　　作为准帝强者，二魔老的实力之强，阎泰五人都是一清二楚。
<br/>　　可以说，整个苍穹世界，除了那少数的十几位准帝之外，无人能在二魔老的手上活命。
<br/>　　至于上次苏莫闯入九幽魔渊，二魔老之所以没杀死苏莫，其一是苏莫有超强的防御符，其二是二魔老没有追杀。
<br/>　　但现在不同，苏莫就算还有那种防御符，也必死无疑了。
<br/>　　噗！
<br/>　　黑色大枪击中苏莫的胸膛，劲气炸裂，让得苏莫张口喷出一股血箭。
<br/>　　而后，苏莫的身躯如离弦之箭，急速向后倒飞了出去。
<br/>　　“嗯？”
<br/>　　阎泰几人见此，顿时眼眸一凝，眸中露出意外之色。
<br/>　　他们本以为，二魔老这一击，会直接洞穿苏莫的胸膛，将苏莫彻底击杀。
<br/>　　没想到结果却并不是这样，黑色大枪并没有洞穿苏莫的身体。
<br/>　　嗖！
<br/>　　苏莫身形急速倒飞，倒飞两千余里，刚刚稳住身形，喉头一甜，便再次张口喷出一口鲜血。
<br/>　　苏莫脸色略微有些苍白，眼眸抬起，向黑色大枪出现之处望去，便看到哪里的空间轰然炸裂，一名身材高大的黑袍人，从虚空中大步走了出来。
<br/>　　这名黑袍人，身上笼罩着一层浓郁的黑雾，让人根本看不清面容。
<br/>　　但此人身上的气息，阴森、霸道、肆无忌惮，简直是强大到惊天动地，是苏莫从未见过的强大。
<br/>　　若单轮气息而言，此人可能比帝释还要强大。
<br/>　　当然，真实的战力，不是仅从气息就能决定的。
<br/>　　“二魔老！”
<br/>　　阎泰等五人，见到这名高大的黑袍人现身，立刻躬身一拜，面带恭敬之色。
<br/>　　二魔老轻轻摆了摆手，没有看阎泰等人，他一把抓住黑色的大枪，目光直视苏莫的胸膛，口中传出沙哑的声音：“有趣！居然还穿了一件帝级内甲！”
<br/>　　对于一击没有杀死苏莫，二魔老心中也很是意外，他本来也是准备一击灭杀苏莫。
<br/>　　“二魔老？”苏莫闻言皱眉，他从来没听闻过，黄泉魔宗还有这种人物。
<br/>　　他本以为此人是黄泉魔宗宗主，没想到却不是。
<br/>　　看来黄泉魔宗的准帝强者，都有两人不止，因为对方既然是二魔老，想必其上还要大魔老。
<br/>　　嗖！
<br/>　　下一刻，苏莫身形一转，整个人化为了一道剑光，毫不犹豫的转身就逃。
<br/>　　这个所谓的二魔老实在太强，根本不是他能对付的，必须尽快逃走，不然的话他九死一生。
<br/>　　刚才对方一击，已经让他受了重伤，五脏六腑都被震裂了。
<br/>　　若不是有帝级内甲的守护，刚才一击他几乎必死无疑。
<br/>　　苏莫心中暗叹，看来他还是小看了准帝强者的实力，不过，刚才那一击太过突然，若不是他毫无防备，也不会这般狼狈。
<br/>　　“不用挣扎了！”沙哑的声音再次响起，二魔老见苏莫逃走，脚步抬起，大步向苏莫追去。
<br/>　　二魔老的步伐非常的玄妙，一步踏出就能跨越千里，虚空和大地，在他的脚下仿佛缩短了距离。
<br/>　　其看似走的缓慢，实则速度比苏莫御剑飞行，还要快很多，急速和苏莫拉近距离。
<br/>　　“不好！”苏莫见二魔老急速追来，和他快速拉近距离，顿时心中一惊。
<br/>　　若是一旦被追上，今日怕是麻烦了，说不定又要动用大三宝术拼死一战了。
<br/>　　但是，后方还有阎泰等五名强者，今日他绝对讨不了好。
<br/>　　咻！
<br/>　　苏莫极力催动本命灵剑，他的速度已经达到了极致，但本命灵剑虽快，依旧比不上这名二魔老。
<br/>　　眼见对方快速接近，苏莫脸色阴沉了下来，也做好了竭力一战的准备。
<br/>　　没过多久，二魔老与苏莫的距离缩短到了一定的程度，其再次出手了。
<br/>　　“死！”
<br/>　　充满沧桑而又冰冷刺骨的声音，从二魔老的口中响起，他手中的黑色大枪黑光流转，猛然向前一刺。
<br/>　　刹那之间，一道漆黑的枪芒迸射而出，直击苏莫的脑袋。
<br/>　　枪芒长约百余丈，极为的纤细，仿佛就像是一根长长的钢针一般。
<br/>　　但是，这道枪芒的劲力，却是凝练到了极致，锋锐到了巅峰，仿佛能洞穿世间的一切。
<br/>　　细如钢针般的枪芒，瞬间跨越数千里的距离，以肉眼难辨的速度临近了苏莫。
<br/>　　苏莫顿时全身汗毛一炸，生出一股无比危险的感觉。
<br/>　　此时，就算是他想要动用大三宝术，都已经来不及了。
<br/>　　眼见着枪芒将要击中自己的脑袋，电光火石之间，苏莫急忙闪避，并且脑袋一歪，向一边躲避。
<br/>　　嗤！
<br/>　　钢针般的枪芒，从苏莫耳畔一穿而过，仿佛无迹可寻一般，只在空间中留下了一个微小的黑洞。
<br/>　　呼！
<br/>　　苏莫见此顿时松了一口气，然而下一刻，足足数十道同样的枪芒，从后方向他迸射而来。
<br/>　　二魔老再次的出手了！
<br/>　　几乎就在同一时刻，苏莫的视线中，出现了一座通天彻底的巨山。
<br/>　　这座巨山高大万丈，刺破了九天之上的云层，连天接地，巨大无比。
<br/>　　这是蛮荒山脉，圣山！
