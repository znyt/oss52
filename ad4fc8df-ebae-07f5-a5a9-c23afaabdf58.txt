　　一百颗晶石，这不是少数字了，一百多个丧尸绝对是非常多的丧尸，足够吞没许多幸存者了。
<br/>　　一百颗晶石下去。
<br/>　　林峰连破两阶，一鼓作气成为了三阶的高手。
<br/>　　而孙畅和陈勾则还都是一阶。
<br/>　　“三阶的力量，4000斤的力量了。”
<br/>　　林峰再拿那一把武士刀，轻若鸿毛，根本没有任何吃力的感觉。
<br/>　　“要不要继续扫荡下一层？”
<br/>　　林峰看了眼他们。
<br/>　　“干！”
<br/>　　“继续，不怂！”
<br/>　　陈勾和孙畅对视一眼两人二话不说就同意了。
<br/>　　“很好！还是刚才的战略！你们两个拿着盾牌顶着，我来收割！”
<br/>　　林峰也正有此意，他实力大增，正是豪气大发的时候。
<br/>　　如今看来，他们似乎可以把整个教学楼给横扫干净。
<br/>　　然后有机会去拿到教学楼外面，一个位于体育馆内的空投，体育馆就在教学楼旁边，挨得很近的，也是林峰最有希望拿到的。
<br/>　　至于图书馆和操场的空投，那基本上林峰没有指望过，这两个地方，从来都是人很多的地方。
<br/>　　尤其是操场，好几个班级，都在上面上课，此时此刻一眼扫过去好几百个丧尸。密密麻麻的。望而生畏。最主要的是那个地方太开阔了，不像是在楼道里，狭窄，可以利用地形，开阔地方很容易被丧尸袭击，一旦被丧尸扑到，那就根本站不起来了。
<br/>　　林峰三人当即一层层横扫下去。
<br/>　　有了神兵利器，再加上林峰一个三阶的。
<br/>　　可以说完全碾压这群丧尸了。
<br/>　　第六层很快横扫完毕。
<br/>　　这一层，发现了二十多个活着的学生和三个男老师。
<br/>　　看到林峰三人拿着武士刀拿着长枪浑身是血的样子都吓傻了眼了。旋即当看到是活人后，一个个都哇的哭了出来。
<br/>　　“哇。”
<br/>　　一个学妹是哇的一下扑到了林峰怀里。
<br/>　　林峰尴尬的不知所措，他和这个女学生压根不认识，估计对方也是情感爆发出来一瞬间做的本能动作。
<br/>　　“吓死我了。我还以为我要死在这里了。呜呜呜呜呜。”
<br/>　　这个学妹胸部很大，在林峰怀里蹭来蹭去，搞得林峰都起了反应了。
<br/>　　“谢谢你们，你们是那个班级的学生。”
<br/>　　男老师问道。
<br/>　　“我们都是高一三班的，班主任已经死了，你们可以去七层，哪里还有几十个学生和一个英语老师。”
<br/>　　林峰道。
<br/>　　“好好好。对了，同学们把你们的书包整理一下，记住，水啊食物这些东西都带上，课本……可以丢了，书包全部来装食物。”
<br/>　　男老师这句话可以说非常的有意思了。
<br/>　　一个老师让学生们把课本丢了。
<br/>　　不过林峰很欣赏这个老师，怪不得他们可以活到现在，只是因为他们看得清局势。
<br/>　　一群学生没有任何墨迹立即开始按照老师说的去做。服从命令，也是他们可以活到现在的重要因素。
<br/>　　“对了，你们几个这是？”
<br/>　　男老师看着林峰问道。
<br/>　　“我们要去下一层。”
<br/>　　林峰道。
<br/>　　男老师点了点头，“注意安全，量力而行。”
<br/>　　“知道了老师放心吧。”
<br/>　　林峰三人对视一眼立即就下楼去了。
<br/>　　“现在的学生厉害啊。”
<br/>　　“是啊，佩服。”
<br/>　　“高一三班那不是老万的班级。他们班级学生居然这么厉害勇敢，了不起。”
<br/>　　第六层，众人只杀了不到八十个丧尸。
<br/>　　孙畅和陈勾也顺利突破到了二阶。
<br/>　　“继续，第五层！”
<br/>　　一层层扫荡下去。
<br/>　　到最后。
<br/>　　林峰都突破四阶了。
<br/>　　而孙畅和陈勾也都三阶了。
<br/>　　放眼过去，可谓是一路的尸体。
<br/>　　他们如今看到这些丧尸都根本无动于衷了，一点情感波动都不会有。
<br/>　　不知不觉来到了第三层。
<br/>　　这里已经是高二甚至高三的班级了。
<br/>　　此时此刻。
<br/>　　在高三七班。
<br/>　　噗嗤。
<br/>　　一个女老师的脑袋被一个学生给直接踩爆了。
<br/>　　周遭一群人都是畏惧的看着这个人。
<br/>　　此人不是别人，正是学校的扛把子，曹豹，而刚才被他杀死的就是他们班的班主任。
<br/>　　“豹哥，这里有几个人被丧尸抓伤了。怎么办。”
<br/>　　一个小弟拿着桌脚过来道。
<br/>　　“抓伤了？”
<br/>　　曹豹走了过去，这三个男生眼巴巴的看着曹豹。
<br/>　　“豹哥，救救我们，我们这应该可以治好的。”
<br/>　　“没错，这只是个皮外伤。”
<br/>　　噗嗤。
<br/>　　下一刻。
<br/>　　一个男生脑袋就被曹豹给一棍子砸爆了。
<br/>　　旋即不给另外两人反应机会，脑袋全部被曹豹砸爆了。
<br/>　　“哼，没看过电影吗，被丧尸抓伤的人，哪怕是蹭破点皮，都要变成丧尸，这种问题还需要我来教？”
<br/>　　曹豹杀气森森道。仿佛这三个人根本不是他同学。杀起来没任何的手下留情和犹豫。
<br/>　　作为黑道大佬的儿子，七岁，他就砍过人。
<br/>　　十三岁他就亲眼看过自己老爹，把一个还不起赌债的赌徒全家乱刀砍死然后丢进石油捅用水泥封起来丢进江里。
<br/>　　所以杀人对他来说，不是什么底线。也不是多么恐惧的事情。
<br/>　　一群人都害怕的看着曹豹，甚至是无比畏惧。哪怕是他的小弟也是如此。豹哥是真的社会。
<br/>　　“豹哥，我们现在怎么办。”
<br/>　　一个小弟问道。
<br/>　　“黑铁战士？有意思。都给我把这个晶石捏碎了，然后吞掉里面的液体。”
<br/>　　曹豹丢了几个晶石给自己战斗力最强的小弟。
<br/>　　几个小弟根本没有任何质疑全部照做，他们也都顺利成为了黑铁战士。
<br/>　　“这个世界似乎变得开始有意思起来了。”
<br/>　　曹豹冷笑着甩了甩手中的匕首，这把匕首是他放在书包里的，其余小弟也都有弹簧刀。
<br/>　　“走，从学校后门走，那里有我老爸的一个地下赌坊，不出意外，应该可以找到一些砍刀之类的武器。”
<br/>　　“是，豹哥。”
<br/>　　一群人没有任何犹豫全都跟着曹豹。
<br/>　　然而他们刚离开教室没走多远就遇到林峰三人。顿时两群人眼对眼。
<br/>　　“是你小子！没想到你个小子居然还活着！还真是傻人有傻福！”
<br/>　　曹豹冷笑。
<br/>　　“怎么样，上次的教训够不够？对了，我的吴倩倩呢。”
<br/>　　林峰看到是曹豹也是眼中露出一抹阴冷。还真是冤家路窄。
